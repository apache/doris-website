## Description

Used to return a randomly generated UUID string.

## Syntax

```sql
UUID()
```
## Return Value
Returns a randomly generated UUID string.

## Examples

```sql
select UUID();
```

```sql
+--------------------------------------+
| UUID()                               |
+--------------------------------------+
| 29077778-fc5e-4603-8368-6b5f8fd55c24 |
+--------------------------------------+

```
