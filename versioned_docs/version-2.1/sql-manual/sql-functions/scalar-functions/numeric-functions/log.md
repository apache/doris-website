## Description

Returns the logarithm of `x` based on base `b`.

## Syntax

```sql
LOG(<a>,<x>)
```

## Parameters

| Parameter | Description |
|-----------|------------|
| `<a>`   | Base should be greater than 0 and not be exactly 1.0 |
| `<x>`   | Antilogarithm should be greater than 0 |

## Return value

Return a float point number. Special cases:

- If a IS NULL or x IS NULL, return NULL

## Example

```sql
select log(5,1);
```

```text
+---------------+
| log(5.0, 1.0) |
+---------------+
|             0 |
+---------------+
```

```sql
select log(3,20);
```

```text
+--------------------+
| log(3.0, 20.0)     |
+--------------------+
| 2.7268330278608417 |
+--------------------+
```

```sql
select log(2,65536);
```

```text
+-------------------+
| log(2.0, 65536.0) |
+-------------------+
|                16 |
+-------------------+
```

```sql
select log(5,NULL);
```

```text
+------------------------------+
| log(cast(5 as DOUBLE), NULL) |
+------------------------------+
|                         NULL |
+------------------------------+
```

```sql
select log(NULL,3);
```

```text
+------------------------------+
| log(NULL, cast(3 as DOUBLE)) |
+------------------------------+
|                         NULL |
+------------------------------+
```
