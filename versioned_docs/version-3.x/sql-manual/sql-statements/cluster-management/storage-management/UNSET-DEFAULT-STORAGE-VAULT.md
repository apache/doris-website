## Description
Cancel the Specified Default Storage Vault

## Syntax

```sql
UNSET DEFAULT STORAGE VAULT
```

## Permission Control

| Privilege  | Object        | Notes                                                      |
| :--------- | :------------ | :--------------------------------------------------------- |
| ADMIN_PRIV | Storage Vault | Only admin users have permission to execute this statement |

## Example

```sql
UNSET DEFAULT STORAGE VAULT
```
