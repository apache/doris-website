# Metrics Action

## Request

`GET /api/metrics`

## Description

Used to obtain doris metrics infomation.
    
## Path parameters

None

## Query parameters

* `type`

    Optional parameter. Display all metrics by default, and with following values:
    - `core` Output core metrics
    - `json` Output metrics in json format
