## uuid

uuid

### description
#### Syntax

`VARCHAR uuid()`

```
mysql> select uuid();

| uuid()                               |
+--------------------------------------+
| 29077778-fc5e-4603-8368-6b5f8fd55c24 |
+--------------------------------------+

```
