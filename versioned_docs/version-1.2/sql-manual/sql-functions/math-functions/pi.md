## Pi

### description
#### Syntax

`DOUBLE Pi()`
Returns the constant `Pi` value.

### example

```
mysql> select Pi();
+--------------------+
| pi()               |
+--------------------+
| 3.1415926535897931 |
+--------------------+
```
