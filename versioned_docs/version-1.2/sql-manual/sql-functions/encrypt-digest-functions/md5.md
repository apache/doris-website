## MD5

### description
Calculates an MD5 128-bit checksum for the string
#### Syntax

`MD5(str)`

### example

```
MySQL [(none)]> select md5("abc");
+----------------------------------+
| md5('abc')                       |
+----------------------------------+
| 900150983cd24fb0d6963f7d28e17f72 |
+----------------------------------+
1 row in set (0.013 sec)
```
