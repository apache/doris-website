## bitor
### description
#### Syntax

`BITOR(Integer-type lhs, Integer-type rhs)`

Returns the result of the OR operation of two integers.

Integer range: TINYINT、SMALLINT、INT、BIGINT、LARGEINT

### example

```
mysql> select bitor(3,5) ans;
+------+
| ans  |
+------+
|    7 |
+------+

mysql> select bitand(4,7) ans;
+------+
| ans  |
+------+
|    7 |
+------+
```
