## CANCEL-BACKUP

### Name

CANCEL BACKUP

### Description

This statement is used to cancel an ongoing BACKUP task.

grammar:

```sql
CANCEL BACKUP FROM db_name;
```

### Example

1. Cancel the BACKUP task under example_db.

```sql
CANCEL BACKUP FROM example_db;
```

### Keywords

     CANCEL, BACKUP
