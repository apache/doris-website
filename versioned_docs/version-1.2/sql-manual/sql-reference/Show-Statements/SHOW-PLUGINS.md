## SHOW-PLUGINS

### Name

SHOW PLUGINS

### Description

This statement is used to display installed plugins

grammar:

```SQL
SHOW PLUGINS
```

This command will display all user-installed and system built-in plugins

### Example

1. Show installed plugins:

    ```SQL
    SHOW PLUGINS;
    ```

### Keywords

    SHOW, PLUGINS
