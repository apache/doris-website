## SHOW-ROLES

### Name

SHOW ROLES

### Description

This statement is used to display all created role information, including role name, included users and permissions.

grammar:

```SQL
SHOW ROLES
```

### Example

1. View created roles

    ```SQL
    SHOW ROLES
    ```

### Keywords

    SHOW, ROLES
