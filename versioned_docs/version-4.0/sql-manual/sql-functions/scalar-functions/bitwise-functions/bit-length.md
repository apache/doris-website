## 描述

返回字符串或二进制值的位长度（实际长度就是字节数 * 8。）。

## 语法
```sql
BIT_LENGTH(<str>)
```

## 参数
- `<str>` 要返回长度的字符串值。

## 返回值

返回 `<str>` 在二进制表示中所占的位数，包括所有 0 和 1。

## 示例
1. Example 1
    ```sql
    select BIT_LENGTH("abc"), BIT_LENGTH("中国"), BIT_LENGTH(123);
    ```

    ```text
    +-------------------+----------------------+-----------------+
    | BIT_LENGTH("abc") | BIT_LENGTH("中国")   | BIT_LENGTH(123) |
    +-------------------+----------------------+-----------------+
    |                24 |                   48 |              24 |
    +-------------------+----------------------+-----------------+
    ```
2. NULL argument
    ```sql
    select BIT_LENGTH(NULL);
    ```
    ```text
    +------------------+
    | BIT_LENGTH(NULL) |
    +------------------+
    |             NULL |
    +------------------+
    ```
