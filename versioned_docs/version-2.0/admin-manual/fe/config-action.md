# Config Action

## Request

```
GET /rest/v1/config/fe/
```

## Description

Config Action is used to obtain current FE configuration information.
    
## Path parameters

None

## Query parameters

* `conf_item`

    Optional parameters. Return specified item in FE configuration.

## Request body

None
