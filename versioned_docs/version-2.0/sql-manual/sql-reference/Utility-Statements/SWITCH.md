## SWITCH

### Name

SWITCH

### Description

This statement is used to switch catalog.

Syntax:

```sql
SWITCH catalog_name
```

### Example

1. Switch to hive

   ```sql
  	SWITCH hive;
  	```

### Keywords

SWITCH, CATALOG
