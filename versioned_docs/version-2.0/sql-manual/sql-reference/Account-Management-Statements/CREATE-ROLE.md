## CREATE-ROLE

### Name

CREATE ROLE

### Description

The statement user creates a role

```sql
  CREATE ROLE rol_name;
```

This statement creates an unprivileged role, which can be subsequently granted with the GRANT command.

### Example

1. Create a character

    ```sql
    CREATE ROLE role1;
    ```

### Keywords

     CREATE, ROLE
