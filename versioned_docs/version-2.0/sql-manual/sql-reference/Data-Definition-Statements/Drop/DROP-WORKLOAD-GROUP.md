## DROP-WORKLOAD-GROUP

### Name

DROP WORKLOAD GROUP

 

### Description

This statement is used to delete a workload group.

```sql
DROP WORKLOAD GROUP [IF EXISTS] 'rg_name'
```

### Example

1. Delete the workload group named g1:
    
    ```sql
    drop workload group if exists g1;
    ```

### Keywords

    DROP, WORKLOAD, GROUP
