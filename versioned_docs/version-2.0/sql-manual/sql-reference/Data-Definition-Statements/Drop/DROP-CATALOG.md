## DROP-CATALOG

### Name

CREATE CATALOG

### Description

This statement is used to delete the external catalog.

Syntax:

```sql
DROP CATALOG [I EXISTS] catalog_name;
```

### Example

1. Drop catalog hive

   ```sql
   DROP CATALOG hive;
   ```

### Keywords

DROP, CATALOG
