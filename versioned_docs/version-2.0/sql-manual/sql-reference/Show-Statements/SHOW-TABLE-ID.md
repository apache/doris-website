## SHOW-TABLE-ID

### Name

SHOW TABLE ID

### Description

This statement is used to find the corresponding database name, table name according to the table id (only for administrators)

grammar:

```sql
SHOW TABLE [table_id]
```

### Example

  1. Find the corresponding database name, table name according to the table id

     ```sql
     SHOW TABLE 10001;
     ```

### Keywords

    SHOW, TABLE, ID
