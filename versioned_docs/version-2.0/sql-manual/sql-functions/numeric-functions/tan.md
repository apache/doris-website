## tan

### description
#### Syntax

`DOUBLE tan(DOUBLE x)`
Returns the tangent of `x`, where `x` is in radians.

### example

```
mysql> select tan(0);
+----------+
| tan(0.0) |
+----------+
|        0 |
+----------+
mysql> select tan(1);
+--------------------+
| tan(1.0)           |
+--------------------+
| 1.5574077246549023 |
+--------------------+
```
