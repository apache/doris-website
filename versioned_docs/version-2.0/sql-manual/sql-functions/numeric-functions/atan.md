## atan

### description
#### Syntax

`DOUBLE atan(DOUBLE x)`
Returns the arctangent of `x`, where `x` is in radians.

### example

```
mysql> select atan(0);
+-----------+
| atan(0.0) |
+-----------+
|         0 |
+-----------+
mysql> select atan(2);
+--------------------+
| atan(2.0)          |
+--------------------+
| 1.1071487177940904 |
+--------------------+
```
