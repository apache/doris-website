## Alias

## Syntax

``` sql
SHA1( <str> )
```

## Parameters

| parameter | description         |
|-----------|-------------|
| `<str>`   | The sha1 value to be calculated |

## Return Value

Returns the sha1 value of the input string


## Examples

```sql
select sha("123"), sha1("123");
```
