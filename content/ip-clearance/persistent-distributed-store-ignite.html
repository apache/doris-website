    Ignite Persistent Store Codebase Intellectual Property (IP) Clearance Status

    Description

    The donated persistent store is a distributed and transactional disk storage that seamlessly integrates with Apache Ignite 2.0 page memory architecture. One of the main advantages of the store is that Apache Ignite becomes a fully operational from disk (SSD or Flash) without any need to preload the data in memory. Plus, with full SQL support already available in Apache Ignite, this feature will allow Apache Ignite serve as a distributed transactional database, both in memory and on disk, while continuing to support all the existing use cases.
    Refer to these wiki pages for more details:

    • Persistent Store Design
    • Persistent Store Architecture

    Project info

    • Apache Ignite PMC will be responsible for the code.
    • The persistent store will be added under "pds" and "core" modules.
    • Officer or member managing donation: Denis Magda (Apache Ignite PMC Chair)
