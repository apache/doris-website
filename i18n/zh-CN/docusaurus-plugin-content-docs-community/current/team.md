# Doris 团队

<!-- 知识类型: 概念介绍 -->
<!-- 适用场景: 了解社区治理 / 寻找成员名单 -->

Apache Doris 由分布在全球各地的贡献者共同开发与维护，遵循 Apache 软件基金会（ASF）的开放治理模式。社区角色分为三类，对应不同的权限与职责。

## 社区角色

| 角色 | 权限与职责 |
|------|------------|
| **PMC（Project Management Committee）** | 项目管理委员会成员，负责项目整体方向、版本发布、社区治理，以及新 Committer / PMC 的提名与投票 |
| **Committer** | 拥有代码仓库提交权限的活跃贡献者，负责代码、文档、Issue 与 PR 的审阅与合入 |
| **Contributor** | 通过 PR、Issue、文档、布道、答疑等方式参与社区的所有贡献者 |

如果你希望了解从 Contributor 成长为 Committer / PMC 的路径，请参阅 [贡献者成长路径](how-to-contribute/contributor-guide)。
