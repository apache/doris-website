# 订阅 Apache Doris 邮件列表

<!-- 知识类型: 操作步骤 -->
<!-- 适用场景: 新用户入门 / 加入社区 / 跟进社区决策 -->

邮件列表（Mail List）是 Apache 社区最被认可的交流方式，承载日常提问、技术讨论与事务决策。其异步、广播的特性，特别适合分布在全球的开源社区沟通。本文介绍如何用 5 步订阅 Apache Doris 的 `dev` 邮件列表。

## 邮件列表地址速查

订阅、讨论、退订使用的是不同的邮箱地址，按用途对应如下：

| 用途 | 邮箱地址 | 说明 |
|------|----------|------|
| 订阅 | `dev-subscribe@doris.apache.org` | 任意主题与内容均可 |
| 接收确认 | `dev-help@doris.apache.org` | 自动回复，需再次回复完成确认 |
| 日常讨论 | `dev@doris.apache.org` | 订阅成功后用于发问与讨论 |
| 退订 | `dev-unsubscribe@doris.apache.org` | 任意主题与内容均可 |

## 订阅步骤

### 1. 发送订阅邮件

打开自己的邮箱，新建邮件，向 `dev-subscribe@doris.apache.org` 发送一封邮件，主题与内容任意。

![step1](/images/subscribe-mail-list-step1.png)

### 2. 接收来自 dev-help 的确认邮件

完成第一步后，你将收到一封来自 `dev-help@doris.apache.org` 的确认邮件。

> 如果长时间未收到，请检查该邮件是否被拦截，或被自动归入「订阅邮件」「垃圾邮件」「推广邮件」等文件夹。

![step2](/images/subscribe-mail-list-step2.png)

### 3. 回复确认邮件

针对上一步收到的邮件，两种方式任选其一：

- **方式 A**：直接回复该邮件。
- **方式 B**：新建一封邮件，收件人填上一步邮件中的「回复地址」。

无论哪种方式，主题与内容均不限。

![step3](/images/subscribe-mail-list-step3.png)

### 4. 接收欢迎邮件

完成第三步后，将收到一封标题为 **WELCOME to dev@doris.apache.org** 的欢迎邮件。至此订阅完成，社区动态将通过该邮件列表推送给你。

![step4](/images/subscribe-mail-list-step4.png)

### 5. 发起邮件讨论（可选）

成功订阅后，向 `dev@doris.apache.org` 发送邮件，即可发起讨论，所有订阅者都将收到该邮件。

## 退订邮件列表

如果想退订，向 `dev-unsubscribe@doris.apache.org` 发送一封邮件，主题与内容任意，按返回邮件指引完成确认即可。
