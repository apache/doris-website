## 描述

该语句用来查看表的列统计信息。

## 语法

```SQL
SHOW COLUMN [CACHED] STATS <table_name> [ (<column_name> [, ...]) ];
```

## 必选参数

**1. `<table_name>`**

> 需要展示列统计信息的表名。

## 可选参数

**1. `CACHED`**

> 显示 FE 缓存中的统计信息。不指定的时候默认显示统计信息表中持久化的信息。

**2. `<column_name>`**

> 指定需要显示的列名。列名在表中必须存在，多个列名之间用逗号分隔。如果不指定，默认显示所有列的信息。

## 返回值

| 列名 | 说明           |
| -- |--------------|
| column_name | 列名           |
| index_name |   列所属的索引名           |
| count | 列的行数           |
| ndv | 列的基数         |
| num_null | 列的空值数           |
| data_size |   列的总数据量           |
| avg_size_byte |  列的平均字节数           |
| min | 列的最小值         |
| max | 列的最大值           |
| method |   收集方式          |
| type | 收集类型           |
| trigger | 触发方式         |
| query_times | 信息被查询次数          |
| updated_time |   信息更新时间           |
| update_rows | 上次收集时数据更新行数           |
| last_analyze_row_count | 上次收集时表的总行数         |
| last_analyze_version | 上次收集时表的版本值         |

## 权限控制

执行此 SQL 命令的用户必须至少具有以下权限：

| 权限（Privilege） | 对象（Object） | 说明（Notes）                                    |
|:--------------| :------------- |:------------------------------------------------|
| SELECT_PRIV   | 表（Table）    | 当执行 SHOW 时，需要拥有被查询的表的 SELECT_PRIV 权限 |
