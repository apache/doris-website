## 描述

求整数类型，a / b 的余数，浮点类型请使用 fmod 函数。

## 语法

```sql
MOD(<col_a> , <col_b>)
```

## 参数

| 参数 | 说明 |
| -- | -- |
| `<col_a>` | 被除数 |
| `<col_b>` | 除  数 不能为 0|

## 返回值

返回一个整型。特殊情况：

- 当 col_a 为 `NULL` 或 col_b 为 `NULL`时，返回 `NULL`

## 举例

```sql
select mod(10, 3);
```

```text
+----------+
| (10 % 3) |
+----------+
|        1 |
+----------+
```

```sql
select mod(10, 0);
```

```text
+----------+
| (10 % 0) |
+----------+
|     NULL |
+----------+
```
