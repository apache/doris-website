## 描述

使用 UTF-8 编码完成所提供文本的 URL 编码。通常用于对作为 URL 的一部分传递的参数信息进行编码

## 语法

```sql
URL_ENCODE(  <str>  ) 
```

## 必选参数

| 参数 | 描述 |
|------|------|
| `<str>` | 待编码的字符串 |

## 返回值

UTF-8 编码完成所提供文本的 URL 编码

## 示例

```sql
select  URL_ENCODE('Doris Q&A');
```

```sql
+-------------------------+
| url_encode('Doris Q&A') |
+-------------------------+
| Doris+Q%26A             |
+-------------------------+

```
