## 描述

SPACE 函数用于生成由指定数量的空格组成的字符串。

## 语法

```sql
SPACE(<len>)
```

## 参数

| 参数 | 说明 |
| ------- | ----------------------------------------- |
| `<len>` | 要生成的空格数量。类型：INT |

## 返回值

返回 VARCHAR 类型，包含指定数量的空格字符。

特殊情况：
- 如果参数为 NULL，返回 NULL
- 如果 `<len>` 小于或等于 0，返回空字符串

## 示例

1. 基本用法：生成指定数量空格
```sql
SELECT space(5);
```
```text
+----------+
| space(5) |
+----------+
|          |
+----------+
```

2. 零或负数处理
```sql
SELECT space(0), space(-5);
```
```text
+----------+-----------+
| space(0) | space(-5) |
+----------+-----------+
|          |           |
+----------+-----------+
```

3. NULL 值处理
```sql
SELECT space(NULL);
```
```text
+-------------+
| space(NULL) |
+-------------+
| NULL        |
+-------------+
```

4. 配合其他函数使用
```sql
SELECT CONCAT('Hello', space(3), 'World');
```
```text
+------------------------------------+
| concat('Hello', space(3), 'World') |
+------------------------------------+
| Hello   World                      |
+------------------------------------+
```
