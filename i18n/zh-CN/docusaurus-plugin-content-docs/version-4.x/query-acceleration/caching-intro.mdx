Apache Doris 提供多层缓存来加速查询：结果级缓存复用相同 SQL 的查询结果，Segment 级缓存复用过滤条件的计算结果，文件级缓存把远端湖仓数据下沉到本地磁盘。请按照负载特征选择合适的缓存方式。
