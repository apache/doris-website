Apache Doris 的自适应优化器和 Pipeline 执行引擎可以让大多数业务开箱即用，但生产环境通常还需要系统化的性能调优。建议先按照调优方法论定位瓶颈，再针对查询或导入侧的具体场景进行优化；遇到难以解释的执行计划时，可以查阅优化技术原理来理解其背后的机制。
