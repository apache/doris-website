## 描述

对单个整数列或表达式中的所有值执行按位 and 运算

## 语法

```sql
GROUP_BIT_AND(<expr>)
```

## 参数

| 参数 | 说明 |
| -- | -- |
| `<expr>` | 支持所有INT类型 |

## 返回值

返回一个整数值

## 举例

```sql
select * from group_bit;
```

```text
+-------+
| value |
+-------+
|     3 |
|     1 |
|     2 |
|     4 |
+-------+
```

```sql
select group_bit_and(value) from group_bit;
```

```text
+------------------------+
| group_bit_and(`value`) |
+------------------------+
|                      0 |
+------------------------+
```
