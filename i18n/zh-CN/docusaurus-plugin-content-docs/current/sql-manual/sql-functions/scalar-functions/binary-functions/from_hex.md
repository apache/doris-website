## 描述
将输入的二进制数据转换成使用十六进制编码转换后的字符串

## 别名

FROM_BINARY

## 语法

```sql
FROM_HEX ( <varbinary> )
```

## 参数

| 参数    | 说明           |
|-------|--------------|
| `<varbinary>` | 待转换的二进制数据 |

## 返回值

将输入的二进制数据使用十六进制编码转换成字符串。

## 举例

```sql
SELECT FROM_HEX(NULL);
```

```text
+----------------+
| FROM_HEX(NULL) |
+----------------+
| NULL           |
+----------------+
```

```sql
SELECT FROM_HEX(X'AB');
```

```text
+-----------------+
| FROM_HEX(X'AB') |
+-----------------+
| AB              |
+-----------------+
```
