## 描述

返回二进制串的字节数。

## 语法

```sql
LENGTH ( <bin> )
```

## 参数

| 参数      | 说明            |
|---------|---------------|
| `<bin>` | 需要计算字节的二进制串    |

## 返回值

二进制串 `<bin>` 的字节数。

## 举例

```sql
SELECT length(x'34AAA'), length(x'34AAA');
```

```text
+------------------+------------------+
| length(x'34AAA') | length(x'34AAA') |
+------------------+------------------+
|                3 |                3 |
+------------------+------------------+
```
