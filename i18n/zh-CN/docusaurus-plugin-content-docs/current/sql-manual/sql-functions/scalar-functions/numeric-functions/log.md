## 描述

返回基于底数`b`的`x`的对数。

## 语法

```sql
LOG(<b>[, <x>])
```

## 参数

| 参数    | 说明 |
|-------| -- |
| `<b>` | 底数 必须大于 0 且不等于 1 |
| `<x>` | 可选，真数必须大于 0，默认为自然数 e |

## 返回值

返回一个浮点数。特殊情况：

- 当 b 为 NULL 或 x 为 NULL 时，返回`NULL`
- 当 任意一个参数 NaN 时，返回 NaN
- 当 参数不满足输入限制时，返回 NULL

## 举例

```sql
select log(5,1);
```

```text
+---------------+
| log(5.0, 1.0) |
+---------------+
|             0 |
+---------------+
```

```sql
select log(3),ln(3);
```

```text
+--------------------+--------------------+
| log(3)             | ln(3)              |
+--------------------+--------------------+
| 1.0986122886681098 | 1.0986122886681098 |
+--------------------+--------------------+
```

```sql
select log(3,20);
```

```text
+--------------------+
| log(3.0, 20.0)     |
+--------------------+
| 2.7268330278608417 |
+--------------------+
```

```sql
select log(2,65536);
```

```text
+-------------------+
| log(2.0, 65536.0) |
+-------------------+
|                16 |
+-------------------+
```

```sql
select log(5,NULL);
```

```text
+------------------------------+
| log(cast(5 as DOUBLE), NULL) |
+------------------------------+
|                         NULL |
+------------------------------+
```

```sql
select log(NULL,3);
```

```text
+------------------------------+
| log(NULL, cast(3 as DOUBLE)) |
+------------------------------+
|                         NULL |
+------------------------------+
```

```sql
select log(2 ,cast('nan' as double));
```

```text
+-------------------------------+
| log(2 ,cast('nan' as double)) |
+-------------------------------+
|                           NaN |
+-------------------------------+
```

```sql
select log(2 ,-1);
```

```text
+------------+
| log(2 ,-1) |
+------------+
|       NULL |
+------------+
```
