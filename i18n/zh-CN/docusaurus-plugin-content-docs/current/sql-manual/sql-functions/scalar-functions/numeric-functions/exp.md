## 描述

返回以`e`为底的`x`的幂。

## 别名

- DEXP

## 语法

```sql
EXP(<x>)
```

## 参数

| 参数 | 说明 |
| -- | -- |
| `<x>` | 自变量 |

## 返回值

返回一个 double。特殊情况：

- 当参数为`NULL`时，返回 NULL

- 当参数为比较大的时候，返回 Infinity

## 举例

```sql
select exp(2);
```

```text
+------------------+
| exp(2.0)         |
+------------------+
| 7.38905609893065 |
+------------------+
```

```sql
select exp(3.4);
```

```text
+--------------------+
| exp(3.4)           |
+--------------------+
| 29.964100047397011 |
+--------------------+
```

```sql
select exp(1000000);
```

```text
+--------------+
| EXP(1000000) |
+--------------+
|     Infinity |
+--------------+
```
