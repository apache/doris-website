## Description

将输入值向零的反方向进位（舍入）到下一个偶数整数。

## Syntax

```sql
EVEN(<a>)
```


## Parameters

| Parameter | Description |
| -- | -- |
| `<a>` | 要舍入为偶数的数值表达式 |

## Return Value

返回一个偶数整数，规则如下：

 - 若 x > 0，则向上舍入到最接近的偶数；

 - 若 x < 0，则向下舍入到最接近的偶数；

 - 若 x 本身为偶数，则直接返回。

## Examples

```sql
select even(2.9);
```

```text
+----------+
| even(2.9) |
+----------+
|        4 |
+----------+
```

```sql
select even(-2.9);
```

```text
+-----------+
| even(-2.9) |
+-----------+
|       -4  |
+-----------+
```

```sql
select even(4);
```

```text
+--------+
| even(4) |
+--------+
|      4 |
+--------+
```
