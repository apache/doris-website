## 描述

返回`x`的双曲正弦值。


## 语法

```sql
SINH(<x>)
```

## 参数

| 参数 | 描述 |  
| -- | -- |  
| `<x>` | 需要计算双曲正弦值的数值 |  

## 返回值

参数`x`的双曲正弦值。

## 示例

```sql
select sinh(0.0);
```

```
+-----------+
| sinh(0.0) |
+-----------+
|         0 |
+-----------+
```

```sql
select sinh(1.0);
```

```
+--------------------+
| sinh(1.0)          |
+--------------------+
| 1.1752011936438014 |
+--------------------+
```

```sql
select sinh(-1.0);
```

```
+---------------------+
| sinh(-1.0)          |
+---------------------+
| -1.1752011936438014 |
+---------------------+
```
