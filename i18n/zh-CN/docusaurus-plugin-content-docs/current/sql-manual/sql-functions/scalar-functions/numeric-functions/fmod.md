## 描述

求浮点数类型，a / b 的余数，整数类型请使用 mod 函数。

## 语法

```sql
MOD(<col_a> , <col_b>)
```

## 参数

| 参数 | 说明 |
| -- | -- |
| `<col_a>` | 被除数 |
| `<col_b>` | 除  数 不能为 0|

## 返回值

返回一个浮点数。特殊情况：

- 当 col_a 为 `NULL` 或 col_b 为 `NULL`时，返回 `NULL`

## 举例

```sql
select fmod(10.1, 3.2);
```

```text
+-----------------+
| fmod(10.1, 3.2) |
+-----------------+
|      0.50000024 |
+-----------------+
```

```sql
select fmod(10.1, 0);
```

```text
+---------------+
| fmod(10.1, 0) |
+---------------+
|          NULL |
+---------------+
```
