## 描述

返回 x 的余割值，x 为弧度值

## 语法

```sql
COSEC(<x>)
```

## 参数

| 参数 | 说明 |
| -- | -- |
| `<x>` | 需要被计算余割值的值 |

## 返回值

返回 x 的余割值

## 举例

```sql
select cosec(1),cosec(2),cosec(1000);
```

```text
+--------------------+--------------------+------------------+
| cosec(1)           | cosec(2)           | cosec(1000)      |
+--------------------+--------------------+------------------+
| 1.1883951057781212 | 1.0997501702946164 | 1.20936599707935 |
+--------------------+--------------------+------------------+
```
