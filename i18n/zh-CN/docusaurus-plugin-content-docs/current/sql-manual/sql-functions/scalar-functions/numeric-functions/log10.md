## 描述

返回以`10`为底的`x`的自然对数。

## 别名

- DLOG10

## 语法

```sql
LOG10(<x>)
```

## 参数

| 参数 | 说明 |
| -- | -- |
| `<x>` | 真数 必须大于 0 |

## 返回值

返回一个浮点数。

- 当 x IS NULL：返回 `NULL`

## 举例

```sql
select log10(1);
```

```text
+--------------------------+
| log10(cast(1 as DOUBLE)) |
+--------------------------+
|                      0.0 |
+--------------------------+
```

```sql
select log10(10);
```

```text
+---------------------------+
| log10(cast(10 as DOUBLE)) |
+---------------------------+
|                       1.0 |
+---------------------------+
```

```sql
select log10(16);
```

```text
+---------------------------+
| log10(cast(16 as DOUBLE)) |
+---------------------------+
|        1.2041199826559248 |
+---------------------------+
```
