## Description

判断给定浮点数的符号位是否为负。

## Syntax

```sql
SIGNBIT(<a>)
```

## Parameters

| Parameter | Description |
| -- | -- |
| `<a>` | 要检查的浮点数参数 |

## Return Value

如果 `<a>` 的符号位为负（即 `<a>` 是负数），返回 true，否则返回 false。

## Examples

```sql
select signbit(-1.0);
```

```text
+-----------------------------+
| signbit(cast(-1 as DOUBLE)) |
+-----------------------------+
| true                        |
+-----------------------------+
```

```sql
select signbit(0.0);
```

```text
+----------------------------+
| signbit(cast(0 as DOUBLE)) |
+----------------------------+
| false                      |
+----------------------------+
```

```sql
select signbit(1.0);
```

```text
+----------------------------+
| signbit(cast(1 as DOUBLE)) |
+----------------------------+
| false                      |
+----------------------------+
```
