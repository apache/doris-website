## 描述

将十进制数字转换为二进制文本。

## 语法

```sql
bin(<a>)
```

## 参数

| 参数 | 说明 |
| -- | -- |
| `<a>` | 需要转换的十进制值(类型为BigInt) |

## 返回值

参数 `<a>` 的二进制表示。当 `<a>` 为负数时，结果为其 64 位补码表示。
当 `a` 为 NULL 时，返回 NULL。

## 举例

```sql
select bin(0);
```

```text
+--------+
| bin(0) |
+--------+
| 0      |
+--------+
```

```sql
select bin(-1);
```

```text
+------------------------------------------------------------------+
| bin(-1)                                                          |
+------------------------------------------------------------------+
| 1111111111111111111111111111111111111111111111111111111111111111 |
+------------------------------------------------------------------+
```

```sql
select bin(123);
```

```text
+----------+
| bin(123) |
+----------+
| 1111011  |
+----------+
```

```sql
select bin(NULL);
```

```text
+-----------+
| bin(NULL) |
+-----------+
| NULL      |
+-----------+
```
