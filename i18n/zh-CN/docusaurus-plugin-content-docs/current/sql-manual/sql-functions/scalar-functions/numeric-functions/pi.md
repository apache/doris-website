## 描述

返回常量`Pi`值。

## 语法

```sql
PI()
```

## 返回值

返回一个常量 Pi，类型为浮点数

## 举例

```sql
select Pi();
```

```text
+-------------------+
| pi()              |
+-------------------+
| 3.141592653589793 |
+-------------------+
```
