## 描述

返回 `x` 的双曲余弦。

## 语法

`COSH(<x>)`

## 参数

| 参数 | 说明 |
| -- | -- |
| `<x>` | 需要被计算双曲余弦的值 |

## 返回值

参数 `x` 的双曲余弦值

## 示例

```sql
select cosh(0);
```
```text
+-------------------------+
| cosh(cast(0 as DOUBLE)) |
+-------------------------+
|                     1.0 |
+-------------------------+
```

```sql
select cosh(1);
```
```text
+-------------------------+
| cosh(cast(1 as DOUBLE)) |
+-------------------------+
|       1.543080634815244 |
+-------------------------+
```

```sql
select cosh(-1);
```
```text
+--------------------------+
| cosh(cast(-1 as DOUBLE)) |
+--------------------------+
|        1.543080634815244 |
+--------------------------+
```
