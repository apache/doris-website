## 描述

返回 `x` 的双曲余弦。

## 语法

`COSH(<x>)`

## 参数

| 参数 | 说明 |
| -- | -- |
| `<x>` | 需要被计算双曲余弦的值 |

## 返回值

参数 `x` 的双曲余弦值

## 特殊情况
- 当 `x` 为 `NaN`，返回 `NaN`
- 当 `x` 为正无穷，返回 `Infinity`
- 当 `x` 为负无穷，返回 `Infinity`
- 当 `x` 为 `NULL`，返回 `NULL`

## 示例

```sql
select cosh(0);
```
```text
+-------------------------+
| cosh(cast(0 as DOUBLE)) |
+-------------------------+
|                     1.0 |
+-------------------------+
```

```sql
select cosh(1);
```
```text
+-------------------------+
| cosh(cast(1 as DOUBLE)) |
+-------------------------+
|       1.543080634815244 |
+-------------------------+
```

```sql
select cosh(-1);
```
```text
+--------------------------+
| cosh(cast(-1 as DOUBLE)) |
+--------------------------+
|        1.543080634815244 |
+--------------------------+
```

```sql
select cosh(cast('nan' as double));
```
```text
+-----------------------------+
| cosh(cast('nan' AS DOUBLE)) |
+-----------------------------+
| NaN                         |
+-----------------------------+
```

```sql
select cosh(cast('inf' as double));
```
```text
+-----------------------------+
| cosh(cast('inf' AS DOUBLE)) |
+-----------------------------+
| Infinity                    |
+-----------------------------+
```

```sql
select cosh(cast('-inf' as double));
```
```text
+------------------------------+
| cosh(cast('-inf' AS DOUBLE)) |
+------------------------------+
| Infinity                     |
+------------------------------+
```
