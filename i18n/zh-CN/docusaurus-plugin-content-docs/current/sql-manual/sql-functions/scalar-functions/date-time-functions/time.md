## 描述
`TIME` 函数可以获取一个`DateTime`值的 `time` 部分.

## 语法

```sql
TIME(<datetime>)
```

## 参数

| 参数           | 描述              |
|----------------|------------------|
| `<datetime>`   | 一个datetime值.   |

## 返回值
返回`TIME`类型的值

## 举例

```sql
SELECT TIME('2025-1-1 12:12:12');
```

```text
mysql> 
+---------------------------+
| time('2025-1-1 12:12:12') |
+---------------------------+
| 12:12:12                  |
+---------------------------+
```
