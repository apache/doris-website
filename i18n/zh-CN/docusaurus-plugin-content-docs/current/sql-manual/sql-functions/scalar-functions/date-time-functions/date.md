## 描述

DATE 函数用于从日期时间值（包含日期和时间）中提取出纯日期部分，忽略时间信息。该函数可将 DATETIME 类型或包含时间的字符串转换为 DATE 类型，仅保留年、月、日信息。

## 语法

```sql
DATE(<datetime>)
```

## 参数

| 参数 | 说明 |
| -- | -- |
| `<datetime>` | 合法的日期表达式,类型为 `datetime` |

## 返回值

若输入有效，返回 DATE 类型的纯日期值（格式为 YYYY-MM-DD），不含时间部分。
特殊情况：
- 输入为 NULL 时，返回 NULL；
- 输入为无效日期时间（如格式错误、超出范围）时，返回 NULL。

## 举例

```sql
---提取日期时间中的日期部分
mysql> select date('2010-12-02 19:28:30');
+-----------------------------+
| date('2010-12-02 19:28:30') |
+-----------------------------+
| 2010-12-02                  |
+-----------------------------+

-- 支持不同分隔符的日期时间字符串(具体日期格式支持那些写法请观看日期类型文档)
mysql> select date('2023/04/01 08:30:00');
+-----------------------------+
| date('2023/04/01 08:30:00') |
+-----------------------------+
| 2023-04-01                  |
+-----------------------------+

---输入为NULL
mysql> select date(NULL);
+------------+
| date(NULL) |
+------------+
| NULL       |
+------------+

---无效日期
mysql> select date('2023-02-30 12:00:00');
+-----------------------------+
| date('2023-02-30 12:00:00') |
+-----------------------------+
| NULL                        |
+-----------------------------+

mysql> select date('0000-00-00 12:00:00');
+-----------------------------+
| date('0000-00-00 12:00:00') |
+-----------------------------+
| NULL                        |
+-----------------------------+

```
