## 描述  
返回给定日期所属的世纪。  
特殊情况：  
- 如果日期在支持的范围 [0001‑01‑01, 9999‑12‑31] 之内且为有效日期，则返回所属的世纪。  
- 如果日期不在上述范围内，或者日期无效，则返回 `NULL`。

## 语法  
```sql
century(<x>)

```
## 举例

```sql
select century('2024-01-01');
```

```text
+------------------------+
| century('2024-01-01')  |
+------------------------+
| 21                     |
+------------------------+
```

```sql
select century('1999-12-31');
```

```text
+------------------------+
| century('1999-12-31')  |
+------------------------+
| 20                     |
+------------------------+
```

```sql
select century('0000-12-25');
```

```text
+---------------------------+
| century('0000-12-25')     |
+---------------------------+
| NULL                      |
+---------------------------+

```
