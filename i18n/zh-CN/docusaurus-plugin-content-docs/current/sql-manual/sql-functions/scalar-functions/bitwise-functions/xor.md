## 描述
用于对两个BOOLEAN值进行按位异或操作。

## 语法
```sql
 <lhs> XOR <rhs>
```

## 参数
| 参数    | 说明           |
|-------|--------------|
| `<lhs>` | 参与按位与运算的第一个BOOLEAN值 |
| `<rhs>` | 参与按位与运算的第二个BOOLEAN值 |

## 返回值
返回两个BOOLEAN值的异或值。

## 示例
```sql
select true XOR false,true XOR true;
```

```text
+------------------+-----------------+
| xor(TRUE, FALSE) | xor(TRUE, TRUE) |
+------------------+-----------------+
|                1 |               0 |
+------------------+-----------------+
```
