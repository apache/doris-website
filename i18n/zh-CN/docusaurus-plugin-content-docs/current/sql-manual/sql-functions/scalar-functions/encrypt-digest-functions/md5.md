## 描述

计算 MD5 128-bit

## 语法

```sql
MD5( <str> )
```

## 参数

| 参数      | 说明 |
|---------| -- |
| `<str>` | 需要被计算 MD5 的值 |

## 返回值
返回字符串的 MD5 值。

## 示例

```sql
select md5("abc");
```

```text
+----------------------------------+
| md5('abc')                       |
+----------------------------------+
| 900150983cd24fb0d6963f7d28e17f72 |
+----------------------------------+
```
