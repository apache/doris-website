## 描述

返回字符串或者二进制串的字节数。

## 语法

```sql
LENGTH ( <str> )
```

## 参数

| 参数      | 说明            |
|---------|---------------|
| `<str>` | 需要计算字节的字符串或者二进制串    |

## 返回值

字符串 `<str>` 的字节数。

## 举例

```sql
SELECT LENGTH("abc"),length("中国")
```

```text
+---------------+------------------+
| length('abc') | length('中国')   |
+---------------+------------------+
|             3 |                6 |
+---------------+------------------+
```

```sql
SELECT length(x'34AAA'), length(x'34AAA');
```

```text
+------------------+------------------+
| length(x'34AAA') | length(x'34AAA') |
+------------------+------------------+
|                3 |                3 |
+------------------+------------------+
```
