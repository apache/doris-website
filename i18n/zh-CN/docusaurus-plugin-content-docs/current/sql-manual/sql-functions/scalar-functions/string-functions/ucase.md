## 描述

用于将字符串转换为大写字母

## 语法

```sql
UCASE( <str> )
UPPER( <str> )
```
## 必选参数

| 参数 | 描述 |
|------|------|
| `<str>` | 需转换为大写字母的字符串 |

## 返回值

转换大写字母后的值

### 示例

```sql
SELECT UCASE("aBc123"),UPPER("aBc123");
```
```sql
+-----------------+-----------------+
| UCASE("aBc123") | UPPER("aBc123") |
+-----------------+-----------------+
| ABC123          | ABC123          |
+-----------------+-----------------+
```

```sql
SELECT UCASE("Кириллица"),UPPER("Бәйтерек");
```
```sql
+-----------------------------+---------------------------+
| UCASE("Кириллица")          | UPPER("Бәйтерек")         |
+-----------------------------+---------------------------+
| КИРИЛЛИЦА                   | БӘЙТЕРЕК                  |
+-----------------------------+---------------------------+
```
