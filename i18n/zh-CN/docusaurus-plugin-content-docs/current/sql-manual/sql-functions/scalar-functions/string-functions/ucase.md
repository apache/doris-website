## 描述

用于将字符串转换为大写字母

## 别名

- UPPER

## 语法

```sql
UCASE( <str> )
```
## 必选参数

| 参数 | 描述 |
|------|------|
| `<str>` | 需转换为大写字母的字符串 |

## 返回值

转换大写字母后的值

### 示例

```sql
SELECT ucase("aBc123");
```
```sql
+-----------------+
| ucase('aBc123') |
+-----------------+
| ABC123          |
+-----------------+
```
