## 描述

通过给定的 X 坐标值，Y 坐标值返回对应的 Point。

当前这个值只是在球面集合上有意义，X/Y 对应的是经度/纬度 (longitude/latitude);

## 语法

```sql
ST_POINT( <x>, <y>)
```
## 参数

| 参数  | 说明  |
|-----|-----|
| `<x>` | 横坐标 |
| `<y>` | 纵坐标 |

## 返回值

给定横坐标以及纵坐标对应的位置信息

## 举例

```sql
SELECT ST_AsText(ST_Point(24.7, 56.7));
```

```text
+---------------------------------+
| st_astext(st_point(24.7, 56.7)) |
+---------------------------------+
| POINT (24.7 56.7)               |
+---------------------------------+
```
