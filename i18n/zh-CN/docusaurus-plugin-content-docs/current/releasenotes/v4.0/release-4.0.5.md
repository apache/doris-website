# 新功能与改进

## 索引与搜索
- 支持将 MATCH 投影作为倒排索引的虚拟列 (#61092)
- 新增 PrefixQuery、PhrasePrefixQuery 和 UnionPostings 支持 (#60701)
- 支持倒排索引查询中的 BM25 评分 (#59847)
- 实现带词频的短语级 BM25 评分 (#60331)
- 将默认搜索模式从 standard 改为 lucene (#61055)

## 安全与认证
- 通过配置新增 LDAPS 支持 (#60275)
- 增强 Azure 对象存储客户端证书问题排查能力 (#61102)

# 问题修复

## 查询与执行
- 修复变体子列导致的查询缓存不一致问题 (#61709)
- 修复 MATCH 在别名槽位上的崩溃问题 (#61584)
- 修复数据量超过 4GB 时 set 操作崩溃的问题 (#61471)
- 修复外连接中 nullable 槽位的 TopN 过滤器问题 (#59074)
- 修复 MV 函数依赖计算错误 (#59933)
- 修复 SUM 和 AVG 中的类型强制转换问题 (#59602)

## 函数与类型
- 新增对 decimalv2 的 NDV 支持 (#61546)
- 修复 decimalv2 的 ABS 结果错误 (#61397)
- 修复 array_apply 中的 LARGEINT 处理问题 (#61081)

## 存储与压缩
- 修复 stale rowset 提交问题 (#61427)
- 修复 compaction 超时计算错误 (#61188)
- 修复 segment 损坏未触发缓存重试的问题 (#61386)
- 修复 move-memtable 相关的流式写入错误 (#60688)

## 索引与搜索
- 修复多索引列的确定性 reader 选择问题 (#61596)
- 修复特殊字符导致的搜索查询解析问题 (#61599)
- 修复 MUST_NOT 对 NULL 行的处理问题 (#61200)
- 修复倒排索引溢出问题 (#61120)

## 导入与流式处理
- 修复 load 内存回归问题 (#61327)
- 修复因忽略 S3 认证错误导致的流式任务卡死 (#61284)
- 修复流式任务调度问题 (#61249)
- 修复流式 insert replay NPE (#61253)
- 修复 FE 重启后 routine load 任务被误取消 (#61046)
- 修复 ALTER 后流式任务配置未刷新的问题 (#61451)

## 云与外部系统
- 修复批量删除中的空 rowset 处理 (#60919)
- 拒绝云模式下的非法管理操作 (#60875)
- 修复 RPC 指标命名问题 (#60745)
- 修复 Iceberg 表读取和重写失败 (#59984, #61112)
- 修复 ES 数组关键字字段的查询问题 (#61236)

## 文件缓存与 IO
- 修复文件缓存并发崩溃 (#60271)
- 修复空块文件处理 (#59646)

## 元数据与作业系统
- 修复作业调度和执行问题 (#61467, #61284)
- 修复无效存储路径上的备份快照上传 (#61251)

## 安全与认证
- 通过适当转义修复 LDAP 注入漏洞 (#61662)
- 修复非法登录后密码锁定失败 (#61592)
- 将 catalog 权限检查限制为有效操作 (#61147)

## 稳定性与其他
- 修复 FlushToken 计数器不一致 (#61684)
- 修复 profile 指标不准确 (#61064, #61601)
- 修复时区相关问题 (#61151)
- 修复 ORC heap-use-after-free 崩溃 (#61138)
- 修复 frontend TVF 元数据不一致 (#61087)
- 修复 insert overwrite 行为 (#61082)
- 修复 local shuffle 策略 (#61057)
- 修复 scan 节点缺少列大小指标 (#61086)
