本文档用于介绍通过`CREATE CATALOG`语句连接并访问 Google Dataproc Metastore 时所支持的参数。

| 属性名称               | 描述                                                                                                              | 默认值 | 是否必须 |
| ------------------ | --------------------------------------------------------------------------------------------------------------- | --- | ---- |
| `hive.metastore.uri` | Metastore 的 URI 地址。可以通过 [Dataproc Metastore Services](https://console.cloud.google.com/dataproc/metastore) 页面获取 | 无   | 是    |
