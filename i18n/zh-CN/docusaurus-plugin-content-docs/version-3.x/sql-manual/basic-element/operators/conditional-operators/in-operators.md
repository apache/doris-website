## 描述

IN 操作符测试一个值是否属于某个值列表或子查询中的成员。

## 操作符介绍

| 操作符 | 作用                                            | 示例                        |
| ------ | ----------------------------------------------- | --------------------------- |
| IN     | 测试是否等于任意的成员。如果是，则返回 TRUE。   | `SELECT 1 IN (SELECT 2)`    |
| NOT IN | 测试是否不等于所有的成员。如果是，则返回 TRUE。 | `SELECT 1 NOT IN (3, 4, 5)` |
