## 描述

使用 SHA2 对信息进行摘要处理。

## 语法

```sql
SHA2( <str>, <digest_length>)
```

## 参数

| 参数      | 说明          |
|---------|-------------|
| `<str>` | 待加密的内容 |
| `<digest_length>` | 摘要长度，支持 224, 256, 384, 512 |


## 返回值
返回输入字符串的 sha2 值

## 示例

```SQL
select sha2('abc', 224), sha2('abc', 384), sha2(NULL, 512);
```
