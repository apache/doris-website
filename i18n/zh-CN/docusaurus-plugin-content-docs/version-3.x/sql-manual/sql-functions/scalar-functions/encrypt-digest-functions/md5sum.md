## 描述

计算 多个字符串 MD5 128-bit

## 语法

```sql
MD5SUM( <str> [ , <str> ... ] )
```

## 参数

| 参数      | 说明 |
|---------| -- |
| `<str>` | 需要被计算 MD5 的值 |

## 返回值
返回多个字符串的 MD5 值。

## 示例

```sql
select md5("abcd"),md5sum("ab","cd");
```

```text
+----------------------------------+----------------------------------+
| md5('abcd')                      | md5sum('ab', 'cd')               |
+----------------------------------+----------------------------------+
| e2fc714c4727ee9395f324cd2e7f331f | e2fc714c4727ee9395f324cd2e7f331f |
+----------------------------------+----------------------------------+
```
