## 描述

将 value 添加到数组的开头

## 语法

```sql
ARRAY_PUSHFRONT(<arr>, <value>)
```

## 参数

| 参数 | 说明 |
|--|---|
| `<arr>` | 对应数组 |
| `<value>` | 待添加的值 |

## 返回值

返回添加 value 后的数组

## 举例

```sql
SELECT ARRAY_PUSHFRONT([1, 2], 3),ARRAY_PUSHFRONT([3, 4], 6);
```

```text
+----------------------------+----------------------------+
| array_pushfront([1, 2], 3) | array_pushfront([3, 4], 6) |
+----------------------------+----------------------------+
| [3, 1, 2]                  | [6, 3, 4]                  |
+----------------------------+----------------------------+
```
