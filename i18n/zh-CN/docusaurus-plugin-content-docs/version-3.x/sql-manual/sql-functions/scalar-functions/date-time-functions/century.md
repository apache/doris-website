## 描述  
返回给定日期所属的世纪。  
特殊情况：  
- 如果日期在支持的范围 [0001‑01‑01, 9999‑12‑31] 之内且为有效日期，则返回所属的世纪。  
- 如果日期不在上述范围内，或者日期无效，则返回 `NULL`。

## 语法  
```sql
century(<x>)


## 举例

```
mysql> select century('1987-01-01');
+-----------------------------+
|           year('1987-01-01')|
+-----------------------------+
|                           20|
+-----------------------------+

mysql> select century('2000-00-00');
+-----------------------------+
|           year('2000-00-00')|
+-----------------------------+
|                         NULL|
+-----------------------------+

select century('1999-12-31');
+------------------------+
| century('1999-12-31')  |
+------------------------+
| 20                     |
+------------------------+

select century('10000-01-01');
+----------------------------+
| century('10000-01-01')     |
+----------------------------+
| NULL                       |
+----------------------------+

```
