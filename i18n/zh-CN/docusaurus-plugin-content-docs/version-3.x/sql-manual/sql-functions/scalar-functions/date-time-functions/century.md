## century
## 描述
## 语法

`INT century(DATETIME date)`


返回 date 类型的 century 部分,支持识别无效部分(例如2000-00-00)

参数为 Date 或者 Datetime 类型

## 举例

```
mysql> select century('1987-01-01');
+-----------------------------+
|           year('1987-01-01')|
+-----------------------------+
|                           20|
+-----------------------------+

mysql> select century('2000-00-00');
+-----------------------------+
|           year('2000-00-00')|
+-----------------------------+
|                         NULL|
+-----------------------------+
```
