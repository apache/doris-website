## 描述

获得日期中对应当年中的哪一天。

## 语法

```sql
DAYOFYEAR(<dt>)
```

## 参数

| 参数 | 说明 |
| -- | -- |
| `<dt>` | 需要被计算的日期表达式 |

## 返回值

返回日期中对应当年中的哪一天。

## 举例

```sql
select dayofyear('2007-02-03 00:00:00');
```

```text
+----------------------------------+
| dayofyear('2007-02-03 00:00:00') |
+----------------------------------+
|                               34 |
+----------------------------------+
```
