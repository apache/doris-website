## 描述

计算参数的余弦值

## 语法

```sql
COS(<a>)
```

## 参数

| 参数 | 说明 |
| -- | -- |
| `<a>` | 浮点数，要计算参数的弧度值 |

## 返回值

参数 `<a>` 的余弦值，弧度制表示。

## 举例

```sql
select cos(1);
```

```text
+---------------------+
| cos(1.0)            |
+---------------------+
| 0.54030230586813977 |
+---------------------+
```

```sql
select cos(0);
```

```text
+------------------------+
| cos(cast(0 as DOUBLE)) |
+------------------------+
|                    1.0 |
+------------------------+
```

```sql
select cos(Pi());
```

```text
+-----------+
| cos(pi()) |
+-----------+
|        -1 |
+-----------+
```
