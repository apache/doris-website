# XOR

## Description

返回两个数字 XOR 的结果。

## Syntax

`BOOLEAN xor BOOLEAN`

## Example

```sql
mysql >select true xor false,true xor true;
+------------------+-----------------+
| (TRUE XOR FALSE) | (TRUE XOR TRUE) |
+------------------+-----------------+
|                1 |               0 |
+------------------+-----------------+
```

## Keywords
	XOR
