## 描述

REPEAT 函数用于将一个字符串重复指定的次数。

## 语法

```sql
REPEAT( <str>, <count> )
```

## 参数

| 参数 | 说明                        |
| -- |---------------------------|
| `<str>` | 需要被重复的字符串                 |
| `<count>` | 重复的次数，必须为非负整数，小于 1 时将返回空串 |

## 返回值

返回重复指定的次数的字符串。特殊情况：

- 任意参数中有一个为 NULL，则返回 NULL

## 举例

```sql
SELECT repeat("a", 3);
```

```text
+----------------+
| repeat('a', 3) |
+----------------+
| aaa            |
+----------------+
```

```sql
SELECT repeat("a", -1);
```

```text
+-----------------+
| repeat('a', -1) |
+-----------------+
|                 |
+-----------------+
```
