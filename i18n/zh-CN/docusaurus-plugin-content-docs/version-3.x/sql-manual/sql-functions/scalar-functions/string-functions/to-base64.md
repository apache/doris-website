## 描述

TO_BASE64 函数用于将输入的字符串转换为 Base64 编码格式。Base64 编码可以将任意二进制数据转换成由 64 个字符组成的字符串。

## 语法

```sql
TO_BASE64(<str>)
```

## 参数
| 参数 | 说明                                        |
| ---- | ------------------------------------------- |
| `<str>` | 需要进行 Base64 编码的字符串。类型：VARCHAR |

## 返回值

返回 VARCHAR 类型，表示 Base64 编码后的字符串。

特殊情况：
- 如果输入为 NULL，返回 NULL
- 如果输入为空字符串，返回空字符串

## 示例

1. 单字符编码
```sql
SELECT to_base64('1');
```
```text
+----------------+
| to_base64('1') |
+----------------+
| MQ==           |
+----------------+
```

2. 多字符编码
```sql
SELECT to_base64('234');
```
```text
+------------------+
| to_base64('234') |
+------------------+
| MjM0             |
+------------------+
```
