## 描述

返回字符串第一个字符对应的 ascii 码

## 语法

```sql
ASCII ( <str> )
```

## 参数

| 参数 | 说明                      |
|---------|-----------------------------|
| `<str>` | 需要被计算第一个字符的 ascii 码的字符串 |

## 返回值

参数 <str> 第一个字符的 ascii 码 

## 举例

```sql
SELECT ASCII('1'),ASCII('234')
```

```text
+------------+--------------+
| ascii('1') | ascii('234') |
+------------+--------------+
|         49 |           50 |
+------------+--------------+
```
