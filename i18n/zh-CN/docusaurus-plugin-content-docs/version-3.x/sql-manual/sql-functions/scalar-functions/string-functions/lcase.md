## 描述

将参数中所有的字符串都转换成小写。

## 别名

- LOWER

## 语法

```sql
LCASE ( <str> )
LOWER ( <str> )
```

## 参数

| 参数      | 说明           |
|---------|--------------|
| `<str>` | 需要被转换成小写的字符串 |

## 返回值

参数 `<str>`  转换成小写的字符串

## 举例

```sql
SELECT LOWER("AbC123"),LCASE("AbC123")
```

```text
+-----------------+-----------------+
| lower('AbC123') | lower('AbC123') |
+-----------------+-----------------+
| abc123          | abc123          |
+-----------------+-----------------+
```

```sql
SELECT LOWER("ҚAZAҚ123"),LCASE("ҒАРЫШ");
```

```text
+---------------------+---------------------+
| LOWER("ҚAZAҚ123")   | LCASE("ҒАРЫШ")      |
+---------------------+---------------------+
| қazaқ123            | ғарыш               |
+---------------------+---------------------+
```
