## 描述

REVERSE 函数用于将字符串中的字符顺序颠倒或将数组中的元素顺序颠倒。

## 语法

```sql
REVERSE( <seq> )
```

## 参数

| 参数      | 说明             |
|---------|----------------|
| `<seq>` | 需要被反转顺序的字符串或数组 |

## 返回值

返回反转顺序后的字符串或数组。特殊情况：

- 任意参数中有一个为 NULL，则返回 NULL

## 举例

```sql
SELECT reverse('hello');
```

```text
+------------------+
| REVERSE('hello') |
+------------------+
| olleh            |
+------------------+
```

```sql
SELECT reverse(['hello', 'world']);
```

```text
+-----------------------------+
| reverse(['hello', 'world']) |
+-----------------------------+
| ["world", "hello"]          |
+-----------------------------+
```
