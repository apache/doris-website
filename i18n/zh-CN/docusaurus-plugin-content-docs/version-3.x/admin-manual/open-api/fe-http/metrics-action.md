`GET /api/metrics`

## Description

获取 doris metrics 信息。
    
## Path parameters

无

## Query parameters

* `type`

    可选参数。默认输出全部 metrics 信息，有以下取值：
    - `core` 输出核心 metrics 信息
    - `json` 以 json 格式输出 metrics 信息
