## DATETIME
## 描述
    DATETIME
    日期时间类型，取值范围是['0000-01-01 00:00:00', '9999-12-31 23:59:59'].
    打印的形式是'yyyy-MM-dd HH:mm:ss'

### note
    如果您使用1.2及以上版本，强烈推荐您使用DATETIMEV2类型替代DATETIME类型。相比DATETIME类型，DATETIMEV2更加高效，并且支持了最多到微秒的时间精度。
