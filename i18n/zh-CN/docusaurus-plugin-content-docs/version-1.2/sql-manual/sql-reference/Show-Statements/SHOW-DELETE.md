## SHOW-DELETE

### Name

SHOW DELETE

## 描述

该语句用于展示已执行成功的历史 delete 任务

语法：

```sql
SHOW DELETE [FROM db_name]
```

## 举例

 1. 展示数据库 database 的所有历史 delete 任务
    
      ```sql
       SHOW DELETE FROM database;
      ```

### Keywords

    SHOW, DELETE
