## SHOW-DATA-TYPES

### Name

SHOW DATA TYPES

## 描述

    该语句用于查看DORIS支持的所有数据类型。

    语法：
        ```sql
        SHOW DATA TYPES;
        ```

## 举例

    1. 查看Doris支持的所有数据类型

        SHOW DATA TYPES;
