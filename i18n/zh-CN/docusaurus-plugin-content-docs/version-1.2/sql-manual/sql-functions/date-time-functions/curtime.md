## curtime,current_time

### Syntax

`TIME CURTIME()`

## 描述

获得当前的时间，以TIME类型返回

## 举例s

```sql
mysql> select current_time();
+----------------+
| current_time() |
+----------------+
| 15:25:47       |
+----------------+
```

### keywords

    CURTIME,CURRENT_TIME
