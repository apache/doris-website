## minutes_diff
## 描述
## 语法

`INT minutes_diff(DATETIME enddate, DATETIME startdate)`

开始时间到结束时间相差几分钟

## 举例

```
mysql> select minutes_diff('2020-12-25 22:00:00','2020-12-25 21:00:00');
+------------------------------------------------------------+
| minutes_diff('2020-12-25 22:00:00', '2020-12-25 21:00:00') |
+------------------------------------------------------------+
|                                                         60 |
+------------------------------------------------------------+
```
