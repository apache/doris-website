## months_diff
## 描述
## 语法

`INT months_diff(DATETIME enddate, DATETIME startdate)`

开始时间到结束时间相差几个月

## 举例

```
mysql> select months_diff('2020-12-25','2020-10-25');
+-----------------------------------------------------------+
| months_diff('2020-12-25 00:00:00', '2020-10-25 00:00:00') |
+-----------------------------------------------------------+
|                                                         2 |
+-----------------------------------------------------------+
```
