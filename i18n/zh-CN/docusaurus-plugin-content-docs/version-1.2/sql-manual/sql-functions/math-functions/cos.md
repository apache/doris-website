## cos

## 描述
## 语法

`DOUBLE cos(DOUBLE x)`
返回`x`的余弦值，`x` 为弧度值.

## 举例

```
mysql> select cos(1);
+---------------------+
| cos(1.0)            |
+---------------------+
| 0.54030230586813977 |
+---------------------+
mysql> select cos(0);
+----------+
| cos(0.0) |
+----------+
|        1 |
+----------+
mysql> select cos(Pi());
+-----------+
| cos(pi()) |
+-----------+
|        -1 |
+-----------+
```
