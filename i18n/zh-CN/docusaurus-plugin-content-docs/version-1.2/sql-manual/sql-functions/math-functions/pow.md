## pow

## 描述

返回第一个参数的值，该值是第二个参数的幂。

:::tip
该函数的其他别名为 `power`、`fpow`、`dpow`
:::

## 语法

`DOUBLE pow(DOUBLE a, DOUBLE b)`
返回`a`的`b`次方.

## 举例

```
mysql> select pow(2,0);
+---------------+
| pow(2.0, 0.0) |
+---------------+
|             1 |
+---------------+
mysql> select pow(2,3);
+---------------+
| pow(2.0, 3.0) |
+---------------+
|             8 |
+---------------+
mysql> select pow(3,2.4);
+--------------------+
| pow(3.0, 2.4)      |
+--------------------+
| 13.966610165238235 |
+--------------------+
```

### keywords
	POW, POWER, FPOW, DPOW
