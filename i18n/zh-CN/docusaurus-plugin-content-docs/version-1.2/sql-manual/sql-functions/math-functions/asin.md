## asin

## 描述
## 语法

`DOUBLE asin(DOUBLE x)`
返回`x`的反正弦值，若 `x`不在`-1`到 `1`的范围之内，则返回 `nan`.

## 举例

```
mysql> select asin(0.5);
+---------------------+
| asin(0.5)           |
+---------------------+
| 0.52359877559829893 |
+---------------------+
mysql> select asin(2);
+-----------+
| asin(2.0) |
+-----------+
|       nan |
+-----------+
```
