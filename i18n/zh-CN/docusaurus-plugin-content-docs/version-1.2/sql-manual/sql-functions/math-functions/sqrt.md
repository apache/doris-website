## sqrt

## 描述
## 语法

`DOUBLE sqrt(DOUBLE x)`
返回`x`的平方根，要求x大于或等于0.

:::tip
该函数的另一个别名为 `dsqrt`。
:::

## 举例

```
mysql> select sqrt(9);
+-----------+
| sqrt(9.0) |
+-----------+
|         3 |
+-----------+
mysql> select sqrt(2);
+--------------------+
| sqrt(2.0)          |
+--------------------+
| 1.4142135623730951 |
+--------------------+
mysql> select sqrt(100.0);
+-------------+
| sqrt(100.0) |
+-------------+
|          10 |
+-------------+
```

### keywords
	SQRT, DSQRT
