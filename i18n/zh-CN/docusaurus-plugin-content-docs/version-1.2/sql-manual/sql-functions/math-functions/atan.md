## atan

## 描述
## 语法

`DOUBLE atan(DOUBLE x)`
返回`x`的反正切值，`x`为弧度值.

## 举例

```
mysql> select atan(0);
+-----------+
| atan(0.0) |
+-----------+
|         0 |
+-----------+
mysql> select atan(2);
+--------------------+
| atan(2.0)          |
+--------------------+
| 1.1071487177940904 |
+--------------------+
```
