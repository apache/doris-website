## jsonb_exists_path

## 描述

用来判断json_path指定的字段在JSONB数据中是否存在，如果存在返回TRUE，不存在返回FALSE

## 语法

```sql
BOOLEAN jsonb_exists_path(JSONB j, VARCHAR json_path)
```
