# Hardware Info Action

## Request

```
GET /rest/v1/hardware_info/fe/
```

## Description

Hardware Info Action 用于获取当前FE的硬件信息。
    
## Path parameters

无

## Query parameters

无

## Request body

无


## Response

			"BuildTime": "二, 05 9月 2019 11:07:42 CST"
		},
		"HarewareInfo": {
			"NetworkParameter": "...",
			"Processor": "...",
			"OS": "...",
