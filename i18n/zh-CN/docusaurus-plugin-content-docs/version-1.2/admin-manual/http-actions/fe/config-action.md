# Config Action

## Request

```
GET /rest/v1/config/fe/
```

## Description

Config Action 用于获取当前 FE 的配置信息
    
## Path parameters

无

## Query parameters

* `conf_item`

    可选参数。返回 FE 的配置信息中的指定项。

## Request body

无
