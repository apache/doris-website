# Metrics Action

## Request

`GET /api/metrics`

## Description

获取doris metrics信息。
    
## Path parameters

无

## Query parameters

* `type`

    可选参数。默认输出全部metrics信息，有以下取值：
    - `core` 输出核心metrics信息
    - `json` 以json格式输出metrics信息
