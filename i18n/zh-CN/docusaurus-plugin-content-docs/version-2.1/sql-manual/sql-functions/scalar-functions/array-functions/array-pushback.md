## 描述

将 value 添加到数组的尾部。

## 语法

```sql
ARRAY_PUSHBACK(<arr>, <value>)
```

## 参数

| 参数 | 说明 |
|--|---|
| `<arr>` | 对应数组 |
| `<value>` | 待添加的值 |

## 返回值

返回添加 value 后的数组

## 举例

```sql
SELECT ARRAY_PUSHBACK([1, 2], 3),ARRAY_PUSHBACK([3, 4], 6);
```

```text
+---------------------------+---------------------------+
| array_pushback([1, 2], 3) | array_pushback([3, 4], 6) |
+---------------------------+---------------------------+
| [1, 2, 3]                 | [3, 4, 6]                 |
+---------------------------+---------------------------+
```
