## 描述

提取日期时间中的日期。

## 语法

```sql
DATE(<datetime>)
```

## 参数

| 参数 | 说明 |
| -- | -- |
| `<datetime>` | 合法的日期表达式 |

## 返回值

返回日期时间的日期部分。

## 举例

```sql
select date('2010-12-02 19:28:30');
```

```text
+----------------------------------------------------+
| date(cast('2010-12-02 19:28:30' as DATETIMEV2(0))) |
+----------------------------------------------------+
| 2010-12-02                                         |
+----------------------------------------------------+
```
