## 描述

返回日期的工作日索引值，即星期日为 1，星期一为 2，星期六为 7。

## 语法

```sql
DAYOFWEEK(<dt>)
```

## 参数

| 参数 | 说明 |
| -- | -- |
| `<dt>` | 需要被计算的日期表达式 |

## 返回值

返回日期的工作日索引值。

## 举例

```sql
select dayofweek('2019-06-25');
```

```text
+----------------------------------+
| dayofweek('2019-06-25 00:00:00') |
+----------------------------------+
|                                3 |
+----------------------------------+
```text

```sql
select dayofweek(cast(20190625 as date)); 
```

```text
+-----------------------------------+
| dayofweek(CAST(20190625 AS DATE)) |
+-----------------------------------+
|                                 3 |
+-----------------------------------+
```
