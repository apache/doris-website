## 描述

计算参数的立方根

## 语法

```sql
cbrt(<a>)
```

## 参数

| 参数 | 说明 |
| -- | -- |
| `<a>` | 浮点参数 |

## 返回值

参数 `<a>` 的立方根，浮点数。

## 举例

```sql
select cbrt(0);
```

```text
+-------------------------+
| cbrt(cast(0 as DOUBLE)) |
+-------------------------+
|                     0.0 |
+-------------------------+
```

```sql
select cbrt(-111);
```

```text
+----------------------------+
| cbrt(cast(-111 as DOUBLE)) |
+----------------------------+
|         -4.805895533705333 |
+----------------------------+
```

```sql
select cbrt(1234);
```

```text
+----------------------------+
| cbrt(cast(1234 as DOUBLE)) |
+----------------------------+
|         10.726014668827325 |
+----------------------------+
```
