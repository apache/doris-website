## 描述

当 point 是一个合法的 POINT 类型时，返回对应的 Y 坐标值

## 语法

```sql
ST_Y( <point>)
```
## 参数

| 参数   | 说明       |
|------|----------|
| `<point>` | 二维点的几何坐标 |

## 返回值

几何坐标中的 Y 值

## 举例

```sql
SELECT ST_Y(ST_Point(24.7, 56.7));
```

```text
+----------------------------+
| ST_Y(ST_Point(24.7, 56.7)) |
+----------------------------+
| 56.7                       |
+----------------------------+
```
