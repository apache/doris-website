## 描述

生成由指定数量的空格组成的字符串。

## 语法

```sql
SPACE ( <len> )
```

## 参数

| 参数      | 说明        |
|---------|-----------|
| `<len>` | 要生成的空格的数量 |

## 返回值

返回指定数量的空格组成的字符串。特殊情况：

- 任意参数中有一个为 NULL，则返回 NULL
- `<len>`小于 0 时，返回空字符串

## 举例

```sql
SELECT space(10);
```

```text
+------------+
| space(10)  |
+------------+
|            |
+------------+
```

```sql
SELECT space(null);
```

```text
+-------------+
| space(NULL) |
+-------------+
| NULL        |
+-------------+
```
