## ALTER ROLE

### Name

ALTER ROLE

### Description

该语句用于修改一个角色

```sql
 ALTER ROLE role_name comment;
```

### Example

1. 修改一个角色的注释

    ```sql
    ALTER ROLE role1 COMMENT "this is my first role";
    ```

### Keywords

    ALTER, ROLE
