## 描述

该语句用于查看 DORIS 支持的所有数据类型。

## 语法

```sql
SHOW DATA TYPES;
```

## 返回值

| 列名       | 说明   |
|----------|------|
| TypeName | 类型名称 |
| Size     | 字节大小 |

## 权限控制

执行此 SQL 命令的用户不需要具有特定的权限

## 示例

- 查看 Doris 支持的所有数据类型

   ```sql
   SHOW DATA TYPES;
   ```
