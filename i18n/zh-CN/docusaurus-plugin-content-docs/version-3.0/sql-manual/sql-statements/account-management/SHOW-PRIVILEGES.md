## 描述

该语句用于展示所有权限项。

语法：

```SQL
SHOW PRIVILEGES
```

## 示例

1. 查看所有权限项

   ```SQL
   SHOW PRIVILEGES
   ```

## 关键词

    SHOW, PRIVILEGES
