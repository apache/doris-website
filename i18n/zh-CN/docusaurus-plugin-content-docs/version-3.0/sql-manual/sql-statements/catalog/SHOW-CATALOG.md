## 描述

该语句用于显示已存在是数据目录（catalog）属性

## 语法

```sql
SHOW CATALOG <catalog_name>
```


## 必填参数

**1. `<catalog_name>`**

需要显示 catalog 的名字

## 返回值


## 权限控制

| 权限（Privilege）                                                                                | 对象（Object） | 说明（Notes）      |
|:---------------------------------------------------------------------------------------------|:-----------|:---------------|
| ADMIN_PRIV / SELECT_PRIV / LOAD_PRIV / ALTER_PRIV / CREATE_PRIV / SHOW_VIEW_PRIV / DROP_PRIV | Catalog    | 需要有上述权限中的一种就可以 |
