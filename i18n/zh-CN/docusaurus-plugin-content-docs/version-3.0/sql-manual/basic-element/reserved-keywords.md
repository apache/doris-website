## 描述

关键字是指在 SQL 中有特殊意义的词语。保留关键字是一类特殊的关键字，如果想使用保留关键字作为对象标识符，则必须使用反引号（`）。列如````FROM````。有关对象标识符的详细信息，请参阅[对象标识符](../basic-element/object-identifiers.md)章节。
