## 描述
用于执行 按位与（bitwise AND）运算。按位与运算会对两个整数的每一位进行比较，当两个对应的二进制位都为 1 时，结果才为 1，否则为 0。

整数范围：TINYINT、SMALLINT、INT、BIGINT、LARGEINT

## 语法
```sql
BITAND( <lhs>, <rhs>)
```

## 参数
| 参数    | 说明           |
|-------|--------------|
| `<lhs>` | 参与按位与运算的第一个数 |
| `<rhs>` | 参与按位与运算的第二个数 |

## 返回值

返回两个整数与运算的结果。


## 举例

```sql
select BITAND(3,5), BITAND(4,7);
```

```text
+---------+---------+
| (3 & 5) | (4 & 7) |
+---------+---------+
|       1 |       4 |
+---------+---------+

```
