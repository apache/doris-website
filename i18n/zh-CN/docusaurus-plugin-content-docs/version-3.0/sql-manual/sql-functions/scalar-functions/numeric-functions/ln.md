## 描述

返回以`e`为底的`x`的自然对数。

## 别名

- DLOG1

## 语法

```sql
LN(<x>)
```

## 参数

| 参数 | 说明 |
| -- | -- |
| `<x>`   | 真数 必须大于 0 |

## 返回值

返回一个浮点数。特殊情况：

- 当 x IS NULL 时，返回`NULL`

## 举例

```sql
select ln(1);
```

```text
+-----------------------+
| ln(cast(1 as DOUBLE)) |
+-----------------------+
|                   0.0 |
+-----------------------+
```

```sql
select ln(e());
```

```text
+---------+
| ln(e()) |
+---------+
|     1.0 |
+---------+
```

```sql
select ln(10);
```

```text
+------------------------+
| ln(cast(10 as DOUBLE)) |
+------------------------+
|      2.302585092994046 |
+------------------------+
```
