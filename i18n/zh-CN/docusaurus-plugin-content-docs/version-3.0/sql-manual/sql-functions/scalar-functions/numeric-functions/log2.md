## 描述

返回以`2`为底的`x`的自然对数。

## 语法

```sql
LOG2(<x>)
```

## 参数

| 参数 | 说明 |
| -- | -- |
| `<x>` | 真数 should be greater than 0 |

## 返回值

返回一个浮点数。特殊情况：

- 当x为NULL时，返回 `NULL`

## 举例

```sql
select log2(1);
```

```text
+-------------------------+
| log2(cast(1 as DOUBLE)) |
+-------------------------+
|                     0.0 |
+-------------------------+
```

```sql
select log2(2);
```

```text
+-------------------------+
| log2(cast(2 as DOUBLE)) |
+-------------------------+
|                     1.0 |
+-------------------------+
```

```sql
select log2(10);
```

```text
+--------------------------+
| log2(cast(10 as DOUBLE)) |
+--------------------------+
|       3.3219280948873626 |
+--------------------------+
```
