## 描述

返回`x`的反正弦值，若 `x`不在`-1`到 `1`的范围之内，则返回 `nan`.

## 语法

```sql
ASIN(<x>)
```

## 参数

| 参数 | 说明 |
| -- | -- |
| `<x>` | 需要被计算反正弦的值 |

## 返回值

参数 x 的反正弦值


## 举例

```sql
select asin(0.5);
```

```text
+---------------------+
| asin(0.5)           |
+---------------------+
| 0.52359877559829893 |
+---------------------+
```

```sql
select asin(2);
```

```text
+-----------+
| asin(2.0) |
+-----------+
|       nan |
+-----------+
```
