## 描述

计算 Map 中元素的个数

## 语法

```sql
MAP_SIZE(<map>)
```

## 参数

| 参数 | 说明 |
| -- | -- |
| `<map>` | 输入的 map 内容 |

## 返回值

返回 Map 中元素的个数

## 举例

```sql
select map_size(map()),map_size(map(1, "100", 0.1, 2));
```

```text
+-----------------+-------------------------------------------------------------------------------------------------+
| map_size(map()) | map_size(map(cast(1 as DECIMALV3(2, 1)), '100', cast(0.1 as DECIMALV3(2, 1)), cast(2 as TEXT))) |
+-----------------+-------------------------------------------------------------------------------------------------+
|               0 |                                                                                               2 |
+-----------------+-------------------------------------------------------------------------------------------------+
```
