## 描述

计算数组中元素的数量

## 别名

- SIZE
- CARDINALITY

## 语法

```sql
ARRAY_SIZE(<arr>) 
```

## 参数

| 参数 | 说明 |
|--|--|
| `<arr>` | 待计算的数组 |

## 返回值

返回数组中元素数量，如果输入数组为NULL，则返回NULL

## 举例

```sql
SELECT ARRAY_SIZE(['a', 'b', 'c']),ARRAY_SIZE([NULL]),ARRAY_SIZE([]);
```

```text
+------------------------------+---------------------+-----------------+
| cardinality(['a', 'b', 'c']) | cardinality([NULL]) | cardinality([]) |
+------------------------------+---------------------+-----------------+
|                            3 |                   1 |               0 |
+------------------------------+---------------------+-----------------+
```
