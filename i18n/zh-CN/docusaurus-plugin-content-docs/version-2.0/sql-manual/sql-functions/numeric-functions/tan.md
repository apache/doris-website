## tan

## 描述
## 语法

`DOUBLE tan(DOUBLE x)`
返回`x`的正切值，`x`为弧度值.

## 举例

```
mysql> select tan(0);
+----------+
| tan(0.0) |
+----------+
|        0 |
+----------+
mysql> select tan(1);
+--------------------+
| tan(1.0)           |
+--------------------+
| 1.5574077246549023 |
+--------------------+
```
