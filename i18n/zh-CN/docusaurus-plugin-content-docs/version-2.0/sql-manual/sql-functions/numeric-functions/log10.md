## log10

## 描述
## 语法

`DOUBLE log10(DOUBLE x)`
返回以`10`为底的`x`的自然对数.

:::tip
该函数的另一个别名为 `dlog10`。
:::

## 举例

```
mysql> select log10(1);
+------------+
| log10(1.0) |
+------------+
|          0 |
+------------+
mysql> select log10(10);
+-------------+
| log10(10.0) |
+-------------+
|           1 |
+-------------+
mysql> select log10(16);
+--------------------+
| log10(16.0)        |
+--------------------+
| 1.2041199826559248 |
+--------------------+
```

### keywords
	LOG10, DLOG10
