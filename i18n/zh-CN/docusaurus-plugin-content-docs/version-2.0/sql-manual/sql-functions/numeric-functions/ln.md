## ln

## 描述
## 语法

`DOUBLE ln(DOUBLE x)`
返回以`e`为底的`x`的自然对数.

:::tip
该函数的另一个别名为 `dlog1`。
:::

## 举例

```
mysql> select ln(1);
+---------+
| ln(1.0) |
+---------+
|       0 |
+---------+
mysql> select ln(e());
+---------+
| ln(e()) |
+---------+
|       1 |
+---------+
mysql> select ln(10);
+--------------------+
| ln(10.0)           |
+--------------------+
| 2.3025850929940459 |
+--------------------+
```

### keywords
	LN, DLOG1
