### 描述

计算 MD5 128-bit

### 语法

```sql
MD5(str)
```

### 示例

```sql
select md5("abc");
```

```
+----------------------------------+
| md5('abc')                       |
+----------------------------------+
| 900150983cd24fb0d6963f7d28e17f72 |
+----------------------------------+
```
