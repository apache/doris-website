### 描述

使用SHA1算法对信息进行摘要处理。

### 语法

``` sql
{ SHA | SHA1 }(str)
```

### 示例

```SQL
select sha("123");
```

```
+------------------------------------------+
| sha1('123')                              |
+------------------------------------------+
| 40bd001563085fc35165329ea1ff5c5ecbdbbeef |
+------------------------------------------+
```
