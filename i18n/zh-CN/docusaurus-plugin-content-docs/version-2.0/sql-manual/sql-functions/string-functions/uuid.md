## uuid

uuid

## 描述
## 语法

`VARCHAR uuid()`

返回一个随机的uuid字符串


## 举例

```
mysql> select uuid();
+--------------------------------------+
| uuid()                               |
+--------------------------------------+
| 29077778-fc5e-4603-8368-6b5f8fd55c24 |
+--------------------------------------+

```
