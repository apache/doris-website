## SHOW-COLUMNS

### Name

SHOW FULL COLUMNS

## 描述

该语句用于指定表的列信息

语法：

```sql
SHOW [FULL] COLUMNS FROM tbl;
```

## 举例

1. 查看指定表的列信息

   ```sql
    SHOW FULL COLUMNS FROM tbl;
   ```

### Keywords

    SHOW, FULL, COLUMNS
