## CLEAN-PROFILE

### Name


:::tip 提示
该功能自 Apache Doris  1.2 版本起支持
:::

CLEAN PROFILE



## 描述

用于手动清理所有历史 query 或 load 的 profile 信息。

语法：

```sql
CLEAN ALL PROFILE;
```

## 举例

1. 清理所有 profile 信息。

	```sql
	CLEAN ALL PROFILE;
	```

### Keywords

    CLEAN, PROFILE
