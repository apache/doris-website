日期类型包括 DATE 和 DATETIME，DATE 类型只存储日期精确到天，DATETIME 类型存储日期和时间，可以精确到微秒。

更多信息参考 [DATE](../datetime/DATE.md)、[DATETIME](../datetime/DATETIME.md) 文档。
