## 描述

计算 SM3 256-bit

## 语法

```sql
SM3( <input> )
```

## 参数

| 参数      | 说明         |
|---------|------------|
| `<input>` | 需要被计算 sm3 的值, 接受字符串和二进制类型 |

## 返回值
返回输入字符串的 sm3 值

## 示例

```sql
-- vb (VarBinary) 和 vc (VarChar) 插入时使用了相同的字符串.
SELECT * FROM mysql_catalog.binary_test.binary_test;
```
```text
+------+------------+------+
| id   | vb         | vc   |
+------+------------+------+
|    1 | 0x616263   | abc  |
|    2 | 0x78797A   | xyz  |
|    3 | NULL       | NULL |
+------+------------+------+
```

```sql
SELECT SM3(vb), SM3(vc) FROM mysql_catalog.binary_test.binary_test;
```
