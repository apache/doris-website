## 描述

用于返回一个随机的 uuid 字符串

## 语法

```sql
UUID()
```
## 返回值

返回一个随机的 uuid 字符串

## 示例

```sql
select UUID();
```

```sql
+--------------------------------------+
| UUID()                               |
+--------------------------------------+
| 29077778-fc5e-4603-8368-6b5f8fd55c24 |
+--------------------------------------+

```
