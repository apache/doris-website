## 描述

对于输入的 uuid 字符串，返回一个 int128 表示。

## 语法

```sql
UUID_TO_INT( <uuid> )
```
## 必选参数

| 参数 | 描述 |
|------|------|
| `<uuid>` | 待解码的字符串 |


## 返回值
返回一个 int128 表示

## 示例

```sql
select uuid_to_int("6ce4766f-6783-4b30-b357-bba1c7600348");
```

```sql
+-----------------------------------------------------+
| uuid_to_int('6ce4766f-6783-4b30-b357-bba1c7600348') |
+-----------------------------------------------------+
| 95721955514869408091759290071393952876              |
+-----------------------------------------------------+
```
