## 请求路径

`POST /api/pad_rowset?tablet_id={int}&start_version={int}&end_version={int}`

## 描述

该功能用于使用一个空的 rowset 填充损坏的副本。

## 请求参数

* `tablet_id`
    table 的 id

* `start_version`
    起始版本

* `end_version`
    终止版本       


## 请求体

无

## 响应

    ```json
    {
        msg: "OK",
        code: 0
    }
    ```
## 示例


    ```shell
    curl -X POST "http://127.0.0.1:8040/api/pad_rowset?tablet_id=123456&start_version=1111111&end_version=1111112"

    ```
