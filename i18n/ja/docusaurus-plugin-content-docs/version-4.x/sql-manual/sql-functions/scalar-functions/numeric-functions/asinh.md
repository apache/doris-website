## デスクリプション

`x`の双曲線逆正弦を返します。

## Syntax

```sql
ASINH(<x>)
```
## パラメータ

| パラメータ | 説明 |
| -- | -- |
| `<x>` | 双曲線逆正弦値を計算する値 |

## 戻り値

パラメータ `x` のasinh値。

## 特殊なケース
- `x`がNaNの場合、NaNを返す
- `x`が正の無限大の場合、Infinityを返す
- `x`が負の無限大の場合、-Infinityを返す
- `x`が+0.0の場合、0を返す
- `x`が-0.0の場合、-0を返す
- 非常に大きな大きさの値はInfinity/-Infinityにオーバーフローする可能性がある（例：±1e308）
- `x`がNULLの場合、NULLを返す

## 例

```sql
select asinh(0.0);
```
```sql
+------------+
| asinh(0.0) |
+------------+
|          0 |
+------------+
```
```sql
select asinh(1.0);
```
```sql
+-------------------+
| asinh(1.0)        |
+-------------------+
| 0.881373587019543 |
+-------------------+
```
```sql
select asinh(-1.0);
```
```sql
+--------------------+
| asinh(-1.0)        |
+--------------------+
| -0.881373587019543 |
+--------------------+
```
```sql
select asinh(cast('nan' as double));
```
```sql
+------------------------------+
| asinh(cast('nan' AS DOUBLE)) |
+------------------------------+
| NaN                          |
+------------------------------+
```
```sql
select asinh(cast('inf' as double));
```
```sql
+------------------------------+
| asinh(cast('inf' AS DOUBLE)) |
+------------------------------+
| Infinity                     |
+------------------------------+
```
```sql
select asinh(cast('-inf' as double));
```
```sql
+-------------------------------+
| asinh(cast('-inf' AS DOUBLE)) |
+-------------------------------+
| -Infinity                     |
+-------------------------------+
```
