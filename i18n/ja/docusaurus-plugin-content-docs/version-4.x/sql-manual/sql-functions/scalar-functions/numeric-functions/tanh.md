## 説明

xの双曲線正接を返します。

## 構文

```sql
TANH(<x>)
```
## パラメータ

| Parameter | デスクリプション |
| -- | -- |
| `<x>` | 双曲線正接を計算する値 |

## 戻り値

パラメータxの双曲線正接。

## 特殊なケース
- `x`がNaNの場合、NaNを返す
- `x`が正の無限大の場合、1を返す
- `x`が負の無限大の場合、-1を返す
- `x`がNULLの場合、NULLを返す

## 例

```sql
select tanh(0),tanh(1);
```
```text
+-------------------------+-------------------------+
| tanh(cast(0 as DOUBLE)) | tanh(cast(1 as DOUBLE)) |
+-------------------------+-------------------------+
|                       0 |      0.7615941559557649 |
+-------------------------+-------------------------+
```
```sql
select tanh(cast('nan' as double));
```
```text
+-----------------------------+
| tanh(cast('nan' AS DOUBLE)) |
+-----------------------------+
| NaN                         |
+-----------------------------+
```
```sql
select tanh(cast('inf' as double));
```
```text
+-----------------------------+
| tanh(cast('inf' AS DOUBLE)) |
+-----------------------------+
| 1                           |
+-----------------------------+
```
```sql
select tanh(cast('-inf' as double));
```
```text
+------------------------------+
| tanh(cast('-inf' AS DOUBLE)) |
+------------------------------+
| -1                           |
+------------------------------+
```
