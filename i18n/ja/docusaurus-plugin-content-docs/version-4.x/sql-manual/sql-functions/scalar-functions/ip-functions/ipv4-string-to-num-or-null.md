## 説明
A.B.C.D形式（10進数をドット区切りにした形式）のIPv4アドレスを含む文字列を受け取ります。対応するIPv4アドレスをビッグエンディアンで表現したBIGINT数値を返します。

## エイリアス
- INET_ATON

## 構文

```sql
IPV4_STRING_TO_NUM_OR_NULL(<ipv4_string>)
```
## パラメータ
| Parameter | デスクリプション                                      |
|-----------|--------------------------------------------------|
| `<ipv4_string>`      | 'A.B.C.D'のようなipv4のString型  |

## Return Value
ビッグエンディアンで対応するIPv4アドレスを表すBIGINT数値を返します。
- 入力文字列が有効なIPv4アドレスでない場合、`NULL`が返されます。

## Example

```sql
select ipv4_string_to_num_or_null('192.168.0.1'); 
```
```text
+-------------------------------------------+ 
| ipv4_string_to_num_or_null('192.168.0.1') | 
+-------------------------------------------+ 
| 3232235521                                | 
+-------------------------------------------+ 
```
```sql
select str, ipv4_string_to_num_or_null(str) from ipv4_str; 
```
```text
+-----------------+---------------------------------+ 
|str              | ipv4_string_to_num_or_null(str) | 
+-----------------+---------------------------------+ 
| 0.0.0.0         | 0                               | 
| 127.0.0.1       | 2130706433                      | 
| 255.255.255.255 | 4294967295                      | 
| invalid         | NULL                            | 
+-----------------+---------------------------------+ 
```
