## デスクリプション

table関数で、タスクの一時tableを生成します。これにより、現在のDorisクラスター内のジョブによって生成されたタスクの情報を表示できます。

## Syntax

```sql
TASKS(
    "type"="<type>"
)
```
## 必須パラメータ
| フィールド         | 説明                                                                                       |
|---------------|---------------------------------------------------------------------------------------------------|
| **`<type>`**  | タスクのタイプ: <br/> `insert`: insert型タスク。 <br/> `mv`: マテリアライズドビュー型タスク。 |


## 戻り値

-  **`tasks("type"="insert")`** insert型タスクの戻り値

   | フィールド名   | 説明                        |
   |--------------|------------------------------------|
   | **TaskId**   | タスクid                            |
   | **JobId**    | ジョブid                             |
   | **JobName**  | ジョブ名                           |
   | **Label**    | ラベル                              |
   | **Status**   | タスクステータス                        |
   | **ErrorMsg** | タスク失敗情報          |
   | **CreateTime**| タスク作成時間                |
   | **FinishTime**| タスク完了時間              |
   | **TrackingUrl**| Tracking URL                     |
   | **LoadStatistic**| ロード統計                |
   | **User**     | ユーザー                               |

-  **`tasks("type"="mv")`** MV型タスクの戻り値

   | フィールド名            | 説明                                                                 |
   |-----------------------|-----------------------------------------------------------------------------|
   | **TaskId**            | タスクid                                                                     |
   | **JobId**             | ジョブid                                                                      |
   | **JobName**           | ジョブ名                                                                    |
   | **MvId**              | マテリアライズドビューID                                                        |
   | **MvName**            | マテリアライズドビュー名                                                      |
   | **MvDatabaseId**      | マテリアライズドビューのDB ID                                              |
   | **MvDatabaseName**    | マテリアライズドビューが属するデータベース名                 |
   | **Status**            | タスクステータス                                                                 |
   | **ErrorMsg**          | タスク失敗情報                                                   |
   | **CreateTime**        | タスク作成時間                                                          |
   | **StartTime**         | タスク実行開始時間                                                     |
   | **FinishTime**        | タスク実行終了時間                                                           |
   | **DurationMs**        | タスク実行時間                                                                |
   | **TaskContext**       | タスク実行パラメータ                                                     |
   | **RefreshMode**       | リフレッシュモード                                                                |
   | **NeedRefreshPartitions** | このタスクでリフレッシュが必要なパーティション情報       |
   | **CompletedPartitions** | このタスクでリフレッシュ済みのパーティション情報          |
   | **Progress**          | タスク実行進捗                                                       |
