## 説明

INオペレータは、値が値のリストまたはサブクエリのメンバーかどうかをテストします。

## オペレータの紹介

| オペレータ | 機能 | 例 |
| ------- | ------------------------------------------------ | ------------------------------ |
| IN     | いずれかのメンバーと等しいかテストします。そうであればTRUEを返します。 | `SELECT 1 IN (SELECT 2)` |
| NOT IN | すべてのメンバーと等しくないかテストします。そうであればTRUEを返します。 | `SELECT 1 NOT IN (3, 4, 5)` |
