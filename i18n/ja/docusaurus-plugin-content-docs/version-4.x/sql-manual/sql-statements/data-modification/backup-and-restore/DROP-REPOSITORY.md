## 説明

このステートメントは、作成されたリポジトリを削除するために使用されます。

## 構文

```sql
DROP REPOSITORY <repo_name>;
```
## 必須パラメータ
**<repo_name>**
> リポジトリの一意の名前。

## アクセス制御要件

| 権限               | オブジェクト                         | 備考                                               |
|:-------------------|:-----------------------------|:----------------------------------------------------|
| ADMIN_PRIV         | クラスタ全体の管理権限 | rootまたはスーパーユーザーのみがリポジトリを作成できます  |


## 使用上の注意
- リポジトリを削除すると、Doris内のマッピングのみが削除され、実際のリポジトリデータは削除されません。削除後、同じLOCATIONを指定することで、リポジトリを再度マッピングできます。

## 例

example_repoという名前のリポジトリを削除：

```sql
DROP REPOSITORY `example_repo`;
```
