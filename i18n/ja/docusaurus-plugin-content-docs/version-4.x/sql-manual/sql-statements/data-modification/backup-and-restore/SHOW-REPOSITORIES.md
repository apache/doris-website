## デスクリプション

この文は、現在作成されているwarehouseを表示するために使用されます

## Syntax

```sql
SHOW REPOSITORIES;
```
## 戻り値

| フィールド | 説明 |
|------------|------|
| **RepoId** | リポジトリの一意識別子（ID） |
| **RepoName** | リポジトリの名前 |
| **CreateTime** | リポジトリの作成時刻 |
| **IsReadOnly** | リポジトリが読み取り専用かどうか。`false`は読み取り専用でないことを意味し、`true`は読み取り専用であることを意味する |
| **Location** | リポジトリ内でデータのバックアップに使用されるルートディレクトリ |
| **Broker** | - |
| **タイプ** | リポジトリタイプ。現在S3とHDFSをサポート |
| **ErrMsg** | リポジトリのエラーメッセージ。エラーが発生しない場合は通常`NULL` |


## 例

作成されたリポジトリを表示：

```sql
SHOW REPOSITORIES;
```
```text
+--------+--------------+---------------------+------------+----------+--------+------+--------+
| RepoId | RepoName     | CreateTime          | IsReadOnly | Location | Broker | タイプ | ErrMsg |
+--------+--------------+---------------------+------------+----------+--------+------+--------+
| 43411  | example_repo | 2025-01-17 18:50:47 | false      | s3://rep1  | -      | S3   | NULL   |
+--------+--------------+---------------------+------------+----------+--------+------+--------+
```
