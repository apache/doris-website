## 説明

この文は FE ノードの基本的なステータス情報を表示するために使用されます。

## 構文

```sql
SHOW FRONTENDS
```
## 戻り値

| カラム                      | 説明                                                                                          |
|--------------------|-----------------------------------------------------------------------------------------------|
| Name               | Doris内の現在のFEの名前。この名前は通常、`fe`で始まるランダムな文字列です。 |
| Host               | 現在のFEのIPアドレスまたはホスト名。                                                |
| EditLogPort        | 現在のFEのbdbje通信ポート。                                               |
| HttpPort           | 現在のFEのhttp通信ポート。                                                                             |
| QueryPort          | 現在のFEのMySQLプロトコル通信ポート。                                                                          |
| RpcPort            | 現在のFEのthrift RPC通信ポート。                                                                       |
| ArrowFlightSqlPort | 現在のFEのArrowFlightプロトコル通信ポート。                                                                    |
| Role               | 現在のFEの役割。可能な値はFOLLOWERとOBSERVERです。                                                           |
| IsMaster           | 現在のFEがMasterとして選出されているかどうか。                                                                           |
| ClusterId          | 現在のDorisクラスターのID。通常はランダムに生成された番号です。                                                                  |
| Join               | 現在のFEノードが現在のDorisクラスターに正常に参加したかどうかを示すために使用されます。                                                                 |
| Alive              | 現在のFEが生きているかどうか。                                                                                    |
| ReplayedJournalId  | 現在のFEがリプレイした最大のメタデータログのID。                                                                         |
| LastStartTime      | 現在のFEが開始したタイムスタンプ。                                                                                  |
| LastHeartbeat      | 現在のFEが送信した最後の成功したハートビートのタイムスタンプ。                                                                           |
| IsHelper           | 現在のFEがbdbjeのhelperノードかどうか。                                                                  |
| ErrMsg             | 現在のFEのハートビートが失敗した際のエラーメッセージ。                                                                              |
| Version            | 現在のFEのバージョン情報。                                                                                   |
| CurrentConnected   | 現在のクライアント接続が現在のFEノードに接続されているかどうか。                                                                          |

## アクセス制御要件

このSQLを実行するユーザーは、少なくとも以下の権限を持つ必要があります：

| 権限  | オブジェクト | 備考 |
|------------|----|----|
| ADMIN_PRIV |    |    |

## 使用上の注意

クエリ結果をさらにフィルタリングする必要がある場合は、Table値関数[frontends()](../../../sql-functions/table-valued-functions/frontends.md)を使用できます。SHOW FRONTENDSは以下のステートメントと同等です：

```sql
SELECT * FROM FRONTENDS();
```
