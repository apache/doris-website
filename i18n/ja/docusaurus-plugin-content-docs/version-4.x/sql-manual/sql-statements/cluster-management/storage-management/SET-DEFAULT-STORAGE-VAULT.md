## 説明

このステートメントは、Dorisでデフォルトのストレージボールトを設定するために使用されます。デフォルトのストレージボールトは、内部tableまたはシステムtableのデータを格納するために使用されます。デフォルトのストレージボールトが設定されていない場合、Dorisは正常に動作できません。一度デフォルトのストレージボールトが設定されると、それを削除することはできません。


## 構文

```sql
SET <vault_name> AS DEFAULT STORAGE VAULT
```
## Required パラメータ

| パラメータ名          | デスクリプション                                                         |
|-------------------|--------------------------------------------------------------|
| `<vault_name>`    | ストレージvaultの名前。デフォルトストレージvaultとして設定したいvaultの一意識別子です。           |

## 使用上の注意:
> 1. ADMINユーザーのみがデフォルトストレージvaultを設定できます。

## 例

1. s3_vaultという名前のストレージvaultをデフォルトストレージvaultとして設定します。

   ```sql
   SET s3_vault AS DEFAULT STORAGE VAULT;
   ```
