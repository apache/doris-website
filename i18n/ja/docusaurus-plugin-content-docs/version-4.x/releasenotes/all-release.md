この文書では、1年以内にリリースされたApache Dorisバージョンの概要を、新しいものから順に掲載しています。

:::tip 最新リリース
🎉 Version 4.0.3がリリースされました。🔗[Release Notes](../releasenotes/v4.0/release-4.0.3)はこちらをご確認ください。Apache Doris 4.0は、新しいAIサポート（ベクター検索、AI Functions）とより優れた全文検索機能により大きく前進しました。これらのアップグレードにより、ユーザーはAIとエージェントの時代において先行し、企業が従来のBI分析からAI駆動のワークロードまで、あらゆるものを処理できるようになります。

<br />

🎉 Version 3.1.4がリリースされました。🔗[Release Notes](../releasenotes/v3.1/release-3.1.4)はこちらをご確認ください。Doris 3.1では、VARIANTデータタイプ用のスパースカラムとスキーマテンプレートを導入し、ログやJSONデータなどの動的フィールドを持つ大規模データセットの保存とクエリをより効率的に実行できるようになりました。レイクハウス機能については、非同期マテリアライズドビューを強化し、IcebergとPaimonのサポートを拡張して、データレイクとデータウェアハウス間のより強固なブリッジを構築します。

<br />

🎉 Version 3.0.8が現在リリースされました。🔗[Release Notes](../releasenotes/v3.0/release-3.0.8)はこちらをご確認ください。バージョン3.Xから、Apache Dorisはクラスターデプロイメント用に、コンピュート・ストレージ結合モードに加えて、コンピュート・ストレージ分離モードをサポートしています。計算レイヤーとストレージレイヤーを分離するクラウドネイティブアーキテクチャにより、ユーザーは複数のコンピュートクラスター間でクエリロードの物理的分離、および読み取りと書き込みロード間の分離を実現できます。

<br />

🎉 Version 2.1.11が現在リリースされました。🔗[Release Notes](../releasenotes/v2.1/release-2.1.11)はこちらをご確認ください。2.1バージョンは、TPC-DS 1TBテストで証明された100%高速な開箱即用クエリによる卓越したパフォーマンス、TrinoやSparkより4-6倍高速なデータレイク分析の強化、新しいVariant型と分析関数スイートによる半構造化データ分析の確実なサポート、クエリ高速化のための非同期マテリアライズドビュー、大規模でのリアルタイム書き込みの最適化、および安定性とランタイムSQLリソース追跡によるワークロード管理の改善を提供します。

:::
