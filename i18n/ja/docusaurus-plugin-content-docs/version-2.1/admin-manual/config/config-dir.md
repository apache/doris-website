# Config Dir

FEとBEの設定ファイルディレクトリは`conf/`です。このディレクトリは、デフォルトのfe.conf、be.confおよびその他のファイルを格納することに加えて、共通設定ファイルの格納ディレクトリとしても使用されます。

ユーザーはいくつかの設定ファイルをその中に格納でき、システムが自動的にそれらを読み取ります。

:::tip Tips
この機能はApache Doris 1.2バージョンからサポートされています
:::

## hdfs-site.xmlとhive-site.xml

Dorisの一部の機能では、HDFS上のデータにアクセスしたり、Hive metastoreにアクセスする必要があります。

対応する機能のステートメントで、様々なHDFS/Hiveパラメータを手動で入力することができます。

しかし、これらのパラメータは非常に多く、すべてを手動で入力するのは非常に面倒です。

そのため、ユーザーはHDFSまたはHiveの設定ファイルhdfs-site.xml/hive-site.xmlを直接`conf/`ディレクトリに配置できます。Dorisはこれらの設定ファイルを自動的に読み取ります。

ユーザーがコマンドで入力した設定は、設定ファイル内の設定項目を上書きします。

このように、ユーザーは少量の設定を入力するだけでHDFS/Hiveへのアクセスを完了できます。
