## 説明

比較条件は、ある式を別の式と比較します。比較の結果はTRUE、FALSE、またはUNKNOWNになります。

## オペレータ紹介

| オペレータ | 機能 | 例 |
| ------------------- | ----------------------------------------------------------- | ------------------- |
| `=` | 等価比較。比較のいずれかの側がUNKNOWNの場合、結果はUNKNOWNになります。 | `SELECT 1 = 1` |
| `<=>` | NULL安全等価比較。等価比較とは異なり、NULL安全等価はNULLを比較可能な値として扱います。両側がNULLの場合はTRUEを返します。片側のみがNULLの場合はFALSEを返します。このオペレータはUNKNOWNを返しません。 | `SELECT NULL <=> NULL` |
| `!=` `<>` | 不等価比較 | `SELECT 1 != 1` |
| `<` `>` | より大きいとより小さい比較 | `SELECT 1 > 1` |
| `<=` `>=` | 以上と以下の比較 | `SELECT 1 >= 1` |
| `<x> BETWEEN <y> AND <z>` | `<x> >= <y> and <x> <= <z>`と同等。`<y>`以上かつ`<z>`以下 | `SELECT 1 BETWEEN 0 AND 2` |
