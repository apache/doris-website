## 説明

データベース内で実行中の常駐データ同期ジョブを`job_name`を指定して停止します。停止すると、ジョブはデータの同期を停止し、占有していたリソースを解放します。

## 構文

```sql
STOP SYNC JOB [<db>.]<job_name>
```
## 必須パラメータ

**1. `<job_name>`**

> 停止するデータ同期ジョブの名前を指定します。

## オプションパラメータ
**1. `<db>`**
> `[<db>.]`プレフィックスを使用してデータベースが指定されている場合、ジョブはそのデータベース内で検索されます。指定されていない場合は、現在のデータベースが使用されます。

## アクセス制御要件

任意のユーザーまたはロールがこの操作を実行できます。

## 例

1. `job_name`という名前のデータ同期ジョブを停止する。

   ```sql
   STOP SYNC JOB `job_name`;
   ```
