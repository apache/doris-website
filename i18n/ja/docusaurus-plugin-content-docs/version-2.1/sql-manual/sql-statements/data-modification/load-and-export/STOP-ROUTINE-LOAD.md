## 説明

この構文はRoutine Loadジョブを停止するために使用されます。PAUSEコマンドとは異なり、停止されたジョブは再開できません。再度データをインポートする必要がある場合は、新しいインポートジョブを作成する必要があります。

## 構文

```sql
STOP ROUTINE LOAD FOR <job_name>;
```
## 必須パラメータ

**1. `<job_name>`**

> 停止するジョブの名前を指定します。以下の形式で指定できます：
>
> - `<job_name>`: 現在のデータベースで指定された名前のジョブを停止
> - `<db_name>.<job_name>`: 指定されたデータベースで指定された名前のジョブを停止

## アクセス制御要件

このSQLコマンドを実行するユーザーは、少なくとも以下の権限を持つ必要があります：

| 権限       | オブジェクト | 注記                                                     |
| :--------- | :----------- | :------------------------------------------------------- |
| LOAD_PRIV  | Table        | SHOW ROUTINE LOADにはテーブルに対するLOAD権限が必要です   |

## 注記

- 停止操作は元に戻せません。停止されたジョブはRESUMEコマンドを使用して再開することはできません
- 停止操作は即座に有効になり、実行中のタスクは中断されます
- ジョブを停止する前に、SHOW ROUTINE LOADコマンドを使用してジョブのステータスを確認することをお勧めします
- ジョブを一時的に停止したい場合は、代わりにPAUSEコマンドを使用してください

## 例

- test1という名前のroutine loadジョブを停止

   ```sql
   STOP ROUTINE LOAD FOR test1;
   ```
- 指定されたデータベースでルーチンロードジョブを停止する

   ```sql
   STOP ROUTINE LOAD FOR example_db.test1;
   ```
