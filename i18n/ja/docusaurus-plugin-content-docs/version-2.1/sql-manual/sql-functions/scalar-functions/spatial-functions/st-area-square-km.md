## 説明

地球の球面上の領域の面積を計算します


## 構文

```sql
ST_AREA_SQUARE_KM( <geo>)
```
## パラメータ

| パラメータ | 説明     |
| -- |--------|
| `<geo>` | 地球の球面位置 |

## 戻り値

ST_Area_Square_Km( <geo>):  返される単位は平方キロメートル

## 例

```sql
SELECT ST_Area_Square_Km(ST_Polygon("POLYGON ((0 0, 1 0, 1 1, 0 1, 0 0))"));
```
```text
+----------------------------------------------------------------------+
| st_area_square_km(st_polygon('POLYGON ((0 0, 1 0, 1 1, 0 1, 0 0))')) |
+----------------------------------------------------------------------+
|                                                   12364.036567076409 |
+----------------------------------------------------------------------+
```
