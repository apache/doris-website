## 説明

pointが有効なPOINT型の場合、対応するy座標値を返します

## 構文

```sql
ST_Y( <point>)
```
## パラメータ

| パラメータ | 説明 |
|------|----------|
| `<point>` | 二次元点の幾何座標 |

## 戻り値

幾何座標のY値

## 例

```sql
SELECT ST_Y(ST_Point(24.7, 56.7));
```
```text
+----------------------------+
| ST_Y(ST_Point(24.7, 56.7)) |
+----------------------------+
| 56.7                       |
+----------------------------+
```
