## 説明

複数の式を比較し、その中の最小値を返します。いずれかのパラメータが`NULL`の場合、この関数は`NULL`を返します。

## 構文

```sql
LEAST(<expr> [, ...])
```
## パラメータ

| パラメータ  | 説明 |
|------------|-------------|
| `<expr>` | 比較される式。サポートされる型には `TINYINT`、`SMALLINT`、`INT`、`BIGINT`、`LARGEINT`、`FLOAT`、`DOUBLE`、`STRING`、`DATETIME`、および `DECIMAL` が含まれます。 |

## 戻り値

- 指定された式の中で最小値を返します。
- いずれかのパラメータが `NULL` の場合、この関数は `NULL` を返します。

## 例

```sql
SELECT LEAST(-1, 0, 5, 8);
```
```text
+--------------------+
| LEAST(-1, 0, 5, 8) |
+--------------------+
|                 -1 |
+--------------------+
```
```sql
SELECT LEAST(-1, 0, 5, NULL);
```
```text
+-----------------------+
| LEAST(-1, 0, 5, NULL) |
+-----------------------+
| NULL                  |
+-----------------------+
```
```sql
SELECT LEAST(6.3, 4.29, 7.6876);
```
```text
+--------------------------+
| LEAST(6.3, 4.29, 7.6876) |
+--------------------------+
|                     4.29 |
+--------------------------+
```
```sql
SELECT LEAST('2022-02-26 20:02:11', '2020-01-23 20:02:11', '2020-06-22 20:02:11');
```
```text
+----------------------------------------------------------------------------+
| LEAST('2022-02-26 20:02:11', '2020-01-23 20:02:11', '2020-06-22 20:02:11') |
+----------------------------------------------------------------------------+
| 2020-01-23 20:02:11                                                        |
+----------------------------------------------------------------------------+
```
