## 説明

指定された入力日付の月の最終日の日付を返します。返される日は月によって異なります：
- 28日 - 平年の2月
- 29日 - うるう年の2月
- 30日 - 4月、6月、9月、11月
- 31日 - 1月、3月、5月、7月、8月、10月、12月

## 構文

```sql
LAST_DAY(<date>)
```
## パラメータ

| パラメータ | 説明                                      |
|-----------|--------------------------------------------------|
| `<date>` | 入力する日時値、型はDATETIMEまたはDATEを指定可能 |

## 戻り値

指定された入力日付の月の最終日を表すDATE型の値を返します。

## 例

```sql
SELECT LAST_DAY('2000-02-03');
```
```text
+-----------------------------------------------+
| last_day(cast('2000-02-03' as DATETIMEV2(0))) |
+-----------------------------------------------+
| 2000-02-29                                    |
+-----------------------------------------------+
```
