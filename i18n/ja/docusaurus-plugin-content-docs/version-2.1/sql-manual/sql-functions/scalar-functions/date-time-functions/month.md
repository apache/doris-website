## 説明

datetime値から月の値を抽出します。戻り値は1から12の範囲で、年の12か月を表します。

## 構文

```sql
MONTH(<date>)
```
## パラメータ

| パラメータ | 説明                                      |
|-----------|--------------------------------------------------|
| `<date>`  | 入力される日時の値。DATE、DATETIME、またはDATETIMEV2型を指定可能 |

## 戻り値

月の値を表すINT型を返します：
- 範囲：1から12
- 1は1月を表し、12は12月を表します。
- 入力がNULLの場合、関数はNULLを返します。

## 例

```sql
SELECT MONTH('1987-01-01');
```
```text
+--------------------------------------------+
| month(cast('1987-01-01' as DATETIMEV2(0))) |
+--------------------------------------------+
|                                          1 |
+--------------------------------------------+
```
