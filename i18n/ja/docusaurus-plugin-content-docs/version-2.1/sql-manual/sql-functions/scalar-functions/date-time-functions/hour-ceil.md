## 説明

日付を指定された時間間隔期間の最も近い切り上げタイムスタンプに変換します。

## 構文

```sql
HOUR_CEIL(<datetime>)
HOUR_CEIL(<datetime>, <origin>)
HOUR_CEIL(<datetime>, <period>)
HOUR_CEIL(<datetime>, <period>, <origin>)
```
## パラメータ

| パラメータ | 説明 |
| -- | -- |
| `<datetime>` | 有効な日付式 |
| `<period>`| 各期間を構成する時間数を指定します|
| `<origin>` | 時間の開始点。指定されていない場合、デフォルトは0001-01-01T00:00:00です |

## 戻り値

指定された時間間隔期間の最も近い切り上げられたタイムスタンプを返します。

## 例

```sql
select hour_ceil("2023-07-13 22:28:18", 5);
```
```text
+------------------------------------------------------------+
| hour_ceil(cast('2023-07-13 22:28:18' as DATETIMEV2(0)), 5) |
+------------------------------------------------------------+
| 2023-07-14 02:00:00                                        |
+------------------------------------------------------------+
```
