## Description

数値 `x` を指定された桁数に四捨五入します。
- `d` が指定されていない場合、`x` は最も近い整数に四捨五入されます。
- `d` が負の値の場合、結果は小数点の左側の `d` 桁で四捨五入されます。
- `x` または `d` が `null` の場合、`null` を返します。
- `d` が列で、最初の引数が `Decimal` 型の場合、結果の `Decimal` は入力の `Decimal` と同じ小数点以下の桁数を持ちます。


## Alias

- `DROUND`

## Syntax

```sql
ROUND(<x> [ , <d> ])
```
## パラメータ

| パラメータ | 説明 |
| -- | -- |
| `<x>` | 丸める数値 |
| `<d>` | オプション、丸める小数点以下の桁数 |

## 例

```sql
select round(2.4);
```
```text
+------------+
| round(2.4) |
+------------+
|          2 |
+------------+
```
```sql
select round(2.5);
```
```text
+------------+
| round(2.5) |
+------------+
|          3 |
+------------+
```
```sql
select round(-3.4);
```
```text
+-------------+
| round(-3.4) |
+-------------+
|          -3 |
+-------------+
```
```sql
select round(-3.5);
```
```text
+-------------+
| round(-3.5) |
+-------------+
|          -4 |
+-------------+
```
```sql
select round(1667.2725, 2);
```
```text
+---------------------+
| round(1667.2725, 2) |
+---------------------+
|             1667.27 |
+---------------------+
```
```sql
select round(1667.2725, -2);
```
```text
+----------------------+
| round(1667.2725, -2) |
+----------------------+
|                 1700 |
+----------------------+
```

## 使用上の注意
2.5は3に丸められます。2に丸めたい場合は、`round_bankers`関数を使用してください。
