## 説明

入力パラメータの基数変換を行います。

## 構文

```sql
CONV(<input>, <from_base>, <to_base>)
```
## パラメータ

| パラメータ | 説明 |
| -- | -- |
| `<input>` | 変換対象のパラメータ、文字列または整数 |
| `<from_base>` | 数値、変換元の基数、`[2,36]`の範囲内 |
| `<to_base>` | 数値、変換先の基数、`[2,36]`の範囲内 |

## 戻り値

変換された変換先進数`<to_base>`での数値が文字列として返されます。

## 例

```sql
SELECT CONV(15,10,2);
```
```text
+-----------------+
| conv(15, 10, 2) |
+-----------------+
| 1111            |
+-----------------+
```
```sql
SELECT CONV('ff',16,10);
```
```text
+--------------------+
| conv('ff', 16, 10) |
+--------------------+
| 255                |
+--------------------+
```
```sql
SELECT CONV(230,10,16);
```
```text
+-------------------+
| conv(230, 10, 16) |
+-------------------+
| E6                |
+-------------------+
```
