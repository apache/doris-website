## 説明

'y' / 'x' の逆正接を返します。

## 構文

```sql
ATAN2(<y>, <x>)
```
## パラメータ

| パラメータ | 説明 |
| -- | -- |
| `<x>` | 原点(0,0)からx軸に沿った水平距離（座標）を表す値。 |
| `<y>` | 原点(0,0)からy軸に沿った垂直距離（座標）を表す値。 |

## 戻り値

パラメータ`y` / `x`のatan2値。

## 例

```sql
select atan2(0.1, 0.2);
```
```text
+---------------------+
| atan2(0.1, 0.2)     |
+---------------------+
| 0.46364760900080609 |
+---------------------+
```
```sql
select atan2(1.0, 1.0);
```
```text
+---------------------+
| atan2(1.0, 1.0)     |
+---------------------+
| 0.78539816339744828 |
+---------------------+
```
