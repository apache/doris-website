## 説明

ベース`b`を底とする`x`の対数を返します。

## 構文

```sql
LOG(<b>,<x>)
```
## パラメータ

| パラメータ | 説明 |
|-----------|------------|
| `<b>`     | 底は0より大きく、かつ正確に1.0であってはならない |
| `<x>`     | 真数は0より大きくなければならない |

## 戻り値

b浮動小数点数を返す。特殊なケース：

- aがNULLまたはxがNULLの場合、NULLを返す

## 例

```sql
select log(5,1);
```
```text
+---------------+
| log(5.0, 1.0) |
+---------------+
|             0 |
+---------------+
```
```sql
select log(3,20);
```
```text
+--------------------+
| log(3.0, 20.0)     |
+--------------------+
| 2.7268330278608417 |
+--------------------+
```
```sql
select log(2,65536);
```
```text
+-------------------+
| log(2.0, 65536.0) |
+-------------------+
|                16 |
+-------------------+
```
```sql
select log(5,NULL);
```
```text
+------------------------------+
| log(cast(5 as DOUBLE), NULL) |
+------------------------------+
|                         NULL |
+------------------------------+
```
```sql
select log(NULL,3);
```
```text
+------------------------------+
| log(NULL, cast(3 as DOUBLE)) |
+------------------------------+
|                         NULL |
+------------------------------+
```
