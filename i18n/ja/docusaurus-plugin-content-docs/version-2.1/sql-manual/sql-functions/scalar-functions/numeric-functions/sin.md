## 説明

パラメータのサインを計算します

## 構文

```sql
SIN(<a>)
```
## パラメータ

| パラメータ | 説明 |
| -- | -- |
| `<a>` | 浮動小数点数、計算するパラメータのラジアン値 |

## 戻り値

パラメータ `<a>` の正弦値、ラジアンで表現されます。

## 例

```sql
select sin(1);
```
```text
+------------------------+
| sin(cast(1 as DOUBLE)) |
+------------------------+
|     0.8414709848078965 |
+------------------------+
```
```sql
select sin(0);
```
```text
+------------------------+
| sin(cast(0 as DOUBLE)) |
+------------------------+
|                    0.0 |
+------------------------+
```
```sql
select sin(Pi());
```
```text
+------------------------------------+
| sin(pi())                          |
+------------------------------------+
| 0.00000000000000012246467991473532 |
+------------------------------------+
```
