## 説明

RTRIM関数は、文字列の右側（末尾）から連続するスペースまたは指定された文字を削除するために使用されます。

## 構文

```sql
RTRIM( <str> [, <trim_chars> ] )
```
## パラメータ

| パラメータ      | 説明                                                                                                                                                                                                                                                                      |
|----------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `<str>`        | トリミングが必要な文字列。                                                                                                                                                                                                                                                             |
| `<trim_chars>` | オプションパラメータ。このパラメータが提供される場合、RTRIM関数は`<str>`の右側に現れる`<trim_chars>`内のすべての文字を削除します。このパラメータが提供されない場合、RTRIM関数は`<str>`の右側の空白文字のみを削除します。 |

## 戻り値

末尾の空白または`<trim_chars>`が削除された文字列を返します。特殊なケース：

- いずれかのパラメータがNULLの場合、NULLが返されます。


## 例

```sql
SELECT rtrim('ab d   ') str;
```
```text
+------+
| str  |
+------+
| ab d |
+------+
```
```sql
SELECT rtrim('ababccaab','ab') str;
```
```text
+---------+
| str     |
+---------+
| ababcca |
+---------+
```
