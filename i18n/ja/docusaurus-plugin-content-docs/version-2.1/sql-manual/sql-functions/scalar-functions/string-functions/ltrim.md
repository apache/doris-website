## 説明

LTRIM関数は、文字列の左側（先頭）から連続するスペースまたは指定された文字を削除するために使用されます。

## 構文

```sql
LTRIM( <str> [, <trim_chars> ] )
```
## パラメータ

| パラメータ      | 説明                                                                                                                                                                                                                                                                           |
|----------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `<str>`        | トリムが必要な文字列                                                                                                                                                                        |
| `<trim_chars>` | オプションパラメータ。このパラメータが指定された場合、LTRIM関数は`<str>`の左側に現れる`<trim_chars>`のすべての文字を削除します。このパラメータが指定されない場合、LTRIM関数は`'<str>'`の左側のスペース文字のみを削除します。 |

## 戻り値

先頭のスペースまたは`<trim_chars>`が削除された文字列を返します。特殊な場合:

- いずれかのパラメータがNULLの場合、NULLが返されます。

## 例

```sql
SELECT ltrim('   ab d') str;
```
```text
+------+
| str  |
+------+
| ab d |
+------+
```
```sql
SELECT ltrim('ababccaab','ab') str;
```
```text
+-------+
| str   |
+-------+
| ccaab |
+-------+
```
