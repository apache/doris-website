## 説明

REVERSE関数は、文字列内の文字の順序を逆にしたり、配列内の要素の順序を逆にしたりするために使用されます。

## 構文

```sql
REVERSE( <seq> )
```
## パラメータ

| パラメータ | 説明             |
|-----------|----------------|
| `<seq>`   | 順序を逆にする必要がある文字列または配列。 |

## 戻り値

順序が逆になった文字列または配列を返します。特殊なケース：

- いずれかのパラメータがNULLの場合、NULLが返されます。

## 例

```sql
SELECT reverse('hello');
```
```text
+------------------+
| REVERSE('hello') |
+------------------+
| olleh            |
+------------------+
```
```sql
SELECT reverse(['hello', 'world']);
```
```text
+-----------------------------+
| reverse(['hello', 'world']) |
+-----------------------------+
| ["world", "hello"]          |
+-----------------------------+
```
