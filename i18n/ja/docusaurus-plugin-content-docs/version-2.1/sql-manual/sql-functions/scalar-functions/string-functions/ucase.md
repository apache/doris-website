## 説明

文字列を大文字に変換するために使用されます

## 構文

```sql
UCASE( <str> )
UPPER( <str> )
```
## 必須パラメータ
| パラメータ | 説明 |
|------|------|
| `<str>` | 大文字に変換する文字列

## 戻り値

大文字に変換した後の値

## 例

```sql
SELECT UCASE("aBc123"),UPPER("aBc123");
```
```sql
+-----------------+-----------------+
| UCASE("aBc123") | UPPER("aBc123") |
+-----------------+-----------------+
| ABC123          | ABC123          |
+-----------------+-----------------+
```
```sql
SELECT UCASE("Кириллица"),UPPER("Бәйтерек");
```
```sql
+-----------------------------+---------------------------+
| UCASE("Кириллица")          | UPPER("Бәйтерек")         |
+-----------------------------+---------------------------+
| КИРИЛЛИЦА                   | БӘЙТЕРЕК                  |
+-----------------------------+---------------------------+
```
