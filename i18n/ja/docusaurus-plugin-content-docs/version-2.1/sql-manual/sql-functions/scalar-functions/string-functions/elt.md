## 説明

指定されたインデックスの文字列を返します。特殊なケース：

- 指定されたインデックスに文字列が存在しない場合、NULLが返されます。

## 構文

```sql
ELT ( <pos> , <str> [ , <str> ] )
```
## パラメータ

| パラメータ | 説明 |
|-----------|------------|
| `<pos>`   | 指定されたインデックス値 |
| `<str>`   | インデックス対象の文字列 |

## 戻り値

パラメータ `<str>` インデックス対象の文字列。特殊なケース：

- 指定されたインデックスに文字列が存在しない場合、NULLが返されます。

## 例

```sql
SELECT ELT(1, 'aaa', 'bbb'),ELT(2, 'aaa', 'bbb'), ELT(0, 'aaa', 'bbb'),ELT(2, 'aaa', 'bbb')
```
```text
+----------------------+----------------------+----------------------+----------------------+
| elt(1, 'aaa', 'bbb') | elt(2, 'aaa', 'bbb') | elt(0, 'aaa', 'bbb') | elt(2, 'aaa', 'bbb') |
+----------------------+----------------------+----------------------+----------------------+
| aaa                  | bbb                  | NULL                 | bbb                  |
+----------------------+----------------------+----------------------+----------------------+
```
