## 説明

配列内の要素数を数える

## エイリアス

- SIZE
- CARDINALITY

## 構文

```sql
ARRAY_SIZE(<arr>) 
```
## パラメータ

| パラメータ | 説明 |
|--|--|
| `<arr>` | 計算対象の配列 |

## 戻り値

配列内の要素数を返します。入力配列がNULLの場合、NULLを返します。

## 例

```sql
SELECT ARRAY_SIZE(['a', 'b', 'c']),ARRAY_SIZE([NULL]),ARRAY_SIZE([]);
```
```text
+------------------------------+---------------------+-----------------+
| cardinality(['a', 'b', 'c']) | cardinality([NULL]) | cardinality([]) |
+------------------------------+---------------------+-----------------+
|                            3 |                NULL |               0 |
+------------------------------+---------------------+-----------------+
```
