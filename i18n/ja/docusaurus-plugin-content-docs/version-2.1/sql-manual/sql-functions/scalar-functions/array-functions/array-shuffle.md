## 説明

配列内の要素をランダムに並べ替えます

## エイリアス

- SHUFFLE

## 構文

```sql
ARRAY_SHUFFLE(<array>, <seed>)
```
## Parameters

| Parameter | Description |
|--|--|
| `<array>` | ランダムに並び替えられる配列 |
| `<seed>` | 疑似乱数を生成するために使用される疑似乱数ジェネレータの初期値を設定するオプションパラメータ |

## Return Value

配列内の要素をランダム化します。パラメータarray1はランダムに配置される配列で、オプションパラメータseedは疑似乱数ジェネレータが疑似乱数を生成するために使用する初期値です。shuffleはarray_shuffleと同じ機能を持ちます。

## Example

```sql
SELECT ARRAY_SHUFFLE([1, 2, 3, 6]),ARRAY_SHUFFLE([1, 4, 3, 5, NULL],1);
```
```text
+-----------------------------+--------------------------------------+
| array_shuffle([1, 2, 3, 6]) | array_shuffle([1, 4, 3, 5, NULL], 1) |
+-----------------------------+--------------------------------------+
| [2, 6, 3, 1]                | [4, 1, 3, 5, null]                   |
+-----------------------------+--------------------------------------+
```
