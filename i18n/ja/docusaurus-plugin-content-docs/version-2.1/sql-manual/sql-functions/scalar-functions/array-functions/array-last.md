## 説明

ラムダbool式と配列を入力パラメータとして使用します。ラムダ式は他の入力ARRAY パラメータの内部データを評価するために使用されます。
lambda(arr1[i])が0以外の値を返す配列内の最後の要素を返します。

## 構文

```sql
ARRAY_LAST(<lambda>, <arr>)
```
## パラメータ

| Parameter | Description | 
| --- | --- |
| `<lambda>` | 入力パラメータが指定された配列の列数と一致する必要があるlambda式。この式は有効なスカラ関数を実行できますが、集約関数はサポートしていません。 |
| `<arr>` | ARRAY配列 |

## 戻り値

最後の非ゼロ値のインデックスを返します。そのようなインデックスが見つからない場合は、NULLを返します。

## 例

```sql
select array_last(x->x>2, [1,2,3,0]) ;
```
```text
+------------------------------------------------------------------------------------------------+
| array_last(array_filter(ARRAY(1, 2, 3, 0), array_map([x] -> x(0) > 2, ARRAY(1, 2, 3, 0))), -1) |
+------------------------------------------------------------------------------------------------+
|                                                                                              3 |
+------------------------------------------------------------------------------------------------+
```
```sql
select array_last(x->x>4, [1,2,3,0]) ; 
```
```text
+------------------------------------------------------------------------------------------------+
| array_last(array_filter(ARRAY(1, 2, 3, 0), array_map([x] -> x(0) > 4, ARRAY(1, 2, 3, 0))), -1) |
+------------------------------------------------------------------------------------------------+
|                                                                                           NULL |
+------------------------------------------------------------------------------------------------+
```
