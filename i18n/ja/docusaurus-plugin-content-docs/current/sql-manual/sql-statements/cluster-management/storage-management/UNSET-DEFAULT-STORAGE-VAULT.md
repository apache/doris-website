## 説明
指定されたデフォルトストレージボルトをキャンセルする

## 構文

```sql
UNSET DEFAULT STORAGE VAULT
```
## 権限制御

| Privilege  | Object        | Notes                                                      |
| :--------- | :------------ | :--------------------------------------------------------- |
| ADMIN_PRIV | Storage Vault | 管理者ユーザーのみがこのステートメントを実行する権限を持ちます |

## 例

```sql
UNSET DEFAULT STORAGE VAULT
```
