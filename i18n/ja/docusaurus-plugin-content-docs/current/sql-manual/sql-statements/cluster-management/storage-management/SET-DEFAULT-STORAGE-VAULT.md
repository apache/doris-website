## 説明

このステートメントは、Dorisでデフォルトのストレージボルトを設定するために使用されます。デフォルトのストレージボルトは、内部テーブルまたはシステムテーブルのデータを保存するために使用されます。デフォルトのストレージボルトが設定されていない場合、Dorisは正常に動作できません。デフォルトのストレージボルトが一度設定されると、削除することはできません。


## 構文

```sql
SET <vault_name> AS DEFAULT STORAGE VAULT
```
## 必須パラメータ

| パラメータ名          | 説明                                                         |
|-------------------|--------------------------------------------------------------|
| `<vault_name>`    | ストレージvaultの名前。デフォルトのストレージvaultとして設定したいvaultの一意識別子です。           |

## 使用上の注意:
> 1. ADMIN ユーザーのみがデフォルトのストレージvaultを設定できます。

## 例

1. s3_vault という名前のストレージvaultをデフォルトのストレージvaultとして設定する。

   ```sql
   SET s3_vault AS DEFAULT STORAGE VAULT;
   ```
