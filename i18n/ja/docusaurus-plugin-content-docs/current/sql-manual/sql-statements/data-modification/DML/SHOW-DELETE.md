## 説明

このステートメントは、正常に実行された過去の削除タスクを表示するために使用されます

## 構文

```sql
SHOW DELETE [FROM <db_name>]
```
## オプションパラメータ
**<db_name>** : 表示するデータベースの名前

## 戻り値
| Column            | Description                                                                      |
|-------------------|----------------------------------------------------------------------------------|
| `TableName`       | 削除操作が実行されたテーブルの名前。                   |
| `PartitionName`   | 削除操作によって影響を受けたパーティションの名前。                      |
| `CreateTime`      | 削除操作が発行されたときのタイムスタンプ。                              |
| `DeleteCondition` | 削除操作に使用された条件で、どの行が削除されたかを指定する。 |
| `State`           | 削除操作のステータス                                               |


## 例

1. testデータベースに移動してすべての履歴削除タスクを表示する

     ```sql
    show delete;
    ```
    ```text
    +---------------+---------------+---------------------+-----------------+----------+
    | TableName     | PartitionName | CreateTime          | DeleteCondition | State    |
    +---------------+---------------+---------------------+-----------------+----------+
    | iceberg_table | *             | 2025-03-14 15:53:32 | id EQ "1"       | FINISHED |
    +---------------+---------------+---------------------+-----------------+----------+
    ```
2. データベース database の全ての履歴削除タスクを表示する

     ```sql
     show delete from tpch;
     ```
     ```
     +-----------+---------------+---------------------+-------------------+----------+
     | TableName | PartitionName | CreateTime          | DeleteCondition   | State    |
     +-----------+---------------+---------------------+-------------------+----------+
     | customer  | *             | 2025-03-14 15:45:19 | c_custkey EQ "18" | FINISHED |
     +-----------+---------------+---------------------+-------------------+----------+
     ```
