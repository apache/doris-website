## 説明

IN演算子は、値が値のリストまたはサブクエリのメンバーであるかどうかをテストします。

## 演算子の紹介

| 演算子 | 機能 | 例 |
| ------- | ------------------------------------------------ | ------------------------------ |
| IN     | いずれかのメンバーと等しいかをテストします。等しい場合はTRUEを返します。 | `SELECT 1 IN (SELECT 2)` |
| NOT IN | すべてのメンバーと等しくないかをテストします。等しくない場合はTRUEを返します。 | `SELECT 1 NOT IN (3, 4, 5)` |
