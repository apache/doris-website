## 説明

xの正割を返します。xはラジアン単位の値で、入力と出力はdoubleのみサポートされています。null値を入力するとnull値が返されます。

## 構文

```sql
SEC(<x>)
```
## パラメータ

| パラメータ | 説明 |
| -- | -- |
| `<x>` | セカントを計算する値 |

## 戻り値

xのセカントを表すDouble型の値を返します。

## 特殊なケース
- `x`がNaNの場合、NaNを返します
- `x`が正または負の無限大の場合、NaNを返します
- `x`がNULLの場合、NULLを返します

## 例

```sql
select sec(1),sec(2),sec(1000);
```
```text
+--------------------+--------------------+--------------------+
| sec(1)             | sec(2)             | sec(1000)          |
+--------------------+--------------------+--------------------+
| 1.8508157176809255 | -2.402997961722381 | 1.7781600385912715 |
+--------------------+--------------------+--------------------+
```
null値を入力してください。

```sql
select sec(null);
```
```text
+--------------------+
| sec(null)          |
+--------------------+
|      NULL          |
+--------------------+
```
```sql
select sec(cast('nan' as double));
```
```text
+---------------------------+
| sec(cast('nan' AS DOUBLE))|
+---------------------------+
| NaN                       |
+---------------------------+
```
```sql
select sec(cast('inf' as double));
```
```text
+---------------------------+
| sec(cast('inf' AS DOUBLE))|
+---------------------------+
| NaN                       |
+---------------------------+
```
```sql
select sec(cast('-inf' as double));
```
```text
+----------------------------+
| sec(cast('-inf' AS DOUBLE))|
+----------------------------+
| NaN                        |
+----------------------------+
```
