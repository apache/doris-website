## 説明

パラメータのサインを計算します

## 構文

```sql
SIN(<a>)
```
## パラメータ

| パラメータ | 説明 |
| -- | -- |
| `<a>` | 浮動小数点数、計算するパラメータのラジアン値 |

## 戻り値

パラメータ`<a>`のサイン値、ラジアンで表現される。

## 特別なケース
- `a`がNaNの場合、NaNを返す
- `a`が正または負の無限大の場合、NaNを返す
- `a`がNULLの場合、NULLを返す

## 例

```sql
select sin(1);
```
```text
+------------------------+
| sin(cast(1 as DOUBLE)) |
+------------------------+
|     0.8414709848078965 |
+------------------------+
```
```sql
select sin(0);
```
```text
+------------------------+
| sin(cast(0 as DOUBLE)) |
+------------------------+
|                    0.0 |
+------------------------+
```
```sql
select sin(Pi());
```
```text
+------------------------------------+
| sin(pi())                          |
+------------------------------------+
| 0.00000000000000012246467991473532 |
+------------------------------------+
```
```sql
select sin(cast('nan' as double));
```
```text
+----------------------------+
| sin(cast('nan' AS DOUBLE)) |
+----------------------------+
| NaN                        |
+----------------------------+
```
```sql
select sin(cast('inf' as double));
```
```text
+----------------------------+
| sin(cast('inf' AS DOUBLE)) |
+----------------------------+
| NaN                        |
+----------------------------+
```
```sql
select sin(cast('-inf' as double));
```
```text
+-----------------------------+
| sin(cast('-inf' AS DOUBLE)) |
+-----------------------------+
| NaN                         |
+-----------------------------+
```
