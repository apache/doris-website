## 説明

実用的な意味はなく、MySQLプロトコルとの互換性のためのものです。

## 構文

```sql
VERSION()
```
## 戻り値

MySQLプロトコルと互換性があり、固定の戻り値は"5.7.99"です。

## 例

```sql
select version();
```
```text
+-----------+
| version() |
+-----------+
| 5.7.99    |
+-----------+
```
