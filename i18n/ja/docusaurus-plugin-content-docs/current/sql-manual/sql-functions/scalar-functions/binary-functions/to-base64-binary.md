## 説明

TO_BASE64_BINARY関数は、入力されたバイナリ文字列をBase64エンコード形式に変換します。Base64エンコードは、任意のバイナリデータを64文字で構成された文字列に変換します。

## 構文

```sql
TO_BASE64_BINARY(<bin>)
```
## パラメータ
| パラメータ | 説明                                    |
| --------- | ---------------------------------------------- |
| `<bin>` | Base64エンコードされるバイナリ。型: VARBINARY |

## 戻り値

VARCHAR型を返し、Base64エンコードされた文字列を表します。

特殊なケース:
- 入力がNULLの場合、NULLを返します
- 入力が空文字列の場合、空文字列を返します

## 例

1. 単一文字のエンコード

```sql
SELECT to_base64_binary(x'12');
```
```text
+-------------------------+
| to_base64_binary(x'12') |
+-------------------------+
| Eg==                    |
+-------------------------+
```
2. 複数文字エンコーディング

```sql
SELECT to_base64_binary(x'234AAA');
```
```text
+-----------------------------+
| to_base64_binary(x'234AAA') |
+-----------------------------+
| I0qq                        |
+-----------------------------+
```
