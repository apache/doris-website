## 説明

REVERSE関数は、入力シーケンスの順序を逆にするために使用されます。文字列引数の場合、文字の順序を逆にした文字列を返し、配列引数の場合、要素の順序を逆にした配列を返します。この関数は文字単位で逆順にし、UTF-8マルチバイト文字を正しく処理できます。

## 構文

```sql
REVERSE(<seq>)
```
## パラメータ

| パラメータ | 説明 |
|---------|----------------|
| `<seq>` | 反転する文字列または配列。型: VARCHAR または ARRAY |

## 戻り値

入力と同じ型の反転されたシーケンスを返します:
- 文字列入力: VARCHAR型を返し、文字順序が反転された文字列
- 配列入力: ARRAY型を返し、要素順序が反転された配列

特殊ケース:
- パラメータがNULLの場合、NULLを返します
- 文字列が空の場合、空文字列を返します
- 配列が空の場合、空配列を返します
- 単一文字の文字列は反転後も同じままです

## 例

1. 基本的な文字列の反転

```sql
SELECT REVERSE('hello');
```
```text
+------------------+
| REVERSE('hello') |
+------------------+
| olleh            |
+------------------+
```
2. 配列要素の反転

```sql
SELECT REVERSE(['hello', 'world']);
```
```text
+-----------------------------+
| REVERSE(['hello', 'world']) |
+-----------------------------+
| ["world", "hello"]          |
+-----------------------------+
```
3. NULL値の処理

```sql
SELECT REVERSE(NULL);
```
```text
+---------------+
| REVERSE(NULL) |
+---------------+
| NULL          |
+---------------+
```
4. 空文字列と空配列

```sql
SELECT REVERSE(''), REVERSE([]);
```
```text
+-------------+-------------+
| REVERSE('') | REVERSE([]) |
+-------------+-------------+
|             | []          |
+-------------+-------------+
```
5. 単一文字と単一要素

```sql
SELECT REVERSE('A'), REVERSE(['single']);
```
```text
+--------------+--------------------+
| REVERSE('A') | REVERSE(['single']) |
+--------------+--------------------+
| A            | ["single"]         |
+--------------+--------------------+
```
6. 数値と特殊文字

```sql
SELECT REVERSE('12345'), REVERSE('!@#$%');
```
```text
+------------------+------------------+
| REVERSE('12345') | REVERSE('!@#$%') |
+------------------+------------------+
| 54321            | %$#@!            |
+------------------+------------------+
```
7. UTF-8 マルチバイト文字

```sql
SELECT REVERSE('ṭṛì ḍḍumai'), REVERSE('ḍḍumannàri');
```
```text
+-----------------------+------------------------+
| REVERSE('ṭṛì ḍḍumai') | REVERSE('ḍḍumannàri') |
+-----------------------+------------------------+
| iamuḍḍ ìṛṭ            | irànnaumuḍḍ            |
+-----------------------+------------------------+
```
8. 混合文字タイプ

```sql
SELECT REVERSE('Hello123'), REVERSE('test@email.com');
```
```text
+---------------------+--------------------------+
| REVERSE('Hello123') | REVERSE('test@email.com') |
+---------------------+--------------------------+
| 321olleH            | moc.liame@tset           |
+---------------------+--------------------------+
```
9. 複数要素配列

```sql
SELECT REVERSE([1, 2, 3, 4, 5]), REVERSE(['a', 'b', 'c']);
```
```text
+---------------------------+------------------------+
| REVERSE([1, 2, 3, 4, 5])  | REVERSE(['a', 'b', 'c']) |
+---------------------------+------------------------+
| [5, 4, 3, 2, 1]           | ["c", "b", "a"]        |
+---------------------------+------------------------+
```
10. 回文テスト

```sql
SELECT REVERSE('level'), REVERSE('12321');
```
```text
+------------------+-------------------+
| REVERSE('level') | REVERSE('12321')  |
+------------------+-------------------+
| level            | 12321             |
+------------------+-------------------+
```
## 説明

REVERSE関数は、文字列内の文字の順序を逆にする、または配列内の要素の順序を逆にするために使用されます。

## 構文

```sql
REVERSE( <seq> )
```
## パラメータ

| パラメータ | 説明             |
|-----------|----------------|
| `<seq>`   | 順序を逆にする必要がある文字列または配列。 |

## 戻り値

順序が逆になった文字列または配列を返します。特殊なケース:

- いずれかのパラメータがNULLの場合、NULLが返されます。

## 例

```sql
SELECT reverse('hello');
```
```text
+------------------+
| REVERSE('hello') |
+------------------+
| olleh            |
+------------------+
```
```sql
SELECT reverse(['hello', 'world']);
```
```text
+-----------------------------+
| reverse(['hello', 'world']) |
+-----------------------------+
| ["world", "hello"]          |
+-----------------------------+
```
