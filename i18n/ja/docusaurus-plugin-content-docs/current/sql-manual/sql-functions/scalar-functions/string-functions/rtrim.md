## 説明

RTRIM関数は、文字列の右側（末尾）から連続するスペースまたは指定された文字列を削除するために使用されます。第2パラメータが指定されていない場合、末尾のスペースが削除されます。指定されている場合、この関数は右側から指定された完全な文字列を削除します。

## 構文

```sql
RTRIM(<str>[, <rhs>])
```
## パラメータ

| パラメータ | 説明 |
| --------- | ----------- |
| `<str>` | 処理対象の文字列。型: VARCHAR |
| `<rhs>` | オプションパラメータ、削除する末尾の文字。型: VARCHAR |

## 戻り値

VARCHAR型の値を返します。

特殊なケース:
- いずれかのパラメータがNULLの場合、NULLが返されます
- rhsが指定されていない場合、末尾のスペースを削除します
- rhsが指定されている場合、完全なrhs文字列を削除します（文字単位ではありません）

## 例

1. 末尾のスペースを削除

```sql
SELECT rtrim('ab d   ') str;
```
```text
+------+
| str  |
+------+
| ab d |
+------+
```
2. 指定された末尾文字列を削除する

```sql
SELECT rtrim('ababccaab', 'ab') str;
```
```text
+---------+
| str     |
+---------+
| ababcca |
+---------+
```
3. UTF-8文字サポート

```sql
SELECT rtrim('ṭṛì ḍḍumai   ');
```
```text
+---------------------------+
| rtrim('ṭṛì ḍḍumai   ')    |
+---------------------------+
| ṭṛì ḍḍumai                |
+---------------------------+
```
4. 一致するサフィックスがない場合、元の文字列を返す

```sql
SELECT rtrim('Hello World', 'xyz');
```
```text
+---------------------------------+
| rtrim('Hello World', 'xyz')     |
+---------------------------------+
| Hello World                     |
+---------------------------------+
```
5. NULL値の処理

```sql
SELECT rtrim(NULL), rtrim('Hello', NULL);
```
```text
+-------------+------------------------+
| rtrim(NULL) | rtrim('Hello', NULL)   |
+-------------+------------------------+
| NULL        | NULL                   |
+-------------+------------------------+
```
6. 空文字列の処理

```sql
SELECT rtrim(''), rtrim('abc', '');
```
```text
+-----------+-------------------+
| rtrim('') | rtrim('abc', '')  |
+-----------+-------------------+
|           | abc               |
+-----------+-------------------+
```
7. 重複する末尾パターンの除去

```sql
SELECT rtrim('abcabcabc', 'abc');
```
```text
+-------------------------------+
| rtrim('abcabcabc', 'abc')     |
+-------------------------------+
|                               |
+-------------------------------+
```
8. 複数の出現箇所がある場合、末尾のマッチのみを削除

```sql
SELECT rtrim('HelloHelloWorld', 'Hello');
```
```text
+---------------------------------------+
| rtrim('HelloHelloWorld', 'Hello')     |
+---------------------------------------+
| HelloHelloWorld                       |
+---------------------------------------+
```
### キーワード
