# TIME

## 説明
`TIME`関数はDatetime値の`time`部分を取得します。

## 構文

```sql
TIME(<datetime>)
```
## パラメータ

| パラメータ     | 説明                  |
|----------------|-----------------------|
| `<datetime>`   | datetime値。          |

## 戻り値
`TIME`型の値を返します

## 例

```sql
SELECT TIME('2025-1-1 12:12:12');
```
```text
mysql> 
+---------------------------+
| time('2025-1-1 12:12:12') |
+---------------------------+
| 12:12:12                  |
+---------------------------+
```
