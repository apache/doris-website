## 説明

同じサイズの2つのベクトルのスカラー積を計算します

## 構文

```sql
INNER_PRODUCT(<array1>, <array2>)
```
## パラメータ

| パラメータ | 説明 |
| -- |--|
| `<array1>` | 最初のベクトル。入力配列のサブタイプは TINYINT、SMALLINT、INT、BIGINT、LARGEINT、FLOAT、DOUBLE をサポートします。要素数は array2 と一致している必要があります |
| `<array1>` | 2番目のベクトル。入力配列のサブタイプは TINYINT、SMALLINT、INT、BIGINT、LARGEINT、FLOAT、DOUBLE をサポートします。要素数は array1 と一致している必要があります |

## 戻り値

同じサイズの2つのベクトルのスカラー積を返します。入力配列が NULL の場合、または配列内の任意の要素が NULL の場合は、NULL が返されます。

## 例

```sql
SELECT INNER_PRODUCT([1, 2], [2, 3]),INNER_PRODUCT([3, 6], [4, 7]);
```
```text
+-------------------------------+-------------------------------+
| inner_product([1, 2], [2, 3]) | inner_product([3, 6], [4, 7]) |
+-------------------------------+-------------------------------+
|                             8 |                            54 |
+-------------------------------+-------------------------------+
```
