# Statistics

Dorisは、Hive、Iceberg、Paimonなどの外部データソースからのtableに対する自動または手動の統計情報収集をサポートしています。統計情報の精度は、コスト推定の精度を直接決定し、これは最適なクエリプランの選択にとって重要です。これにより、特に複雑なクエリシナリオにおいて、クエリ実行効率を大幅に向上させることができます。

詳細については、「External table Collection」セクションのStatisticsドキュメントを参照してください。
