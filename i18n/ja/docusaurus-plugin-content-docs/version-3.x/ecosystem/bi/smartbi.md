## Smartbi 紹介

Smartbi は、Oracle、SQL Server、MySQL、Doris を含む様々なデータソースに接続できるソフトウェアサービスとアプリケーションコネクタのコレクションであり、ユーザーがデータを簡単に統合・クレンジングできるようにします。Smartbi のデータモデリング機能により、ユーザーは複雑なリレーショナルモデルを作成できるだけでなく、データ分析式を記述し、データ関係を確立することで、高度なデータ分析と可視化の基盤を築くことができます。SmartBI は、様々なタイプのチャート、地理的マップ、インタラクティブダッシュボード、カスタマイズ可能な可視化ツールを含む多様な可視化オプションを提供します。これらの強力なツールは、ユーザーがデータをより直感的かつ包括的に理解し、表現することを支援し、データ分析の効果と効率を向上させます。

## 前提条件

https://www.smartbi.info/download にアクセスして Smartbi をダウンロード・インストールできます。

## データ接続とアプリケーション

1. Smartbi にログインし、Data Connections をクリックします。

   ![main page](/images/bi-smartbi-en-1.png)

2. data connections をクリックすると、接続対象のデータベースリストが表示されるので、Doris database を選択します。

   ![selectdb](/images/bi-smartbi-en-2.png)

3. データベースを選択後、Doris database の接続情報を入力します

   ![data source connection](/images/bi-smartbi-en-3.png)

4. 情報が正しく、ネットワークが接続されている場合、Test passed を示すメッセージが表示されます。

   ![test passed](/images/bi-smartbi-en-4.png)

5. 接続が成功すると、データ分析モジュール Self-service dashboard で必要なレポート情報をカスタマイズ・設定できます。

   ![data analysis](/images/bi-smartbi-en-5.png)
