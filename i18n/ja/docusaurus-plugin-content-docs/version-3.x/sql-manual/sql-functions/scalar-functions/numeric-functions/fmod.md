## デスクリプション

浮動小数点型のa / bの剰余を求めます。整数型の場合は、mod関数を使用してください。

## Syntax

```sql
MOD(<col_a> , <col_b>)
```
## パラメータ

| Parameter | デスクリプション |
|-----------|------------|
| `<col_a>`   | 被除数 |
| `<col_b>`   | 除数は0であってはならない |

## 戻り値

浮動小数点型を返します。特殊なケース：

col_a IS NULL または col_b IS NULL の場合、NULLを返します。

## 例

```sql
select fmod(10.1, 3.2);
```
```text
+-----------------+
| fmod(10.1, 3.2) |
+-----------------+
|      0.50000024 |
+-----------------+
```
```sql
select fmod(10.1, 0);
```
```text
+---------------+
| fmod(10.1, 0) |
+---------------+
|          NULL |
+---------------+
```
