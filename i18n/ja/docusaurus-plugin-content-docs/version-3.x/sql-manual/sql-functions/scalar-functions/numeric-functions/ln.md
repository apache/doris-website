## 説明

`x`の自然対数を底`e`で返します。

## エイリアス

- DLOG1

## 構文

```sql
LN(<x>)
```
## パラメータ

| Parameter | デスクリプション |
|-----------|------------|
| `<x>`   | 逆対数は0より大きい値である必要があります |

## Return value

浮動小数点数を返します。特殊なケース：

- x が NULL の場合、NULL を返します

## Example

```sql
select ln(1);
```
```text
+-----------------------+
| ln(cast(1 as DOUBLE)) |
+-----------------------+
|                   0.0 |
+-----------------------+
```
```sql
select ln(e());
```
```text
+---------+
| ln(e()) |
+---------+
|     1.0 |
+---------+
```
```sql
select ln(10);
```
```text
+------------------------+
| ln(cast(10 as DOUBLE)) |
+------------------------+
|      2.302585092994046 |
+------------------------+
```
