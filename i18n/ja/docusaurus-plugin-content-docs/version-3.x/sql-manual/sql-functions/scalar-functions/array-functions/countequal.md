## 説明

配列内の値要素の数を決定します

## 構文

```sql
COUNTEQUAL(<arr>, <value>)
```
## パラメータ

| Parameter | デスクリプション |
|--|--|
| `<arr>` | 入力配列 |
| `<value>` | 判定要素 |

## Return Value

返される判定結果は以下の通りです：num: 配列内のvalueの数；0: valueが配列arr内に存在しない；NULL: 配列がNULLの場合。

## Example

```sql
SELECT COUNTEQUAL(NULL,1),COUNTEQUAL([1, 2, 3, 'c'],2),COUNTEQUAL([],'b');
```
```text
+---------------------+---------------------------------------------------+------------------------------------------+
| countequal(NULL, 1) | countequal(['1', '2', '3', 'c'], cast(2 as TEXT)) | countequal(cast([] as ARRAY<TEXT>), 'b') |
+---------------------+---------------------------------------------------+------------------------------------------+
|                NULL |                                                 1 |                                        0 |
+---------------------+---------------------------------------------------+------------------------------------------+
```
