## 説明

配列内の要素を昇順にソートします

## 構文

```sql
ARRAY_SORT(<arr>)
```
## パラメータ

| Parameter | デスクリプション |
|--|--|
| `<arr>` | 対応する配列 |

## Return Value

昇順でソートされた配列を返します。入力配列がNULLの場合、NULLを返します。配列要素にNULLが含まれている場合、出力されるソート済み配列ではNULLが最初に配置されます。

## Example

```sql
SELECT ARRAY_SORT([1, 2, 3, 6]),ARRAY_SORT([1, 4, 3, 5, NULL]),ARRAY_SORT([NULL]);
```
```text
+--------------------------+--------------------------------+--------------------+
| array_sort([1, 2, 3, 6]) | array_sort([1, 4, 3, 5, NULL]) | array_sort([NULL]) |
+--------------------------+--------------------------------+--------------------+
| [1, 2, 3, 6]             | [null, 1, 3, 4, 5]             | [null]             |
+--------------------------+--------------------------------+--------------------+
```
