## デスクリプション
日付計算関数。日付を日数値に変換するために使用されます。つまり、西暦0年12月31日（基準日）から指定された日付までの総日数を計算します。

## Syntax

```sql
TO_DAYS(<datetime_or_date_value>)
```
## 必須パラメータ
| Parameter                  | デスクリプション                       |
|----------------------------|-----------------------------------|
| `<datetime_or_date_value>` | `datetime` または `date` 型の日時 |


## 例

2007年10月7日からの日数を取得する

```sql
select to_days('2007-10-07');
```
```text
+---------------------------------------+
| to_days(cast('2007-10-07' as DATEV2)) |
+---------------------------------------+
|                                733321 |
+---------------------------------------+
```
```sql
select to_days('2007-10-07 10:03:09');
```
```text
+------------------------------------------------+
| to_days(cast('2007-10-07 10:03:09' as DATEV2)) |
+------------------------------------------------+
|                                         733321 |
+------------------------------------------------+
```
