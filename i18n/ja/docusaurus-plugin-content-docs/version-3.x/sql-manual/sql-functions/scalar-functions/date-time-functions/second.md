## 説明
この関数は、指定されたdatetime値の秒部分を返します。秒の範囲は0から59です。

## 構文

```sql
SECOND(<datetime>)
```
## パラメータ

| Parameter    | デスクリプション                                                        |
|--------------|--------------------------------------------------------------------|
| `<datetime>` | 入力する日付または日時の値。DATEまたはDATETIME型をサポートします。 |

## 戻り値
- 入力された日時値の秒部分を表す整数を返します。範囲は0から59です。
- 入力がNULLの場合、関数はNULLを返します。
- 入力が無効な日付（例：0000-00-00 00:00:00）の場合、関数はNULLを返します。

## 例

```sql
select second('2018-12-31 23:59:59');
```
```text
+---------------------------------------------+
| second(cast('2018-12-30' as DATETIMEV2(0))) |
+---------------------------------------------+
|                                           0 |
+---------------------------------------------+
```
