## デスクリプション

IPv6タイプのアドレスを受け取り、指定されたバイト数を削除したアドレスをテキスト形式で含む文字列を返します。


## Syntax

```sql
CUT_IPV6(<ipv6>, <cut_ipv6_bytes>, <cut_ipv4_bytes>)
```
## パラメータ
| Parameter | デスクリプション                                      |
|-----------|--------------------------------------------------|
| `<ipv6>`      | IPv6型のアドレス |
| `<cut_ipv6_bytes>`     | ipv6から切り取りたいバイト数         |
| `<cut_ipv4_bytes>`     | 最初のパラメータがipv4の場合、ipv4から切り取りたいバイト数           |

## Return Value
指定されたバイト数でipv6を切り取ったテキストの値を返します。

## Example

```sql
select cut_ipv6(to_ipv6('2001:0DB8:AC10:FE01:FEED:BABE:CAFE:F00D'), 10, 0);
```
```text
+---------------------------------------------------------------------+
| cut_ipv6(to_ipv6('2001:0DB8:AC10:FE01:FEED:BABE:CAFE:F00D'), 10, 0) |
+---------------------------------------------------------------------+
| 2001:db8:ac10::                                                     |
+---------------------------------------------------------------------+
```
