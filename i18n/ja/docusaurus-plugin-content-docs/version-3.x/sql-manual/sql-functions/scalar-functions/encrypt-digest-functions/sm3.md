## 説明

SM3 256ビット計算

## 構文

```sql
SM3( <str> )
```
## パラメータ


| parameter | description |
|-----------|-------------|
| `<str>`   | 計算する必要があるsm3の値  |

## Return Value
入力文字列のsm3値を返します

## Examples

```sql
select sm3("abcd");
```
```text
+------------------------------------------------------------------+
| sm3('abcd')                                                      |
+------------------------------------------------------------------+
| 82ec580fe6d36ae4f81cae3c73f4a5b3b5a09c943172dc9053c69fd8e18dca1e |
+------------------------------------------------------------------+
```
