## デスクリプション

SHA1アルゴリズムを使用して情報をダイジェストします。

## Alias
SHA

## Syntax

``` sql
SHA1( <str> )
```
## パラメータ

| parameter | description         |
|-----------|-------------|
| `<str>`   | 計算対象のsha1値 |

## 戻り値

入力文字列のsha1値を返します


## 例

```sql
select sha("123"), sha1("123");
```
```text
+------------------------------------------+------------------------------------------+
| sha1('123')                              | sha1('123')                              |
+------------------------------------------+------------------------------------------+
| 40bd001563085fc35165329ea1ff5c5ecbdbbeef | 40bd001563085fc35165329ea1ff5c5ecbdbbeef |
+------------------------------------------+------------------------------------------+
```
