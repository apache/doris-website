## STRING
### デスクリプション
STRING (M)
可変長文字列です。デフォルトサポートは1048576バイト（1M）で、最大2147483643バイト（2G）まで調整可能です。Stringタイプの長さは、beのstring_type_length_soft_limit_bytes設定（文字列タイプ長のソフトリミット）によっても制限されます。Stringタイプは値列でのみ使用でき、キー列、パーティション列、バケット列では使用できません。

注意：可変長文字列はUTF-8エンコーディングで格納されるため、通常英語文字は1バイト、中国語文字は3バイトを占有します。
