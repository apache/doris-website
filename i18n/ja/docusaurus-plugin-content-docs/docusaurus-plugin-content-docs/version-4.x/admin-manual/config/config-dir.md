# Config Dir

FEとBEの設定ファイルディレクトリは`conf/`です。このディレクトリは、デフォルトのfe.conf、be.confなどのファイルを格納することに加えて、共通の設定ファイル格納ディレクトリとしても使用されます。

ユーザーはその中にいくつかの設定ファイルを格納することができ、システムは自動的にそれらを読み取ります。

## hdfs-site.xmlとhive-site.xml

Dorisの一部の機能では、HDFS上のデータにアクセスしたり、Hive metastoreにアクセスしたりする必要があります。

機能の対応するステートメントで、様々なHDFS/Hiveパラメータを手動で記入することができます。

しかし、これらのパラメータは非常に多く、すべてを手動で記入するのは非常に面倒です。

そのため、ユーザーはHDFSまたはHiveの設定ファイルhdfs-site.xml/hive-site.xmlを直接`conf/`ディレクトリに配置することができます。Dorisはこれらの設定ファイルを自動的に読み取ります。

ユーザーがコマンドで記入した設定は、設定ファイル内の設定項目を上書きします。

このようにして、ユーザーは少量の設定を記入するだけで、HDFS/Hiveへのアクセスを完了することができます。
