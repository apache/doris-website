## Smartbi紹介

Smartbiは、Oracle、SQL Server、MySQL、Dorisなど様々なデータソースに接続できるソフトウェアサービスとアプリケーションコネクタのコレクションで、ユーザーが簡単にデータを統合し、クレンジングできるようにします。Smartbiのデータモデリング機能により、ユーザーは複雑なリレーショナルモデルを作成できるだけでなく、データ分析式を記述し、データ関係を確立することで、高度なデータ分析と可視化の基盤を築くことができます。SmartBIは、様々な種類のチャート、地理マップ、インタラクティブダッシュボード、カスタマイズ可能な可視化ツールなど、多様な可視化オプションを提供します。これらの強力なツールは、ユーザーがデータをより直感的かつ包括的に理解し、提示することを支援し、データ分析の効果と効率を向上させます。

## 前提条件

https://www.smartbi.info/download にアクセスしてSmartbiをダウンロードし、インストールできます。

## データ接続とアプリケーション

1. Smartbiにログインし、Data Connectionsをクリックします。

   ![main page](/images/bi-smartbi-en-1.png)

2. data connectionsをクリックすると、接続するデータベースのリストが表示され、Dorisデータベースを選択できます。

   ![selectdb](/images/bi-smartbi-en-2.png)

3. データベースを選択した後、Dorisデータベースの接続情報を入力します。

   ![data source connection](/images/bi-smartbi-en-3.png)

4. 情報が正しく、ネットワークが接続されている場合、Testが合格したことを示すメッセージが表示されます。

   ![test passed](/images/bi-smartbi-en-4.png)

5. 接続が成功した後、データ分析モジュールのSelf-service dashboardで必要なレポート情報をカスタマイズし、設定できます。

   ![data analysis](/images/bi-smartbi-en-5.png)
