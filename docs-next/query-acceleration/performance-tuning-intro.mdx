Apache Doris ships an adaptive optimizer and pipeline execution engine that serve most workloads well out of the box, but production systems still benefit from systematic tuning. Start with the tuning methodology to identify bottlenecks, then drill into the matching query- or load-side topic, and consult the optimizer internals when you need to understand *why* a plan looks the way it does.
