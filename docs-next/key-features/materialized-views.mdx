Materialized views help Doris answer repeated queries faster by storing precomputed results and letting the optimizer reuse them when possible.

## When to use

Use materialized views for workloads with stable query patterns, repeated aggregations, or expensive joins that are queried often.

## Key points

- Best results come from predictable query shapes.
- Refresh strategy affects freshness and cost.
- Query rewrite is the main acceleration mechanism.
