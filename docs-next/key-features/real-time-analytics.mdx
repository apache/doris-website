Apache Doris is designed for analytical workloads that need recent data to be queryable quickly. It is a good fit when events keep arriving and users still expect interactive response times.

## When to use

Use this pattern for monitoring, dashboards, and operational analytics where the value of data drops quickly over time.

## Key points

- Keep ingest paths short.
- Prefer query patterns that can tolerate continuously changing data.
- Combine streaming ingestion with low-latency query execution.
