Hybrid search helps you serve retrieval use cases that need both semantic matching and lexical matching. Doris can act as the unified query layer for this kind of workload.

## When to use

Use this feature for ranking, recommendation, and search experiences where neither pure vector search nor pure keyword search is enough.

## Key points

- Tune both retrieval modes together.
- Keep the index and scoring strategy aligned with your search quality goals.
- Treat search relevance as part of the analytical workload.
