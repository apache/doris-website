Multi-catalog federation lets Doris talk to multiple external data systems through catalog abstractions. That makes cross-system data access and management much simpler.

## When to use

Use it when you need to query external systems such as Hive or JDBC sources without moving everything into Doris first.

## Key points

- Catalog choice depends on the source system.
- Federation helps reduce data duplication.
- It is useful for hybrid data landscapes.
