Lakehouse analytics lets Doris query data that lives in external table formats and storage systems while still serving SQL analytics with low latency.

## When to use

Use it when your data is split between Doris and lakehouse systems, and you want a single SQL layer for exploration and reporting.

## Key points

- Keep data access patterns clear.
- Match the table format and catalog to the source system.
- Use Doris when query latency matters more than raw file scanning.
