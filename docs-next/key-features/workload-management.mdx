Workload management gives Doris a way to separate and control different types of query traffic. It helps keep important workloads stable when the cluster is busy.

## When to use

Use it when multiple teams or applications share the same cluster and need resource isolation or predictable service levels.

## Key points

- Useful for multi-tenant or mixed workloads.
- Resource controls should match business priorities.
- Isolation helps prevent noisy neighbors.
