# lower

# description

# Syntax

INT lower(VARCHAR str)

将参数中所有的字符串都转换成小写

# example

mysql> SELECT lower("AbC123");
+-----------------+
| lower('AbC123') |
+-----------------+
| abc123          |
+-----------------+
