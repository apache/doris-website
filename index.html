Core Capabilities

Three Pillars. One Engine

01 / 03

Real-Time Analytics

The fastest end-to-end engine from ingestion to insight.

  • Streaming IngestionIngest from Kafka and database CDC with second-level freshness.
  • Incremental TransformMaterialized views refresh complex transforms in minutes, not hours.
  • Sub-Second QueriesStay sub-second under high concurrency, even on petabyte-scale data.
Explore real-time analytics(Comming Soon)
High Concurrency / Low Latency
02 / 03

Lakehouse Analytics

Real-time analytics, applied directly to your open lakehouse.

  • Extensive Catalog SupportQuery Iceberg, Delta Lake, and Hudi tables in place, with no data copy.
  • Lightning-Fast Query EngineVectorized execution and smart metadata caching make lake queries fast.
  • Incremental Bi-directional SyncStream data both ways between open lake formats and Doris internal storage.
Explore lakehouse analytics(Comming Soon)
Iceberg / Delta Lake / Hudi
03 / 03

Hybrid Search

First-class search across structured, text, and vector data.

  • JSON AnalyticsAuto-columnar storage for agent traces and tool-call metadata, with 50x faster analysis.
  • Text SearchFast BM25 retrieval for RAG, agent memory, and observability logs.
  • Vector SearchBuilt-in HNSW and IVF indexes for semantic retrieval over embeddings.
Explore hybrid search(Comming Soon)
JSON / Text / Vector
