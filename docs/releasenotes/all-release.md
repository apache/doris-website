ðŸŽ‰ Version 3.0.8 released now. Check out the ðŸ”—[Release Notes](../releasenotes/v3.0/release-3.0.8) here. Starting from version 3.X, Apache Doris supports a compute-storage decoupled mode in addition to the compute-storage coupled mode for cluster deployment. With the cloud-native architecture that decouples the computation and storage layers, users can achieve physical isolation between query loads across multiple compute clusters, as well as isolation between read and write loads.
