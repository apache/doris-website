# Query Detail Action

## Request

`GET /api/query_detail`

## Description

Used to obtain information about all queries after a specified time point

## Path parameters

None

## Query parameters

* `event_time`

    At the specified time point (Unix timestamp, in milliseconds), obtain query information after that time point.

æ—
