Doris supports complex types such as Array, Map, Struct, and JSON.

Doris provides various functions specifically designed for these complex types.

For detailed information on the supported functions, please refer to the [SQL manual - Array functions](../sql-manual/sql-functions/scalar-functions/array-functions/array), [SQL manual - Map functions](../sql-manual/basic-element/sql-data-types/semi-structured/MAP) data type, [SQL manual - Struct functions](../sql-manual/sql-functions/scalar-functions/struct-functions/struct), and [SQL manual - Json functions](../sql-manual/sql-functions/scalar-functions/json-functions/json-parse).
