## Description

Returns the constant `Pi` value.

## Syntax

```sql
PI()
```

## Return value

Returns a constant Pi of type floating-point number.

## Example

```sql
select Pi();
```

```text
+-------------------+
| pi()              |
+-------------------+
| 3.141592653589793 |
+-------------------+
```
