## Description

Return the XOR value of two numbers.

## Syntax
`BOOLEAN xor BOOLEAN`

## Example

```sql
mysql >select true xor false,true xor true;
+------------------+-----------------+
| (TRUE XOR FALSE) | (TRUE XOR TRUE) |
+------------------+-----------------+
|                1 |               0 |
+------------------+-----------------+

```

## Keywords
	XOR
