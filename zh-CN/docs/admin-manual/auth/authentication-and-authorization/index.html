и®ӨиҜҒе’Ңйүҙжқғ

Doris зҡ„жқғйҷҗз®ЎзҗҶзі»з»ҹеҸӮз…§дәҶ Mysql зҡ„жқғйҷҗз®ЎзҗҶжңәеҲ¶пјҢеҒҡеҲ°дәҶиЎҢзә§еҲ«з»ҶзІ’еәҰзҡ„жқғйҷҗжҺ§еҲ¶пјҢеҹәдәҺи§’иүІзҡ„жқғйҷҗи®ҝй—®жҺ§еҲ¶пјҢе№¶дё”ж”ҜжҢҒзҷҪеҗҚеҚ•жңәеҲ¶гҖӮ

еҗҚиҜҚи§ЈйҮҠвҖӢ

  1. з”ЁжҲ·ж ҮиҜҶ User Identity

    еңЁжқғйҷҗзі»з»ҹдёӯпјҢдёҖдёӘз”ЁжҲ·иў«иҜҶеҲ«дёәдёҖдёӘ User IdentityпјҲз”ЁжҲ·ж ҮиҜҶпјүгҖӮз”ЁжҲ·ж ҮиҜҶз”ұдёӨйғЁеҲҶз»„жҲҗпјҡusername е’Ң hostгҖӮе…¶дёӯ username дёәз”ЁжҲ·еҗҚпјҢз”ұиӢұж–ҮеӨ§е°ҸеҶҷз»„жҲҗгҖӮhost иЎЁзӨәиҜҘз”ЁжҲ·й“ҫжҺҘжқҘиҮӘзҡ„ IPгҖӮUser Identity д»Ҙ username@'host' зҡ„ж–№ејҸе‘ҲзҺ°пјҢиЎЁзӨәжқҘиҮӘ host зҡ„ usernameгҖӮ

    User Identity зҡ„еҸҰдёҖз§ҚиЎЁзҺ°ж–№ејҸдёә username@['domain']пјҢе…¶дёӯ domain дёәеҹҹеҗҚпјҢеҸҜд»ҘйҖҡиҝҮ DNS и§ЈжһҗдёәдёҖз»„ IPгҖӮжңҖз»ҲиЎЁзҺ°дёәдёҖз»„ username@'host'пјҢжүҖд»ҘеҗҺйқўжҲ‘д»¬з»ҹдёҖдҪҝз”Ё username@'host' жқҘиЎЁзӨәгҖӮ

  2. жқғйҷҗ Privilege

    жқғйҷҗдҪңз”Ёзҡ„еҜ№иұЎжҳҜиҠӮзӮ№гҖҒж•°жҚ®зӣ®еҪ•гҖҒж•°жҚ®еә“жҲ–иЎЁгҖӮдёҚеҗҢзҡ„жқғйҷҗд»ЈиЎЁдёҚеҗҢзҡ„ж“ҚдҪңи®ёеҸҜгҖӮ

  3. и§’иүІ Role

    Doris еҸҜд»ҘеҲӣе»әиҮӘе®ҡд№үе‘ҪеҗҚзҡ„и§’иүІгҖӮи§’иүІеҸҜд»Ҙиў«зңӢеҒҡжҳҜдёҖз»„жқғйҷҗзҡ„йӣҶеҗҲгҖӮж–°еҲӣе»әзҡ„з”ЁжҲ·еҸҜд»Ҙиў«иөӢдәҲжҹҗдёҖи§’иүІпјҢеҲҷиҮӘеҠЁиў«иөӢдәҲиҜҘи§’иүІжүҖжӢҘжңүзҡ„жқғйҷҗгҖӮеҗҺз»ӯеҜ№и§’иүІзҡ„жқғйҷҗеҸҳжӣҙпјҢд№ҹдјҡдҪ“зҺ°еңЁжүҖжңүеұһдәҺиҜҘи§’иүІзҡ„з”ЁжҲ·жқғйҷҗдёҠгҖӮ

  4. з”ЁжҲ·еұһжҖ§ User Property

    з”ЁжҲ·еұһжҖ§зӣҙжҺҘйҷ„еұһдәҺжҹҗдёҖз”ЁжҲ·пјҢиҖҢдёҚжҳҜз”ЁжҲ·ж ҮиҜҶгҖӮеҚі user@'192.%' е’Ң user@['domain'] йғҪжӢҘжңүеҗҢдёҖз»„з”ЁжҲ·еұһжҖ§пјҢиҜҘеұһжҖ§еұһдәҺз”ЁжҲ· userпјҢиҖҢдёҚжҳҜ user@'192.%' жҲ– user@['domain']гҖӮ

    з”ЁжҲ·еұһжҖ§еҢ…жӢ¬дҪҶдёҚйҷҗдәҺпјҡз”ЁжҲ·жңҖеӨ§иҝһжҺҘж•°гҖҒеҜје…ҘйӣҶзҫӨй…ҚзҪ®зӯүзӯүгҖӮ

и®ӨиҜҒе’ҢйүҙжқғжЎҶжһ¶вҖӢ

з”ЁжҲ·зҷ»еҪ• Apache Doris зҡ„иҝҮзЁӢпјҢеҲҶдёәи®ӨиҜҒе’ҢйүҙжқғдёӨйғЁеҲҶгҖӮ

  • и®ӨиҜҒпјҡж №жҚ®з”ЁжҲ·жҸҗдҫӣзҡ„еҮӯжҚ®пјҲеҰӮз”ЁжҲ·еҗҚгҖҒе®ўжҲ· IPгҖҒеҜҶз ҒпјүзӯүпјҢиҝӣиЎҢиә«д»ҪйӘҢиҜҒгҖӮйӘҢиҜҒйҖҡиҝҮеҗҺпјҢдјҡе°Ҷз”ЁжҲ·дёӘдҪ“жҳ е°„еҲ°зі»з»ҹеҶ…зҡ„з”ЁжҲ·ж ҮиҜҶпјҲUser IdentityпјүдёҠгҖӮ
  • йүҙжқғпјҡеҹәдәҺиҺ·еҸ–еҲ°зҡ„з”ЁжҲ·ж ҮиҜҶпјҢж №жҚ®з”ЁжҲ·ж ҮиҜҶжүҖеҜ№еә”зҡ„жқғйҷҗпјҢжЈҖжҹҘз”ЁжҲ·жҳҜеҗҰжңүзӣёеә”ж“ҚдҪңзҡ„жқғйҷҗгҖӮ

и®ӨиҜҒвҖӢ

Doris ж”ҜжҢҒеҶ…зҪ®и®ӨиҜҒж–№жЎҲе’Ңд»ҘеҸҠ LDAP зҡ„и®ӨиҜҒж–№жЎҲгҖӮ

Doris еҶ…зҪ®и®ӨиҜҒж–№жЎҲвҖӢ

еҹәдәҺ Doris иҮӘиә«еӯҳеӮЁзҡ„з”ЁжҲ·еҗҚпјҢеҜҶз ҒзӯүдҝЎжҒҜжқҘи®ӨиҜҒгҖӮ

з®ЎзҗҶе‘ҳйҖҡиҝҮ CREATE USER е‘Ҫд»ӨжқҘеҲӣе»әз”ЁжҲ·пјҢйҖҡиҝҮ SHOW ALL GRANTS жқҘжҹҘзңӢеҲӣе»әзҡ„жүҖжңүз”ЁжҲ·гҖӮ

з”ЁжҲ·зҷ»еҪ•ж—¶пјҢдјҡеҲӨж–ӯз”ЁжҲ·еҗҚпјҢеҜҶз ҒеҸҠе®ўжҲ·з«Ҝзҡ„ IP ең°еқҖжҳҜеҗҰжӯЈзЎ®гҖӮ

еҜҶз Ғзӯ–з•ҘвҖӢ

Doris ж”ҜжҢҒд»ҘдёӢеҜҶз Ғзӯ–з•ҘпјҢеҸҜд»Ҙеё®еҠ©з”ЁжҲ·жӣҙеҘҪзҡ„иҝӣиЎҢеҜҶз Ғз®ЎзҗҶгҖӮ

  1. PASSWORD_HISTORY

    жҳҜеҗҰе…Ғи®ёеҪ“еүҚз”ЁжҲ·йҮҚзҪ®еҜҶз Ғж—¶дҪҝз”ЁеҺҶеҸІеҜҶз ҒгҖӮеҰӮ PASSWORD_HISTORY 10 иЎЁзӨәзҰҒжӯўдҪҝз”ЁиҝҮеҺ» 10 ж¬Ўи®ҫзҪ®иҝҮзҡ„еҜҶз Ғдёәж–°еҜҶз ҒгҖӮеҰӮжһңи®ҫзҪ®дёә PASSWORD_HISTORY DEFAULTпјҢеҲҷдјҡдҪҝз”Ёе…ЁеұҖеҸҳйҮҸ password_history дёӯзҡ„еҖјгҖӮ0 иЎЁзӨәдёҚеҗҜз”ЁиҝҷдёӘеҠҹиғҪгҖӮй»ҳи®Өдёә 0гҖӮ

    зӨәдҫӢпјҡ

    • и®ҫзҪ®е…ЁеұҖеҸҳйҮҸпјҡSET GLOBAL password_history = 10
    • дёәз”ЁжҲ·и®ҫзҪ®пјҡALTER USER user1@'ip' PASSWORD_HISTORY 10

  2. PASSWORD_EXPIRE

    и®ҫзҪ®еҪ“еүҚз”ЁжҲ·еҜҶз Ғзҡ„иҝҮжңҹж—¶й—ҙгҖӮеҰӮ PASSWORD_EXPIRE INTERVAL 10 DAY иЎЁзӨәеҜҶз ҒдјҡеңЁ 10 еӨ©еҗҺиҝҮжңҹгҖӮPASSWORD_EXPIRE NEVER иЎЁзӨәеҜҶз ҒдёҚиҝҮжңҹгҖӮеҰӮжһңи®ҫзҪ®дёә PASSWORD_EXPIRE DEFAULTпјҢеҲҷдјҡдҪҝз”Ёе…ЁеұҖеҸҳйҮҸ default_password_lifetime дёӯзҡ„еҖјпјҲеҚ•дҪҚдёә еӨ©пјүгҖӮй»ҳи®Өдёә NEVERпјҲжҲ– 0пјүпјҢиЎЁзӨәдёҚдјҡиҝҮжңҹгҖӮ

    зӨәдҫӢпјҡ

    • и®ҫзҪ®е…ЁеұҖеҸҳйҮҸпјҡSET GLOBAL default_password_lifetime = 1
    • дёәз”ЁжҲ·и®ҫзҪ®пјҡALTER USER user1@'ip' PASSWORD_EXPIRE INTERVAL 10 DAY

  3. FAILED_LOGIN_ATTEMPTS е’Ң PASSWORD_LOCK_TIME

    и®ҫзҪ®еҪ“еүҚз”ЁжҲ·зҷ»еҪ•ж—¶пјҢеҰӮжһңдҪҝз”Ёй”ҷиҜҜзҡ„еҜҶз Ғзҷ»еҪ• n ж¬ЎеҗҺпјҢиҙҰжҲ·е°Ҷиў«й”Ғе®ҡпјҢе№¶и®ҫзҪ®й”Ғе®ҡж—¶й—ҙгҖӮеҰӮ FAILED_LOGIN_ATTEMPTS 3 PASSWORD_LOCK_TIME 1 DAY иЎЁзӨәеҰӮжһң 3 ж¬Ўй”ҷиҜҜзҷ»еҪ•пјҢеҲҷиҙҰжҲ·дјҡиў«й”Ғе®ҡдёҖеӨ©гҖӮз®ЎзҗҶе‘ҳеҸҜд»ҘйҖҡиҝҮ ALTER USER иҜӯеҸҘдё»еҠЁи§Јй”Ғиў«й”Ғе®ҡзҡ„иҙҰжҲ·гҖӮ

    зӨәдҫӢпјҡ

    • дёәз”ЁжҲ·и®ҫзҪ®пјҡALTER USER user1@'ip' FAILED_LOGIN_ATTEMPTS 3 PASSWORD_LOCK_TIME 1 DAY

  4. еҜҶз ҒејәеәҰ

    иҜҘеҠҹиғҪз”ұе…ЁеұҖеҸҳйҮҸ validate_password_policy жҺ§еҲ¶гҖӮй»ҳи®Өдёә NONE/0пјҢеҚідёҚжЈҖжҹҘеҜҶз ҒејәеәҰгҖӮеҰӮжһңи®ҫзҪ®дёә STRONG/2пјҢеҲҷеҜҶз Ғеҝ…йЎ»еҢ…еҗ«вҖңеӨ§еҶҷеӯ—жҜҚвҖқпјҢвҖңе°ҸеҶҷеӯ—жҜҚвҖқпјҢвҖңж•°еӯ—вҖқе’ҢвҖңзү№ж®Ҡеӯ—з¬ҰвҖқдёӯзҡ„ 3 йЎ№пјҢе№¶дё”й•ҝеәҰеҝ…йЎ»еӨ§дәҺзӯүдәҺ 8гҖӮ

    зӨәдҫӢпјҡ

    Root з”ЁжҲ·й»ҳи®ӨжӢҘжңүиҜҘжқғйҷҗгҖӮеҗҢж—¶жӢҘжңү Grant_priv е’Ң Node_priv зҡ„з”ЁжҲ·пјҢеҸҜд»Ҙе°ҶиҜҘжқғйҷҗиөӢдәҲе…¶д»–з”ЁжҲ·гҖӮ

    иҜҘжқғйҷҗеҸӘиғҪиөӢдәҲ Global зә§еҲ«гҖӮ

  2. Grant_priv

    жқғйҷҗеҸҳжӣҙжқғйҷҗгҖӮе…Ғи®ёжү§иЎҢеҢ…жӢ¬жҺҲжқғгҖҒж’ӨжқғгҖҒж·»еҠ /еҲ йҷӨ/еҸҳжӣҙ з”ЁжҲ·/и§’иүІ зӯүж“ҚдҪңгҖӮ

    з»ҷе…¶д»–з”ЁжҲ·/и§’иүІжҺҲжқғж—¶пјҢ2.1.2 зүҲжң¬д№ӢеүҚпјҢеҪ“еүҚз”ЁжҲ·еҸӘйңҖиҰҒзӣёеә”еұӮзә§зҡ„ Grant_priv жқғйҷҗпјҢ2.1.2 зүҲжң¬д№ӢеҗҺеҪ“еүҚз”ЁжҲ·д№ҹиҰҒжңүжғіиҰҒжҺҲжқғзҡ„иө„жәҗзҡ„жқғйҷҗгҖӮ

  2. дёҚж”ҜжҢҒеҲ йҷӨжҲ–жӣҙж”№й»ҳи®ӨеҲӣе»әзҡ„и§’иүІжҲ–з”ЁжҲ·зҡ„жқғйҷҗгҖӮ

  3. operator и§’иүІзҡ„з”ЁжҲ·жңүдё”еҸӘжңүдёҖдёӘпјҢеҚі RootгҖӮadmin и§’иүІзҡ„з”ЁжҲ·еҸҜд»ҘеҲӣе»әеӨҡдёӘгҖӮ

  4. дёҖдәӣеҸҜиғҪдә§з”ҹеҶІзӘҒзҡ„ж“ҚдҪңиҜҙжҳҺ

    1. еҹҹеҗҚдёҺ ip еҶІзӘҒпјҡ

      еҒҮи®ҫеҲӣе»әдәҶеҰӮдёӢз”ЁжҲ·пјҡ

      CREATE USER user1@['domain'];

      е№¶дё”жҺҲжқғпјҡ

      GRANT SELECT_PRIV ON *.* TO user1@['domain']

      иҜҘ domain иў«и§ЈжһҗдёәдёӨдёӘ IPпјҡip1 е’Ң ip2гҖӮ

      еҒҮи®ҫд№ӢеҗҺпјҢжҲ‘д»¬еҜ№ user1@'ip1' иҝӣиЎҢдёҖж¬ЎеҚ•зӢ¬жҺҲжқғпјҡ

      GRANT ALTER_PRIV ON . TO user1@'ip1';

      еҲҷ user1@'ip1' зҡ„жқғйҷҗдјҡиў«дҝ®ж”№дёә Select_priv е’Ң Alter_privгҖӮе№¶дё”еҪ“жҲ‘д»¬еҶҚж¬ЎеҸҳжӣҙ user1@['domain'] зҡ„жқғйҷҗж—¶пјҢuser1@'ip1' д№ҹдёҚдјҡи·ҹйҡҸж”№еҸҳгҖӮ

    2. йҮҚеӨҚ ip еҶІзӘҒпјҡ

      еҒҮи®ҫеҲӣе»әдәҶеҰӮдёӢз”ЁжҲ·пјҡ

      CREATE USER user1@'%' IDENTIFIED BY "12345";
      CREATE USER user1@'192.%' IDENTIFIED BY "abcde";

      еңЁдјҳе…Ҳзә§дёҠпјҢ'192.%' дјҳе…ҲдәҺ '%'пјҢеӣ жӯӨпјҢеҪ“з”ЁжҲ· user1 д»Һ 192.168.1.1 иҝҷеҸ°жңәеҷЁе°қиҜ•дҪҝз”ЁеҜҶз Ғ '12345' зҷ»йҷҶ Doris дјҡиў«жӢ’з»қгҖӮ

  5. еҝҳи®°еҜҶз Ғ

    еҰӮжһңеҝҳи®°дәҶеҜҶз Ғж— жі•зҷ»йҷҶ DorisпјҢеҸҜд»ҘеңЁ FE зҡ„ config ж–Үд»¶дёӯж·»еҠ  skip_localhost_auth_check=true еҸӮж•°пјҢе№¶дё”йҮҚеҗҜ FEпјҢд»ҺиҖҢж— еҜҶз ҒеңЁжң¬жңәйҖҡиҝҮ root зҷ»йҷҶ DorisгҖӮ

    зҷ»йҷҶеҗҺпјҢеҸҜд»ҘйҖҡиҝҮ SET PASSWORD е‘Ҫд»ӨйҮҚзҪ®еҜҶз ҒгҖӮ

  6. д»»дҪ•з”ЁжҲ·йғҪдёҚиғҪйҮҚзҪ® root з”ЁжҲ·зҡ„еҜҶз ҒпјҢйҷӨдәҶ root з”ЁжҲ·иҮӘе·ұгҖӮ

  7. Admin_priv жқғйҷҗеҸӘиғҪеңЁ GLOBAL еұӮзә§жҺҲдәҲжҲ–ж’Өй”ҖгҖӮ

  8. current_user() е’Ң user()

    з”ЁжҲ·еҸҜд»ҘйҖҡиҝҮ SELECT current_user() е’Ң SELECT user() еҲҶеҲ«жҹҘзңӢ current_user е’Ң userгҖӮе…¶дёӯ current_user иЎЁзӨәеҪ“еүҚз”ЁжҲ·жҳҜд»Ҙе“Әз§Қиә«д»ҪйҖҡиҝҮи®ӨиҜҒзі»з»ҹзҡ„пјҢиҖҢ user еҲҷжҳҜз”ЁжҲ·еҪ“еүҚе®һйҷ…зҡ„ User IdentityгҖӮ

    дёҫдҫӢиҜҙжҳҺпјҡ

    еҒҮи®ҫеҲӣе»әдәҶ user1@'192.%' иҝҷдёӘз”ЁжҲ·пјҢз„¶еҗҺд»ҘдёәжқҘиҮӘ 192.168.10.1 зҡ„з”ЁжҲ· user1 зҷ»йҷҶдәҶзі»з»ҹпјҢеҲҷжӯӨж—¶зҡ„ current_user дёә user1@'192.%'пјҢиҖҢ user дёә user1@'192.168.10.1'гҖӮ

    жүҖжңүзҡ„жқғйҷҗйғҪжҳҜиөӢдәҲжҹҗдёҖдёӘ current_user зҡ„пјҢзңҹе®һз”ЁжҲ·жӢҘжңүеҜ№еә”зҡ„ current_user зҡ„жүҖжңүжқғйҷҗгҖӮ

жңҖдҪіе®һи·өвҖӢ

иҝҷйҮҢдёҫдҫӢдёҖдәӣ Doris жқғйҷҗзі»з»ҹзҡ„дҪҝз”ЁеңәжҷҜгҖӮ

  1. еңәжҷҜдёҖ

    Doris йӣҶзҫӨзҡ„дҪҝз”ЁиҖ…еҲҶдёәз®ЎзҗҶе‘ҳпјҲAdminпјүгҖҒејҖеҸ‘е·ҘзЁӢеёҲпјҲRDпјүе’Ңз”ЁжҲ·пјҲClientпјүгҖӮе…¶дёӯз®ЎзҗҶе‘ҳжӢҘжңүж•ҙдёӘйӣҶзҫӨзҡ„жүҖжңүжқғйҷҗпјҢдё»иҰҒиҙҹиҙЈйӣҶзҫӨзҡ„жҗӯе»әгҖҒиҠӮзӮ№з®ЎзҗҶзӯүгҖӮејҖеҸ‘е·ҘзЁӢеёҲиҙҹиҙЈдёҡеҠЎе»әжЁЎпјҢеҢ…жӢ¬е»әеә“е»әиЎЁгҖҒж•°жҚ®зҡ„еҜје…Ҙе’Ңдҝ®ж”№зӯүгҖӮз”ЁжҲ·и®ҝй—®дёҚеҗҢзҡ„ж•°жҚ®еә“е’ҢиЎЁжқҘиҺ·еҸ–ж•°жҚ®гҖӮ

    еңЁиҝҷз§ҚеңәжҷҜдёӢпјҢеҸҜд»Ҙдёәз®ЎзҗҶе‘ҳиөӢдәҲ ADMIN жқғйҷҗжҲ– GRANT жқғйҷҗгҖӮеҜ№ RD иөӢдәҲеҜ№д»»ж„ҸжҲ–жҢҮе®ҡж•°жҚ®еә“иЎЁзҡ„ CREATEгҖҒDROPгҖҒALTERгҖҒLOADгҖҒSELECT жқғйҷҗгҖӮеҜ№ Client иөӢдәҲеҜ№д»»ж„ҸжҲ–жҢҮе®ҡж•°жҚ®еә“иЎЁ SELECT жқғйҷҗгҖӮеҗҢж—¶пјҢд№ҹеҸҜд»ҘйҖҡиҝҮеҲӣе»әдёҚеҗҢзҡ„и§’иүІпјҢжқҘз®ҖеҢ–еҜ№еӨҡдёӘз”ЁжҲ·зҡ„жҺҲжқғж“ҚдҪңгҖӮ

  2. еңәжҷҜдәҢ

    дёҖдёӘйӣҶзҫӨеҶ…жңүеӨҡдёӘдёҡеҠЎпјҢжҜҸдёӘдёҡеҠЎеҸҜиғҪдҪҝз”ЁдёҖдёӘжҲ–еӨҡдёӘж•°жҚ®гҖӮжҜҸдёӘдёҡеҠЎйңҖиҰҒз®ЎзҗҶиҮӘе·ұзҡ„з”ЁжҲ·гҖӮеңЁиҝҷз§ҚеңәжҷҜдёӢгҖӮз®ЎзҗҶе‘ҳз”ЁжҲ·еҸҜд»ҘдёәжҜҸдёӘж•°жҚ®еә“еҲӣе»әдёҖдёӘжӢҘжңү DATABASE еұӮзә§ GRANT жқғйҷҗзҡ„з”ЁжҲ·гҖӮиҜҘз”ЁжҲ·д»…еҸҜд»ҘеҜ№з”ЁжҲ·иҝӣиЎҢжҢҮе®ҡзҡ„ж•°жҚ®еә“зҡ„жҺҲжқғгҖӮ

  3. й»‘еҗҚеҚ•

    Doris жң¬иә«дёҚж”ҜжҢҒй»‘еҗҚеҚ•пјҢеҸӘжңүзҷҪеҗҚеҚ•еҠҹиғҪпјҢдҪҶжҲ‘д»¬еҸҜд»ҘйҖҡиҝҮжҹҗдәӣж–№ејҸжқҘжЁЎжӢҹй»‘еҗҚеҚ•гҖӮеҒҮи®ҫе…ҲеҲӣе»әдәҶеҗҚдёә user@'192.%' зҡ„з”ЁжҲ·пјҢиЎЁзӨәе…Ғи®ёжқҘиҮӘ 192.* зҡ„з”ЁжҲ·зҷ»еҪ•гҖӮжӯӨж—¶еҰӮжһңжғізҰҒжӯўжқҘиҮӘ 192.168.10.1 зҡ„з”ЁжҲ·зҷ»еҪ•гҖӮеҲҷеҸҜд»ҘеҶҚеҲӣе»әдёҖдёӘз”ЁжҲ· cmy@'192.168.10.1' зҡ„з”ЁжҲ·пјҢе№¶и®ҫзҪ®дёҖдёӘж–°зҡ„еҜҶз ҒгҖӮеӣ дёә 192.168.10.1 зҡ„дјҳе…Ҳзә§й«ҳдәҺ 192.%пјҢжүҖд»ҘжқҘиҮӘ 192.168.10.1 е°ҶдёҚиғҪеҶҚдҪҝз”Ёж—§еҜҶз ҒиҝӣиЎҢзҷ»еҪ•гҖӮ
