## 説明

現在のユーザー名とそのIPルールホワイトリストを取得します。

## 構文

```sql
CURRENT_USER()
```
## 戻り値

現在のユーザー名とそのIP許可リストを返します。

形式:`<user_name>@<ip_white_list>`

## 例

- rootユーザー、IP制限なし

```sql
select current_user();
```
```text
+----------------+
| current_user() |
+----------------+
| 'root'@'%'     |
+----------------+
```
- dorisユーザー、IPホワイトリストは192.168.*

```sql
select current_user();
```
```text
+---------------------+
| current_user()      |
+---------------------+
| 'doris'@'192.168.%' |
+---------------------+
```
