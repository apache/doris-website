## 説明

指定された数のスペースで構成される文字列を生成します。

## 構文

```sql
SPACE ( <len> )
```
## パラメータ

| パラメータ | 説明                      |
|-----------|--------------------------|
| `<len>`   | 生成するスペースの数 |

## 戻り値

指定された数のスペースで構成される文字列を返します。特殊なケース：

- いずれかのパラメータがNULLの場合、NULLが返されます。
- `<len>`が0未満の場合、空文字列が返されます。

## 例

1. 指定された数のスペースを生成

```sql
SELECT space(5);
```
```text
+----------+
| space(5) |
+----------+
|          |
+----------+
```
2. ゼロまたは負の数の処理

```sql
SELECT space(0), space(-5);
```
```text
+----------+-----------+
| space(0) | space(-5) |
+----------+-----------+
|          |           |
+----------+-----------+
```
3. NULL値の処理

```sql
SELECT space(NULL);
```
```text
+-------------+
| space(NULL) |
+-------------+
| NULL        |
+-------------+
```
4. 他の関数との併用

```sql
SELECT CONCAT('Hello', space(3), 'World');
```
```text
+------------------------------------+
| concat('Hello', space(3), 'World') |
+------------------------------------+
| Hello   World                      |
+------------------------------------+
```
### キーワード
