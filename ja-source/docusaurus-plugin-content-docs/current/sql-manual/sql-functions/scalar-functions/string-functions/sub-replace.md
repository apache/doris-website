## 説明

`sub_replace`関数は文字列内の部分文字列を置換するために使用されます。置換対象の部分文字列と、それを置き換える対象文字列を指定できます。この関数は、`str`内の`start`から始まり長さ`len`の部分文字列を`new_str`で置き換えた新しい文字列を返します。`start`または`len`が負の整数の場合、NULLを返します。`len`のデフォルト値は`new_str`の長さです。

## 構文

```sql
sub_replace(<str>, <new_str>, [ ,<start> [ , <len> ] ])
```
## パラメータ

| パラメータ | 説明 |
| -- | -- |
| `<str>` | 置換が行われる対象の文字列 |
| `<new_str>` | 指定されたサブストリングを置き換える文字列 |
| `<start>` | `start`は置換操作が開始される位置で、文字列のどの位置から置換を開始するかを示す |
| `<len>` | `len`は置換されるサブストリングの長さを指定するオプションパラメータ |

## 戻り値

置換後の文字列を返す。

## 例

1. 基本的な使用法：位置と長さを指定した置換

```sql
SELECT sub_replace('doris', '***', 1, 2);
```
```text
+-----------------------------------+
| sub_replace('doris', '***', 1, 2) |
+-----------------------------------+
| d***is                            |
+-----------------------------------+
```
2. デフォルト長の置換を使用する

```sql
SELECT sub_replace('hello', 'Hi', 0);
```
```text
+--------------------------------+
| sub_replace('hello', 'Hi', 0)  |
+--------------------------------+
| Hillo                          |
+--------------------------------+
```
3. 負のパラメータはNULLを返す

```sql
SELECT sub_replace('hello', 'Hi', -1, 2);
```
```text
+------------------------------------+
| sub_replace('hello', 'Hi', -1, 2)  |
+------------------------------------+
| NULL                               |
+------------------------------------+
```
4. NULL値の処理

```sql
SELECT sub_replace(NULL, 'new', 0, 3);
```
```text
+-------------------------------------+
| sub_replace(NULL, 'new', 0, 3)      |
+-------------------------------------+
| NULL                                |
+-------------------------------------+
```
5. UTF-8文字列

```sql
SELECT sub_replace('doris', 'ṛìḍḍ', 1, 2);
```
```text
+-------------------------------------------+
| sub_replace('doris', 'ṛìḍḍ', 1, 2)        |
+-------------------------------------------+
| dṛìḍḍis                                   |
+-------------------------------------------+
```
6. 開始位置が文字列長を超えています

```sql
SELECT sub_replace('hello', 'Hi', 9, 2);
```
```text
+----------------------------------+
| sub_replace('hello', 'Hi', 9, 2) |
+----------------------------------+
| NULL                             |
+----------------------------------+
```
7. 指定された置換長が残りの文字列長を超えています

```sql
SELECT sub_replace('hello', 'Hi', 1, 9);
```
```text
+----------------------------------+
| sub_replace('hello', 'Hi', 1, 9) |
+----------------------------------+
| hHi                              |
+----------------------------------+
```
### キーワード

    SUB_REPLACE, REPLACE
