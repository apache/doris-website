## 説明

TRIM_IN関数は、文字列の先頭と末尾から指定された文字を削除するために使用されます。文字セットが指定されていない場合、デフォルトで先頭と末尾のスペースを削除します。文字セットが指定されている場合、両端から指定されたすべての文字を削除します（セット内での順序に関係なく）。

TRIM_INの主要な特徴は、指定されたセットから任意の文字の組み合わせを削除することです。一方、TRIM関数は完全な文字列マッチングに基づいて文字を削除します。

## 構文

```sql
TRIM_IN(<str>[, <rhs>])
```
## パラメータ

| パラメータ | 説明 |
| --------- | ------------------------------------------------------------------------ |
| `<str>` | 処理対象の文字列。型: VARCHAR |
| `<rhs>` | オプションパラメータ、削除対象の文字セット。型: VARCHAR |

## 戻り値

VARCHAR型を返し、処理された文字列を表します。

特殊なケース:
- strがNULLの場合、NULLを返す
- rhsが指定されていない場合、先頭と末尾のすべてのスペースを削除する
- rhsが指定されている場合、rhs内に含まれない文字に遭遇するまで、両端からrhs内に含まれるすべての文字を削除する

## 例

1. 先頭と末尾のスペースを削除

```sql
SELECT trim_in('   ab d   ') str;
```
```text
+------+
| str  |
+------+
| ab d |
+------+
```
2. 指定された文字セットを削除する

```sql
SELECT trim_in('ababccaab', 'ab') str;
```
```text
+------+
| str  |
+------+
| cc   |
+------+
```
3. TRIM関数との比較

```sql
SELECT trim_in('ababccaab', 'ab'), trim('ababccaab', 'ab');
```
```text
+-----------------------------+--------------------------+
| trim_in('ababccaab', 'ab')  | trim('ababccaab', 'ab')  |
+-----------------------------+--------------------------+
| cc                          | ababccaab                |
+-----------------------------+--------------------------+
```
4. 文字セットの順序は重要ではありません

```sql
SELECT trim_in('abcHelloabc', 'cba');
```
```text
+--------------------------------+
| trim_in('abcHelloabc', 'cba')  |
+--------------------------------+
| Hello                          |
+--------------------------------+
```
5. UTF-8文字サポート

```sql
SELECT trim_in('+++ṭṛì ḍḍumai+++', '+');
```
```text
+--------------------------------------+
| trim_in('+++ṭṛì ḍḍumai+++', '+')    |
+--------------------------------------+
| ṭṛì ḍḍumai                           |
+--------------------------------------+
```
6. NULL値の処理

```sql
SELECT trim_in(NULL, 'abc');
```
```text
+-----------------------+
| trim_in(NULL, 'abc')  |
+-----------------------+
| NULL                  |
+-----------------------+
```
7. 空文字の処理

```sql
SELECT trim_in('', 'abc'), trim_in('abc', '');
```
```text
+--------------------+--------------------+
| trim_in('', 'abc') | trim_in('abc', '') |
+--------------------+--------------------+
|                    | abc                |
+--------------------+--------------------+
```
### キーワード

    TRIM_IN, TRIM
