## 説明

入力されたエンコードされたLARGEINTについて、元のUUID文字列に変換します。

## 構文

```sql
INT_TO_UUID(<int128>)
```
## パラメータ

| パラメータ | 説明 |
|------------|-----------------|
| `<int128>` | エンコードされた LARGEINT 値 |

## 戻り値

パラメータ `<int128>` の元の UUID 文字列を返します。

- 入力が NULL の場合、NULL を返します

## 例

```sql
SELECT INT_TO_UUID(95721955514869408091759290071393952876)
```
```text
+-----------------------------------------------------+
| int_to_uuid(95721955514869408091759290071393952876) |
+-----------------------------------------------------+
| 6ce4766f-6783-4b30-b357-bba1c7600348                |
+-----------------------------------------------------+
```
```sql
SELECT INT_TO_UUID(NULL);
```
```text
+-------------------+
| INT_TO_UUID(NULL) |
+-------------------+
| NULL              |
+-------------------+
```
