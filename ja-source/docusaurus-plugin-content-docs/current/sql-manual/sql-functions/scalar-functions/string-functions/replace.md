## 説明

REPLACE関数は、文字列内の指定された部分文字列のすべての出現箇所を新しい部分文字列に置き換えるために使用されます。この関数は、文字列内の部分文字列の一致するすべてのインスタンスを置き換え、グローバル置換操作を実行します。

**REPLACE_EMPTY関数との違い:**
- `REPLACE()`は、空文字列を含む指定された部分文字列を置き換えます
- `REPLACE_EMPTY()`は、空の値または空文字列を指定された値に置き換えるために特別に使用されます

## 構文

```sql
REPLACE(<str>, <old>, <new>)
```
## パラメータ

| パラメータ | 説明 |
|-----------|-------------|
| `<str>` | 置換が実行される元の文字列。型: VARCHAR |
| `<old>` | 置換対象の部分文字列。strで見つからない場合、置換は実行されません。型: VARCHAR |
| `<new>` | oldを置換するために使用される新しい部分文字列。型: VARCHAR |

## 戻り値

置換後の新しい文字列を表すVARCHAR型を返します。

置換ルール:
- 文字列内の一致するすべてのold部分文字列を置換します
- 置換は大文字小文字を区別します
- oldが空文字列の場合、元の文字列を返します（操作は実行されません）
- newが空文字列の場合、一致するすべてのold部分文字列を効果的に削除します

特殊ケース:
- いずれかのパラメータがNULLの場合、NULLを返します
- strが空文字列の場合、空文字列を返します
- oldが空文字列の場合、元のstrを返します（置換なし）
- oldがstrで見つからない場合、元のstrを返します

## 例

1. 基本的な置換操作

```sql
SELECT REPLACE('hello world', 'world', 'universe');
```
```text
+---------------------------------------------+
| REPLACE('hello world', 'world', 'universe') |
+---------------------------------------------+
| hello universe                              |
+---------------------------------------------+
```
2. 複数のマッチを置換

```sql
SELECT REPLACE('apple apple apple', 'apple', 'orange');
```
```text
+------------------------------------------------+
| REPLACE('apple apple apple', 'apple', 'orange') |
+------------------------------------------------+
| orange orange orange                           |
+------------------------------------------------+
```
3. 部分文字列を削除する（空文字列で置換）

```sql
SELECT REPLACE('banana', 'a', '');
```
```text
+---------------------------+
| REPLACE('banana', 'a', '') |
+---------------------------+
| bnn                       |
+---------------------------+
```
4. NULL値の処理

```sql
SELECT REPLACE(NULL, 'old', 'new'), REPLACE('test', NULL, 'new'), REPLACE('test', 'old', NULL);
```
```text
+------------------------------+------------------------------+------------------------------+
| REPLACE(NULL, 'old', 'new')  | REPLACE('test', NULL, 'new') | REPLACE('test', 'old', NULL) |
+------------------------------+------------------------------+------------------------------+
| NULL                         | NULL                         | NULL                         |
+------------------------------+------------------------------+------------------------------+
```
5. UTF-8文字置換

```sql
SELECT REPLACE('ṭṛì ḍḍumai test ṭṛì ḍḍumannàri', 'ṭṛì', 'replaced');
```
```text
+-----------------------------------------------------------+
| REPLACE('ṭṛì ḍḍumai test ṭṛì ḍḍumannàri', 'ṭṛì', 'replaced') |
+-----------------------------------------------------------+
| replaced ḍḍumai test replaced ḍḍumannàri                  |
+-----------------------------------------------------------+
```
