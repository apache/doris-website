### 説明

CRC32を使用して結果を計算します。

### 構文

```sql
CRC32( <str> )
```
## パラメータ

| parameter | description |
| -- | -- |
| `<str>` | CRC計算に使用される値 |

## 戻り値

この文字列の巡回冗長検査値を返します。

## 例

```sql
select crc32("abc"),crc32("中国");
```
```text
+--------------+-----------------+
| crc32('abc') | crc32('中国')   |
+--------------+-----------------+
|    891568578 |       737014929 |
+--------------+-----------------+
```
