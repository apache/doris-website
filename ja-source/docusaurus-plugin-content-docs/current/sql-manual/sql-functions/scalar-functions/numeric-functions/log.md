## 説明

底`b`を基とした`x`の対数を返します。

## 構文

```sql
LOG(<b>[, <x>])
```
## パラメータ

| パラメータ | 説明 |
|-----------|------------|
| `<b>`     | 基数は0より大きく、かつ正確に1.0でない値である必要があります |
| `<x>`     | オプション、対数底は0より大きい値である必要があります、デフォルト値は e です |

## 戻り値

浮動小数点数を返します。特殊なケース：

- b が NULL または x が NULL の場合、NULL を返します
- いずれかのパラメータが NaN の場合、NaN を返します
- パラメータが入力制約を満たさない場合、NULL を返します

## 例

```sql
select log(5,1);
```
```text
+---------------+
| log(5.0, 1.0) |
+---------------+
|             0 |
+---------------+
```
```sql
select log(3),ln(3);
```
```text
+--------------------+--------------------+
| log(3)             | ln(3)              |
+--------------------+--------------------+
| 1.0986122886681098 | 1.0986122886681098 |
+--------------------+--------------------+
```
```sql
select log(3,20);
```
```text
+--------------------+
| log(3.0, 20.0)     |
+--------------------+
| 2.7268330278608417 |
+--------------------+
```
```sql
select log(2,65536);
```
```text
+-------------------+
| log(2.0, 65536.0) |
+-------------------+
|                16 |
+-------------------+
```
```sql
select log(5,NULL);
```
```text
+------------------------------+
| log(cast(5 as DOUBLE), NULL) |
+------------------------------+
|                         NULL |
+------------------------------+
```
```sql
select log(NULL,3);
```
```text
+------------------------------+
| log(NULL, cast(3 as DOUBLE)) |
+------------------------------+
|                         NULL |
+------------------------------+
```
```sql
select log(2 ,cast('nan' as double));
```
```text
+-------------------------------+
| log(2 ,cast('nan' as double)) |
+-------------------------------+
|                           NaN |
+-------------------------------+
```
```sql
select log(2 ,-1);
```
```text
+------------+
| log(2 ,-1) |
+------------+
|       NULL |
+------------+
```
