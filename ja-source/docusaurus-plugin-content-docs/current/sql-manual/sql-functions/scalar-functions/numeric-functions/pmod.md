## 説明

モジュラーシステム内でのモジュロ演算 x mod y の最小の正の解を返します。これは (x % y + y) % y を計算することで求められます。

## 構文

```sql
PMOD(<x> , <y>)
```
## パラメータ

| Parameter | Description |
|-----------|------------|
| `<x>` | 被除数 |
| `<y>` | 除数　0であってはいけません |

## 戻り値

整数または浮動小数点数を返します。特殊なケース：

- x = 0の場合、0を返します。
- xがNULLまたはyがNULLの場合、NULLを返します。

## 例

```sql
SELECT PMOD(13,5);
```
```text
+-------------+
| pmod(13, 5) |
+-------------+
|           3 |
+-------------+
```
```sql
SELECT PMOD(-13,5);
```
```text
+--------------+
| pmod(-13, 5) |
+--------------+
|            2 |
+--------------+
```
```sql
SELECT PMOD(0,-12);
```
```text
+--------------+
| pmod(0, -12) |
+--------------+
|            0 |
+--------------+
```
```sql
SELECT PMOD(0,null);
```
```text
+-------------------------------+
| pmod(cast(0 as DOUBLE), NULL) |
+-------------------------------+
|                          NULL |
+-------------------------------+
```
