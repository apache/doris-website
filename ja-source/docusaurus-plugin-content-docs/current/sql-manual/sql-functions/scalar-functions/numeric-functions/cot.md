## 説明

xの余接を返します。xはラジアン単位の値で、入力と出力はdoubleのみサポートされています。入力がnull値の場合、null値を返します。

## 構文

```sql
COT(<x>)
```
## パラメータ

| パラメータ | 説明 |
| -- | -- |
| `<x>` | コタンジェントを計算する値 |

## 戻り値

xのコタンジェントを意味するDouble型の値を返します。

## 特殊なケース
- `x`がNaNの場合、NaNを返します
- `x`が正または負の無限大の場合、NaNを返します
- `x`がNULLの場合、NULLを返します

## 例

```sql
select cot(1),cot(2),cot(1000);
```
```text
+--------------------+----------------------+--------------------+
| cot(1)             | cot(2)               | cot(1000)          |
+--------------------+----------------------+--------------------+
| 0.6420926159343306 | -0.45765755436028577 | 0.6801221323348698 |
+--------------------+----------------------+--------------------+
```
null値を入力してください。

```sql
select cot(null);
```
```text
+--------------------+
| cot(null)          |
+--------------------+
|      NULL          |
+--------------------+
```
```sql
select cot(cast('nan' as double));
```
```text
+----------------------------+
| cot(cast('nan' AS DOUBLE)) |
+----------------------------+
| NaN                        |
+----------------------------+
```
```sql
select cot(cast('inf' as double));
```
```text
+----------------------------+
| cot(cast('inf' AS DOUBLE)) |
+----------------------------+
| NaN                        |
+----------------------------+
```
```sql
select cot(cast('-inf' as double));
```
```text
+-----------------------------+
| cot(cast('-inf' AS DOUBLE)) |
+-----------------------------+
| NaN                         |
+-----------------------------+
```
