## 説明

`x`の逆双曲線余弦を返します。`x`が`1`未満の場合は`NULL`を返します。

## 構文

```sql
ACOSH(<x>)
```
## パラメータ

| パラメータ | 説明 |  
| -- | -- |  
| `<x>` | 双曲線逆余弦を計算する値 |  

## 戻り値  

パラメータ `x` の双曲線逆余弦値。

## 特殊ケース

- `x` が 1 の場合、0 を返す
- `x` が 1 未満の場合、`NULL` を返す
- `x` が NaN の場合、NaN を返す
- `x` が正の無限大の場合、Infinity を返す
- `x` が負の無限大の場合、`NULL` を返す
- `x` が NULL の場合、NULL を返す

## 例

```sql
select acosh(0.0);
```
```text
+------------+
| acosh(0.0) |
+------------+
|       NULL |
+------------+
```
```sql
select acosh(-1.0);
```
```text
+-------------+
| acosh(-1.0) |
+-------------+
|        NULL |
+-------------+
```
```sql
select acosh(1.0);
```
```text
+------------+
| acosh(1.0) |
+------------+
|          0 |
+------------+
```
```sql
select acosh(1.0000001);
```
```text
+-------------------------+
| acosh(1.0000001)        |
+-------------------------+
| 0.0004472135918947727   |
+-------------------------+
```
```sql
select acosh(cast('nan' as double));
```
```text
+----------------------------+
| acosh(cast('nan' AS DOUBLE)) |
+----------------------------+
| NaN                        |
+----------------------------+
```
```sql
select acosh(cast('inf' as double));
```
```text
+----------------------------+
| acosh(cast('inf' AS DOUBLE)) |
+----------------------------+
| Infinity                   |
+----------------------------+
```
```sql
select acosh(10.0);
```
```text
+-------------------+
| acosh(10.0)       |
+-------------------+
| 2.993222846126381 |
+-------------------+
```
