## 説明

xのコセカントを返します。ここでxはラジアン単位の値で、入力と出力はdoubleのみサポートされています。入力がnull値の場合、null値を返します。

## 構文

```sql
CSC(<x>)
```
## パラメータ

| Parameter | Description |
| -- | -- |
| `<x>` | cosecantを計算する値 |

## 戻り値

xのcosecantを意味するDouble型の値を返します。

## 特殊なケース
- `x`がNaNの場合、NaNを返します
- `x`が正または負の無限大の場合、NaNを返します
- `x`がNULLの場合、NULLを返します

## 例

```sql
select csc(1),csc(2),csc(1000);
```
```text
+--------------------+--------------------+------------------+
| csc(1)             | csc(2)             | csc(1000)        |
+--------------------+--------------------+------------------+
| 1.1883951057781212 | 1.0997501702946164 | 1.20936599707935 |
+--------------------+--------------------+------------------+
```
null値を入力してください。

```sql
select csc(null);
```
```text
+--------------------+
| csc(null)          |
+--------------------+
|      NULL          |
+--------------------+
```
```sql
select csc(cast('nan' as double));
```
```text
+---------------------------+
| csc(cast('nan' AS DOUBLE))|
+---------------------------+
| NaN                       |
+---------------------------+
```
```sql
select csc(cast('inf' as double));
```
```text
+---------------------------+
| csc(cast('inf' AS DOUBLE))|
+---------------------------+
| NaN                       |
+---------------------------+
```
```sql
select csc(cast('-inf' as double));
```
```text
+----------------------------+
| csc(cast('-inf' AS DOUBLE))|
+----------------------------+
| NaN                        |
+----------------------------+
```
