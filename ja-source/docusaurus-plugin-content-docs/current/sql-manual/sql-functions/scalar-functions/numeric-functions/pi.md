## 説明

定数`Pi`の値を返します。

## 構文

```sql
PI()
```
## 戻り値

浮動小数点数型の定数Piを返します。

## 例

```sql
select Pi();
```
```text
+-------------------+
| pi()              |
+-------------------+
| 3.141592653589793 |
+-------------------+
```
