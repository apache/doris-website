## 説明

`x`の自然対数（底`e`）を返します。

## エイリアス

- DLOG1

## 構文

```sql
LN(<x>)
```
## パラメータ

| パラメータ | 説明 |
|-----------|------------|
| `<x>`   | 真数は0より大きい必要があります |

## 戻り値

浮動小数点数を返します。特殊なケース：

- x が NULL の場合、NULL を返します
- x が NaN の場合、NaN を返します  
- x が正の無限大または負の無限大の場合、NULL を返します

## 例

```sql
select ln(1);
```
```text
+-----------------------+
| ln(cast(1 as DOUBLE)) |
+-----------------------+
|                   0.0 |
+-----------------------+
```
```sql
select ln(e());
```
```text
+---------+
| ln(e()) |
+---------+
|     1.0 |
+---------+
```
```sql
select ln(10);
```
```text
+------------------------+
| ln(cast(10 as DOUBLE)) |
+------------------------+
|      2.302585092994046 |
+------------------------+
```
```sql
select ln(cast('inf' as double));
```
```text
+---------------------------+
| ln(cast('inf' as double)) |
+---------------------------+
|                  Infinity |
+---------------------------+
```
```sql
select ln(cast('nan' as double));
```
```text
+---------------------------+
| ln(cast('nan' as double)) |
+---------------------------+
|                       NaN |
+---------------------------+
```
