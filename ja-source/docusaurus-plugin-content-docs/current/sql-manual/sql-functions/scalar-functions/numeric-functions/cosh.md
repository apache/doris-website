## 説明

`x`の双曲線余弦を返します。

## 構文

`COSH(<x>)`

## パラメータ

| パラメータ | 説明 |
| -- | -- |
| `<x>` | 双曲線余弦を計算する値 |

## 戻り値

パラメータ`x`の双曲線余弦

## 特殊なケース
- `x`がNaNの場合、NaNを返します
- `x`が正の無限大の場合、Infinityを返します
- `x`が負の無限大の場合、Infinityを返します
- `x`がNULLの場合、NULLを返します

## 例

```sql
select cosh(0);
```
```text
+-------------------------+
| cosh(cast(0 as DOUBLE)) |
+-------------------------+
|                     1.0 |
+-------------------------+
```
```sql
select cosh(1);
```
```text
+-------------------------+
| cosh(cast(1 as DOUBLE)) |
+-------------------------+
|       1.543080634815244 |
+-------------------------+
```
```sql
select cosh(-1);
```
```text
+--------------------------+
| cosh(cast(-1 as DOUBLE)) |
+--------------------------+
|        1.543080634815244 |
+--------------------------+
```
```sql
select cosh(cast('nan' as double));
```
```text
+-----------------------------+
| cosh(cast('nan' AS DOUBLE)) |
+-----------------------------+
| NaN                         |
+-----------------------------+
```
```sql
select cosh(cast('inf' as double));
```
```text
+-----------------------------+
| cosh(cast('inf' AS DOUBLE)) |
+-----------------------------+
| Infinity                    |
+-----------------------------+
```
```sql
select cosh(cast('-inf' as double));
```
```text
+------------------------------+
| cosh(cast('-inf' AS DOUBLE)) |
+------------------------------+
| Infinity                     |
+------------------------------+
```
