## 説明

値そのものを返します。

## 構文

```sql
POSITIVE(<x>)
```
## パラメータ

| パラメータ | 説明 |
|-----------|------------|
| `<x>`   | 返す必要がある値。 |

## 戻り値

整数または浮動小数点数を返します。特殊なケース：

- パラメータがNULLの場合、NULLを返します。

## 例

```sql
SELECT positive(-10);
```
```text
+---------------+
| positive(-10) |
+---------------+
|           -10 |
+---------------+
```
```sql
SELECT positive(10.111);
```
```text
+------------------+
| positive(10.111) |
+------------------+
|           10.111 |
+------------------+
```
```sql
SELECT positive(12);
```
```text
+--------------+
| positive(12) |
+--------------+
|           12 |
+--------------+
```
```sql
SELECT positive(null);
```
```text
+----------------+
| positive(NULL) |
+----------------+
|           NULL |
+----------------+
```
