### 関数

`ARRAY_SUM`関数は、配列内のすべての数値要素の合計を計算します。

## 構文

```SQL
ARRAY_SUM(ARRAY<T>)
```
## パラメータ

`ARRAY<T>`: **数値型**要素を含む配列。

## 戻り値

- 配列内のすべての非`NULL`要素の合計を返します。

    - すべての要素が`NULL`の場合、`NULL`を返します。

## 使用上の注意

1. 要素の合計には`+`演算子が使用されます。

2. `NULL`である要素は自動的に無視されます。

3. 配列に数値型以外の要素（文字列など）が含まれている場合、実行時エラーが発生します。

## 例

1. 簡単な例

    ```SQL
    SELECT ARRAY_SUM([1, 2, 3, 4]);
    +-------------------------+
    | ARRAY_SUM([1, 2, 3, 4]) |
    +-------------------------+
    |                      10 |
    +-------------------------+
    ```
2. 配列内の`NULL`値の処理

    ```SQL
    SELECT ARRAY_SUM([1, NULL, 3]); 
    +-------------------------+
    | ARRAY_SUM([1, NULL, 3]) |
    +-------------------------+
    |                       4 |
    +-------------------------+

    SELECT ARRAY_SUM(NULL);
    +-----------------+
    | ARRAY_SUM(NULL) |
    +-----------------+
    |            NULL |
    +-----------------+

    SELECT ARRAY_SUM([NULL, NULL]); 
    +-------------------------+
    | ARRAY_SUM([NULL, NULL]) |
    +-------------------------+
    |                    NULL |
    +-------------------------+
    ```
