## 説明

入力されたバイナリデータを16進エンコーディングを使用して変換された文字列に変換します。

## エイリアス

FROM_BINARY

## 構文

```sql
FROM_HEX ( <varbinary> )
```
## パラメータ

| パラメータ | 説明 |
|-------|--------------|
| `<varbinary>` | 入力パラメータはバイナリデータです |

## 戻り値

入力バイナリデータを16進エンコーディングを使用して文字列に変換します。

## 例

```sql
SELECT FROM_HEX(NULL);
```
```text
+----------------+
| FROM_HEX(NULL) |
+----------------+
| NULL           |
+----------------+
```
```sql
SELECT FROM_HEX(X'AB');
```
```text
+-----------------+
| FROM_HEX(X'AB') |
+-----------------+
| AB              |
+-----------------+
```
