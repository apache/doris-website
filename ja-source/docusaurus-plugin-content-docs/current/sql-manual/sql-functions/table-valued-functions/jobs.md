## 説明

一時的なタスクテーブルを生成するテーブル関数で、特定のタスクタイプのジョブ情報を表示できます。

## 構文

```sql
JOBS(
    "type"="<type>"
)
```
## 必須パラメーター
| フィールド         | 説明                                                                                   |
|---------------|-----------------------------------------------------------------------------------------------|
| **`<type>`**  | ジョブの種類: <br/> `insert`: Insert into type ジョブ。 <br/> `mv`: Materialized view ジョブ。 |



## 戻り値

-  **`jobs("type"="insert")`** insert 種類のジョブの戻り値

    | フィールド              | 説明                |
    |--------------------|----------------------------|
    | Id                 | ジョブ ID                     |
    | Name               | ジョブ名                   |
    | Definer            | ジョブ定義者                |
    | ExecuteType        | 実行タイプ             |
    | RecurringStrategy  | 繰り返し戦略         |
    | Status             | ジョブステータス                 |
    | ExecuteSql         | 実行 SQL              |
    | CreateTime         | ジョブ作成時刻          |
    | SucceedTaskCount   | 成功したタスクの数 |
    | FailedTaskCount    | 失敗したタスクの数     |
    | CanceledTaskCount  | キャンセルされたタスクの数   |
    | Comment            | ジョブコメント                |


- **`jobs("type"="mv")`** MV 種類ジョブの戻り値

    | フィールド                | 説明                                                 |
    |----------------------|-------------------------------------------------------------|
    | Id                   | ジョブ ID                                                      |
    | Name                 | ジョブ名                                                    |
    | MvId                 | Materialized View ID                                        |
    | MvName               | Materialized View 名                                      |
    | MvDatabaseId         | materialized view の DB ID                              |
    | MvDatabaseName       | materialized view が属するデータベースの名前 |
    | ExecuteType          | 実行タイプ                                              |
    | RecurringStrategy    | ループ戦略                                               |
    | Status               | ジョブステータス                                                  |
    | CreateTime           | タスク作成時刻                                          |
