## 説明

ユークリッド空間における2点間の距離を計算する（ベクトル値は座標）

## 構文

```sql
L2_DISTANCE(<array1>, <array2>)
```
## パラメータ

| パラメータ | 説明 |
| -- |--|
| `<array1>` | 最初のベクトル（ベクトル値は座標）、入力配列のサブタイプは：TINYINT、SMALLINT、INT、BIGINT、LARGEINT、FLOAT、DOUBLE、要素数はarray2と一致する必要があります |
| `<array2>` | 2番目のベクトル（ベクトル値は座標）、入力配列のサブタイプがサポートするもの：TINYINT、SMALLINT、INT、BIGINT、LARGEINT、FLOAT、DOUBLE、要素数はarray1と一致する必要があります |

## 戻り値

ユークリッド空間における2点間（ベクトル値は座標）の距離を返します。入力配列がNULLの場合、または配列内の任意の要素がNULLの場合、NULLが返されます。

## 例

```sql
SELECT L2_DISTANCE([4, 5], [6, 8]),L2_DISTANCE([3, 6], [4, 5]);
```
```text
+-----------------------------+-----------------------------+
| l2_distance([4, 5], [6, 8]) | l2_distance([3, 6], [4, 5]) |
+-----------------------------+-----------------------------+
|           3.605551275463989 |          1.4142135623730951 |
+-----------------------------+-----------------------------+
```
