## 説明

このステートメントは現在作成されているwarehouseを表示するために使用されます

## 構文

```sql
SHOW REPOSITORIES;
```
## 戻り値

| フィールド | 説明 |
|-----------------|------------------------------------------|
| **RepoId**      | リポジトリの一意識別子（ID） |
| **RepoName**    | リポジトリの名前 |
| **CreateTime**  | リポジトリの作成時刻 |
| **IsReadOnly**  | リポジトリが読み取り専用かどうか。`false`は読み取り専用でないことを意味し、`true`は読み取り専用を意味する |
| **Location**    | リポジトリでデータをバックアップする際に使用されるルートディレクトリ |
| **Broker**      | -                                        |
| **Type**        | リポジトリタイプ、現在はS3とHDFSをサポートしている |
| **ErrMsg**      | リポジトリのエラーメッセージ。通常、エラーが発生しない場合は`NULL` |


## 例

作成されたリポジトリを表示する：

```sql
SHOW REPOSITORIES;
```
```text
+--------+--------------+---------------------+------------+----------+--------+------+--------+
| RepoId | RepoName     | CreateTime          | IsReadOnly | Location | Broker | Type | ErrMsg |
+--------+--------------+---------------------+------------+----------+--------+------+--------+
| 43411  | example_repo | 2025-01-17 18:50:47 | false      | s3://rep1  | -      | S3   | NULL   |
+--------+--------------+---------------------+------------+----------+--------+------+--------+
```
