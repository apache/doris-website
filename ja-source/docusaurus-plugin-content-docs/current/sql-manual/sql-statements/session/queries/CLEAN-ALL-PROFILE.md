## 説明

このコマンドは、すべての履歴クエリまたは負荷プロファイルを手動でクリアするために使用されます。

## 構文

```sql
CLEAN ALL PROFILE
```
## アクセス制御要件

このSQLコマンドを実行するユーザーは、少なくとも以下の権限を持っている必要があります：

| Privilege    | Object    | Notes                                                |
|:--------------|:-----------|:-----------------------------------------------------|
| GRANT_PRIV         | DATABASE   | CLEAN文にはGRANT権限が必要です |

## 例

```sql
CLEAN ALL PROFILE
```
