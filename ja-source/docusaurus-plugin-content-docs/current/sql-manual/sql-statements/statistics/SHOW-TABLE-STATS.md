## 説明

このステートメントは、テーブルの概要統計を表示するために使用されます。

## 構文

```SQL
SHOW TABLE STATS <table_name>;
```
## 必須パラメータ

**1. `<table_name>`**

> テーブル名

## オプションパラメータ

**なし**

## 戻り値

| カラム | 備考           |
| -- |--------------|
| updated_rows | テーブル更新行数           |
| query_times |   テーブルクエリ回数           |
| row_count | テーブル行数           |
| updated_time | テーブル最終更新時刻        |
| columns | 解析されたカラムリスト           |
| trigger |   最後の解析トリガー方法           |
| new_partition |  新しいパーティションの初回読み込みフラグ           |
| user_inject | ユーザー統計情報注入フラグ         |
| enable_auto_analyze | 自動解析有効フラグ         |
| last_analyze_time |   最後の解析時刻          |

## アクセス制御要件

このSQLを実行するユーザーは、少なくとも以下の権限を持つ必要があります：

| 権限 | オブジェクト | 備考                                    |
|:--------------| :------------- |:------------------------------------------------|
| SELECT_PRIV   | Table    | SHOWを実行する際、クエリ対象のテーブルに対するSELECT_PRIV権限が必要です。 |

## 例

1. テーブルtest1の概要統計を表示する。

```sql
SHOW TABLE STATS test1;
```
```text
+--------------+-------------+-----------+---------------------+------------------------+---------+---------------+-------------+---------------------+---------------------+
| updated_rows | query_times | row_count | updated_time        | columns                | trigger | new_partition | user_inject | enable_auto_analyze | last_analyze_time   |

| 0            | 0           | 100000    | 2025-01-17 16:46:31 | [test1:name, test1:id] | MANUAL  | false         | false       | true                | 2025-02-05 12:17:41 |
+--------------+-------------+-----------+---------------------+------------------------+---------+---------------+-------------+---------------------+---------------------+
```
