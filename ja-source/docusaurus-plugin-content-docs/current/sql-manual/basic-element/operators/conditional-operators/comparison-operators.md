## 説明

比較条件は、ある式と別の式を比較します。比較の結果は TRUE、FALSE、または UNKNOWN になります。

## オペレータの紹介

| オペレータ | 機能 | 例 |
| ------------------- | ----------------------------------------------------------- | ------------------- |
| `=` | 等価比較。比較のいずれかの側が UNKNOWN の場合、結果は UNKNOWN になります。 | `SELECT 1 = 1` |
| `<=>` | NULL セーフな等価比較。等価比較とは異なり、NULL セーフな等価比較では NULL を比較可能な値として扱います。両側が NULL の場合は TRUE を返します。一方の側のみが NULL の場合は FALSE を返します。このオペレータは UNKNOWN を返すことはありません。 | `SELECT NULL <=> NULL` |
| `!=` `<>` | 不等価比較 | `SELECT 1 != 1` |
| `<` `>` | より大きい、より小さい比較 | `SELECT 1 > 1` |
| `<=` `>=` | 以上、以下比較 | `SELECT 1 >= 1` |
| `<x> BETWEEN <y> AND <z>` | `<x> >= <y> and <x> <= <z>` と同等。`<y>` 以上かつ `<z>` 以下 | `SELECT 1 BETWEEN 0 AND 2` |
