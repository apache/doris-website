## 説明

演算子の優先順位は、式内で演算子が評価される順序を決定します。式に複数の演算子が含まれている場合、Dorisは演算子の優先順位の高い順に計算を実行します。

## 演算子の優先順位

優先順位は上から下に向かって低くなり、最も高い優先順位が上部にあります。

| 優先順位 | 演算子 |
|----------|--------|
| 1        | !      |
| 2        | + (単項プラス), - (単項マイナス), ~ (単項ビットNOT), ^ |
| 3        | *, /, %, DIV |
| 4        | -, +   |
| 5        | &      |
| 6        | \|     |
| 7        | =(比較), <=>, >=, >, <=, <, <>, !=, IS, LIKE, REGEXP, MATCH, IN |
| 8        | NOT    |
| 9        | AND, && |
| 10       | XOR    |
| 11       | OR     |
| 12       | \|\|   |
