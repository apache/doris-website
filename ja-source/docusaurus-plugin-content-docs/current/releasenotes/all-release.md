この文書は、1年以内にリリースされたApache Dorisのバージョンの概要を、逆時系列順で示しています。

:::tip Latest Release
🎉 Version 4.0.3がリリースされました。🔗[Release 注](../releasenotes/v4.0/release-4.0.3)をこちらでご確認ください。Apache Doris 4.0は、新しいAIサポート（ベクター検索、AI Functions）とより優れた全文検索機能により大きく前進しました。これらのアップグレードは、ユーザーがAIとagent時代において先行できるよう支援し、企業が従来のBI分析からAI駆動のワークロードまでのすべてを処理できるようにします。

<br />

🎉 Version 3.1.4がリリースされました。🔗[Release 注](../releasenotes/v3.1/release-3.1.4)をこちらでご確認ください。Doris 3.1では、VARIANTデータタイプ用のスパースカラムとスキーマテンプレートが導入され、ログやJSONデータなどの動的フィールドを持つ大規模データセットをより効率的に保存およびクエリできるようになりました。レイクハウス機能については、非同期マテリアライズドビューが強化され、IcebergとPaimonのサポートが拡張されて、データレイクとデータウェアハウス間のより強固な橋渡しを構築します。

<br />

🎉 Version 3.0.8がリリースされました。🔗[Release 注](../releasenotes/v3.0/release-3.0.8)をこちらでご確認ください。バージョン3.Xから、Apache Dorisはクラスター配置において計算ストレージ結合モードに加えて計算ストレージ分離モードをサポートします。計算層とストレージ層を分離するクラウドネイティブアーキテクチャにより、ユーザーは複数の計算クラスター間でのクエリ負荷の物理的分離、および読み取りと書き込み負荷の分離を実現できます。

<br />

🎉 Version 2.1.11がリリースされました。🔗[Release 注](../releasenotes/v2.1/release-2.1.11)をこちらでご確認ください。2.1バージョンは、TPC-DS 1TBテストで実証された100%高いout-of-the-boxクエリによる卓越したパフォーマンス、TrinoやSparkより4〜6倍高速な強化されたデータレイク分析、新しいVariantタイプと分析関数スイートによる半構造化データ分析の堅実なサポート、クエリ加速のための非同期マテリアライズドビュー、スケールでの最適化されたリアルタイム書き込み、および安定性とランタイムSQLリソース追跡による改善されたワークロード管理を提供します。

:::
