## 概要

文字セットのすべての照合方法を表示します。このテーブルはMySQLの動作との互換性のためにのみ使用され、実用的な意味はありません。Dorisで実際に使用される文字照合方法を真に反映するものではありません。

## データベース

`information_schema`

## テーブル情報

| カラム名           | タイプ       | 説明                                                     |
| ------------------ | ------------ | -------------------------------------------------------- |
| COLLATION_NAME     | varchar(512) | 文字セット照合方法の名前                                  |
| CHARACTER_SET_NAME | varchar(64)  | 関連する文字セットの名前                                  |
| ID                 | bigint       | 照合方法のID                                             |
| IS_DEFAULT         | varchar(64)  | 現在のデフォルト照合方法かどうかを示します。               |
| IS_COMPILED        | varchar(64)  | サービスにコンパイルされているかどうかを示します           |
| SORTLEN            | bigint       | この照合アルゴリズムで使用されるメモリに関連します         |
