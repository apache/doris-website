| パラメータ                  | 説明                                         | デフォルト値 | 使用シナリオ                                               |
| -------------------------- | --------------------------------------------------- | ------------- | ------------------------------------------------------------ |
| enable_nereids_planner     | 新しいオプティマイザを有効にするかどうか                 | TRUE          | 低バージョンからのアップグレードなどのシナリオでは、最初はfalseに設定し、アップグレード後にtrueに設定することができます |
| enable_nereids_dml         | 新しいオプティマイザのDMLサポートを有効にするかどうか | TRUE          | 低バージョンからのアップグレードなどのシナリオでは、最初はfalseに設定し、アップグレード後にtrueに設定することができます |
| parallel_pipeline_task_num | パイプライン並列度                                | 0             | 低バージョンからのアップグレードなどのシナリオでは、この値は以前は固定値に設定されていましたが、アップグレード後は0に設定でき、システムの適応戦略が並列度を決定することを示します |
| runtime_filter_mode        | Runtime Filterタイプ                                 | GLOBAL        | 低バージョンからのアップグレードなどのシナリオでは、この値はNONEで、Runtime Filterが有効になっていないことを示していましたが、アップグレード後はGLOBALに設定でき、Runtime Filterがデフォルトで有効になることを示します |
