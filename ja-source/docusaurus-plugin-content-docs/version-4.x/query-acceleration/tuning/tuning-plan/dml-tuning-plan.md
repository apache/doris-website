DMLプランのチューニングでは、まずパフォーマンスボトルネックがインポートプロセスによるものか、クエリセクションによるものかを特定する必要があります。クエリセクションにおけるパフォーマンスボトルネックのトラブルシューティングとチューニングについては、詳細を[Plan Tuning](optimizing-table-schema.md)の他のサブセクションを参照してください。

Dorisは複数のデータソースからのデータインポートをサポートしています。Dorisが提供する様々なインポート機能を柔軟に活用することで、各種ソースからのデータを効率的にDorisにインポートして分析することができます。ベストプラクティスの詳細については、[Import 概要](../../../data-operate/import/load-manual.md)を参照してください。
