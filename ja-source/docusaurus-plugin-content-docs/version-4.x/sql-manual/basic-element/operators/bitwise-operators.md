## 説明

ビット演算子は、ビットレベルで1つまたは2つの式に対して指定された操作を実行します。これらの演算子は`BIGINT`型の引数のみを受け取ります。したがって、ビット演算子で処理される式は`BIGINT`型に変換されます。

## 演算子の概要

| 演算子 | 機能 | 例 |
|----------|----------------------------------------------------------------------------------------------|-----------------|
| `&` | ビット単位のAND演算を実行します。対応するビットが両方とも`1`の場合、結果のビットは`1`になります。そうでなければ`0`になります。 | `SELECT 1 & 2` |
| `\|` | ビット単位のOR演算を実行します。対応するビットのいずれかが`1`の場合、結果のビットは`1`になります。そうでなければ`0`になります。 | `SELECT 1 | 2` |
| `^` | ビット単位のXOR演算を実行します。対応するビットが異なる場合、結果のビットは`1`になります。そうでなければ`0`になります。 | `SELECT 1 ^ 2` |
| `~` | ビット単位のNOT演算を実行します。各ビットを反転します：`1`は`0`になり、`0`は`1`になります。 | `SELECT ~1` |
