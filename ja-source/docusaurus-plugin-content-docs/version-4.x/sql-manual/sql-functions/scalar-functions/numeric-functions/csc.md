## デスクリプション

xのcosecantを返します。xはラジアン単位の値で、入力と出力はdoubleのみサポートされています。入力がnull値の場合、null値を返します。

## Syntax

```sql
CSC(<x>)
```
## パラメータ

| Parameter | デスクリプション |
| -- | -- |
| `<x>` | cosecantを計算する値 |

## Return Value

xのcosecantを表すDouble型の値を返します。

## Special Cases
- `x`がNaNの場合、NaNを返します
- `x`が正または負の無限大の場合、NaNを返します
- `x`がNULLの場合、NULLを返します

## Example

```sql
select csc(1),csc(2),csc(1000);
```
```text
+--------------------+--------------------+------------------+
| csc(1)             | csc(2)             | csc(1000)        |
+--------------------+--------------------+------------------+
| 1.1883951057781212 | 1.0997501702946164 | 1.20936599707935 |
+--------------------+--------------------+------------------+
```
Input null値。

```sql
select csc(null);
```
```text
+--------------------+
| csc(null)          |
+--------------------+
|      NULL          |
+--------------------+
```
```sql
select csc(cast('nan' as double));
```
```text
+---------------------------+
| csc(cast('nan' AS DOUBLE))|
+---------------------------+
| NaN                       |
+---------------------------+
```
```sql
select csc(cast('inf' as double));
```
```text
+---------------------------+
| csc(cast('inf' AS DOUBLE))|
+---------------------------+
| NaN                       |
+---------------------------+
```
```sql
select csc(cast('-inf' as double));
```
```text
+----------------------------+
| csc(cast('-inf' AS DOUBLE))|
+----------------------------+
| NaN                        |
+----------------------------+
```
