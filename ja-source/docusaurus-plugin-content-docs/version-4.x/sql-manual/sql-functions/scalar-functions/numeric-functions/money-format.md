## デスクリプション

数値は通貨形式で出力され、整数部分は3桁ごとにカンマで区切られ、小数部分は2桁で保持されます。

## Syntax

```sql
MONEY_FORMAT(<number>)
```
## パラメータ

| Parameter | デスクリプション |
|-----------|------------|
| `<number>`   | フォーマットされる数値 |

## Return value

通貨形式の文字列を返します。特別なケース：

- パラメータがNULLの場合、NULLを返す

## Example

```sql
select money_format(17014116);
```
```text
+------------------------+
| money_format(17014116) |
+------------------------+
| 17,014,116.00          |
+------------------------+
```
```sql
select money_format(1123.456);
```
```text
+------------------------+
| money_format(1123.456) |
+------------------------+
| 1,123.46               |
+------------------------+
```
```sql
select money_format(1123.4);
```
```text
+----------------------+
| money_format(1123.4) |
+----------------------+
| 1,123.40             |
+----------------------+
```
