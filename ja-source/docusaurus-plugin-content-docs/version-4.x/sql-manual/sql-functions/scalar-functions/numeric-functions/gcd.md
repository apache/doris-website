## デスクリプション

2つの整数の最大公約数（GCD）を計算します。

## Syntax

```sql
GCD(<a>, <b>)
```
## パラメータ

| Parameter | デスクリプション |
| -- | -- |
| `<a>` | 最初の整数 |
| `<b>` | 2番目の整数 |

## Return Value

`<a>`と`<b>`の最大公約数を返します。
入力がNULLの場合、NULLを返します。

## Examples

```sql
select gcd(54, 24);
```
```text
+------------+
| gcd(54,24) |
+------------+
|          6 |
+------------+
```
```sql
select gcd(-17, 31);
```
```text
+-------------+
| gcd(17,31)  |
+-------------+
|           1 |
+-------------+
```
```sql
select gcd(0, 10);
```
```text
+-----------+
| gcd(0,10) |
+-----------+
|        10 |
+-----------+
```
```sql
select gcd(54, NULL);
```
```text
+---------------+
| gcd(54, NULL) |
+---------------+
|          NULL |
+---------------+
```
