## 説明

パラメータの立方根を計算します

## 構文

```sql
CBRT(<a>)
```
## パラメータ

| Parameter | デスクリプション |
| -- | -- |
| `<a>` | 浮動小数点パラメータ |

## 戻り値

パラメータ`<a>`の立方根、浮動小数点数。

## 特殊なケース
- `a`がNaNの場合、NaNを返す
- `a`が正の無限大の場合、Infinityを返す
- `a`が負の無限大の場合、-Infinityを返す
- `a`がNULLの場合、NULLを返す

## 例

```sql
select cbrt(0);
```
```text
+-------------------------+
| cbrt(cast(0 as DOUBLE)) |
+-------------------------+
|                     0.0 |
+-------------------------+
```
```sql
select cbrt(-111);
```
```text
+----------------------------+
| cbrt(cast(-111 as DOUBLE)) |
+----------------------------+
|         -4.805895533705333 |
+----------------------------+
```
```sql
select cbrt(1234);
```
```text
+----------------------------+
| cbrt(cast(1234 as DOUBLE)) |
+----------------------------+
|         10.726014668827325 |
+----------------------------+
```
```sql
select cbrt(cast('nan' as double));
```
```text
+------------------------------+
| cbrt(cast('nan' AS DOUBLE))  |
+------------------------------+
| NaN                          |
+------------------------------+
```
```sql
select cbrt(cast('inf' as double));
```
```text
+------------------------------+
| cbrt(cast('inf' AS DOUBLE))  |
+------------------------------+
| Infinity                     |
+------------------------------+
```
```sql
select cbrt(cast('-inf' as double));
```
```text
+-------------------------------+
| cbrt(cast('-inf' AS DOUBLE))  |
+-------------------------------+
| -Infinity                     |
+-------------------------------+
```
