## 説明

`x`の絶対値を返します

## 構文

```sql
ABS(<x>) 
```
## パラメータ

| パラメータ | 説明 |
| -- | -- |
| `<x>` | 絶対値を計算する値 |

## 戻り値

パラメータ `x` の絶対値。
`x` が NULL の場合、NULL を返します。

## 例

```sql
select abs(-2);
```
```text
+---------+
| abs(-2) |
+---------+
|       2 |
+---------+
```
```sql
select abs(3.254655654);
```
```text
+------------------+
| abs(3.254655654) |
+------------------+
|      3.254655654 |
+------------------+
```
```sql
select abs(-3254654236547654354654767);
```
```text
+---------------------------------+
| abs(-3254654236547654354654767) |
+---------------------------------+
| 3254654236547654354654767       |
+---------------------------------+
```
```sql
select abs(NULL);
```
```text
+-----------+
| abs(NULL) |
+-----------+
|      NULL |
+-----------+
```
