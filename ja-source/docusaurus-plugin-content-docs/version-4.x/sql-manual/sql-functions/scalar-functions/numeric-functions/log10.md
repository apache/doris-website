## 説明

`x`の底10に対する自然対数を返します。

## エイリアス

- DLOG10

## 構文

```sql
LOG10(<x>)
```
## パラメータ

| Parameter | デスクリプション |
|-----------|------------|
| `<x>`   | 真数は0より大きくなければなりません |

## Return value

浮動小数点数を返します。特殊ケース：

- x が NULL の場合、`NULL` を返します
- x が NaN の場合、NaN を返します

## Example

```sql
select log10(1);
```
```text
+--------------------------+
| log10(cast(1 as DOUBLE)) |
+--------------------------+
|                      0.0 |
+--------------------------+
```
```sql
select log10(10);
```
```text
+---------------------------+
| log10(cast(10 as DOUBLE)) |
+---------------------------+
|                       1.0 |
+---------------------------+
```
```sql
select log10(16);
```
```text
+---------------------------+
| log10(cast(16 as DOUBLE)) |
+---------------------------+
|        1.2041199826559248 |
+---------------------------+
```
```sql
select log10(NULL);
```
```text
+-------------+
| log10(NULL) |
+-------------+
|        NULL |
+-------------+
```
```sql
select log10(cast('nan' as double));
```
```text
+------------------------------+
| log10(cast('nan' as double)) |
+------------------------------+
|                          NaN |
+------------------------------+
```
