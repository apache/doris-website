## 説明

浮動小数点数のモジュロ a / b を計算します。整数型の場合は、mod関数を使用してください。

## 構文

```sql
FMOD(<col_a> , <col_b>)
```
## パラメータ

| Parameter | デスクリプション |
| -- | -- |
| `<col_a>` | 被除数 |
| `<col_b>` | 除数（0にはできません） |

## Return Value

浮動小数点数を返します。特殊なケース：

- 入力パラメータのいずれかがNULLの場合、NULLを返します。

## Examples

```sql
select fmod(10.1, 3.2);
```
```text
+-----------------+
| fmod(10.1, 3.2) |
+-----------------+
|      0.50000024 |
+-----------------+
```
```sql
select fmod(10.1, 0);
```
```text
+---------------+
| fmod(10.1, 0) |
+---------------+
|          NULL |
+---------------+
```
```sql
select fmod(10.1, NULL);
```
```text
+------------------+
| fmod(10.1, NULL) |
+------------------+
|             NULL |
+------------------+
```
