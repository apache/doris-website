## 説明

`x`の双曲線逆余弦を返します。`x`が`1`未満の場合は`NULL`を返します。

## 構文

```sql
ACOSH(<x>)
```
## パラメータ

| Parameter | デスクリプション |  
| -- | -- |  
| `<x>` | 双曲線逆余弦を計算する値 |  

## Return Value  

パラメータ`x`の双曲線逆余弦値。

## Special Cases

- `x`が1の場合、0を返す
- `x`が1未満の場合、`NULL`を返す
- `x`がNaNの場合、NaNを返す
- `x`が正の無限大の場合、Infinityを返す
- `x`が負の無限大の場合、`NULL`を返す
- `x`がNULLの場合、NULLを返す

## Examples

```sql
select acosh(0.0);
```
```text
+------------+
| acosh(0.0) |
+------------+
|       NULL |
+------------+
```
```sql
select acosh(-1.0);
```
```text

+-------------+
|        NULL |
+-------------+
```
```sql
select acosh(1.0);
```

```text
+------------+
|          0 |
+------------+
```
