## 説明

10進数を2進数のテキストに変換します。

## 構文

```sql
BIN(<a>)
```
## パラメータ

| Parameter | デスクリプション |
| -- | -- |
| `<a>` | 変換される10進数値（型：BigInt） |

## Return Value

パラメータ`<a>`のバイナリ表現。`<a>`が負の値の場合、結果は64ビット補数となります。
`a`がNULLの場合、NULLを返します。

## Examples

```sql
select bin(0);
```
```text
+--------+
| bin(0) |
+--------+
| 0      |
+--------+
```
```sql
select bin(-1);
```
```text
+------------------------------------------------------------------+
| bin(-1)                                                          |
+------------------------------------------------------------------+
| 1111111111111111111111111111111111111111111111111111111111111111 |
+------------------------------------------------------------------+
```
```sql
select bin(123);
```
```text
+----------+
| bin(123) |
+----------+
| 1111011  |
+----------+
```
```sql
select bin(NULL);
```
```text
+-----------+
| bin(NULL) |
+-----------+
| NULL      |
+-----------+
```
