## 説明

`x`の双曲線正弦を返します。

## 構文

```sql
SINH(<x>)
```
## パラメータ

| パラメータ | 説明 |  
| -- | -- |  
| `<x>` | 双曲線正弦値を計算する値 |  

## 戻り値  

パラメータ `x` のsinh値。

## 特殊なケース
- `x` がNaNの場合、NaNを返す
- `x` が正の無限大の場合、Infinityを返す
- `x` が負の無限大の場合、-Infinityを返す
- `x` がNULLの場合、NULLを返す

## 例

```sql
select sinh(0.0);
```
```
+-----------+
| sinh(0.0) |
+-----------+
|         0 |
+-----------+
```
```sql
select sinh(1.0);
```
```
+--------------------+
| sinh(1.0)          |
+--------------------+
| 1.1752011936438014 |
+--------------------+
```
```sql
select sinh(-1.0);
```
```
+---------------------+
| sinh(-1.0)          |
+---------------------+
| -1.1752011936438014 |
+---------------------+
```
```sql
select sinh(cast('nan' as double));
```
```
+------------------------------+
| sinh(cast('nan' AS DOUBLE))  |
+------------------------------+
| NaN                          |
+------------------------------+
```
```sql
select sinh(cast('inf' as double));
```
```
+------------------------------+
| sinh(cast('inf' AS DOUBLE))  |
+------------------------------+
| Infinity                     |
+------------------------------+
```
```sql
select sinh(cast('-inf' as double));
```
```
+-------------------------------+
| sinh(cast('-inf' AS DOUBLE))  |
+-------------------------------+
| -Infinity                     |
+-------------------------------+
```
