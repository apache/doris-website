## 説明

パラメータxの負の値を返します。

## 構文

```sql
NEGATIVE(<x>)
```
## パラメータ

| Parameter | デスクリプション |
|-----------|------------|
| `<x>` | 独立変数は `BIGINT, DOUBLE, and DECIMAL` 型をサポートします |

## 戻り値

整数または浮動小数点数を返します。特殊なケース：

- パラメータがNULLの場合、NULLを返します。
- パラメータが0の場合、0を返します。

## 例

```sql
SELECT negative(-10);
```
```text
+---------------+
| negative(-10) |
+---------------+
|            10 |
+---------------+
```
```sql
SELECT negative(12);
```
```text
+--------------+
| negative(12) |
+--------------+
|          -12 |
+--------------+
```
```sql
SELECT negative(0);
```
```text
+-------------+
| negative(0) |
+-------------+
|           0 |
+-------------+
```
```sql
SELECT negative(null);
```
```text
+----------------+
| negative(NULL) |
+----------------+
|           NULL |
+----------------+
```
