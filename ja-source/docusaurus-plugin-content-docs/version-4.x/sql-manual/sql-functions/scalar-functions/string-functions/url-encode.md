## 概要

提供されたテキストのURL エンコーディングを完了するためにUTF-8エンコーディングを使用します。通常、URLの一部として渡されるパラメータ情報をエンコードするために使用されます。

## 構文

```sql
URL_ENCODE(  <str>  ) 
```
## Required パラメータ

## Required パラメータ
| パラメータ | デスクリプション |
|------|------|
| `<str>` | エンコードする文字列 |

## Return Value

UTF-8エンコーディングにより、提供されたテキストのURLエンコーディングが完了します

## Example

```sql
select  URL_ENCODE('Doris Q&A');
```
```sql
+-------------------------+
| url_encode('Doris Q&A') |
+-------------------------+
| Doris+Q%26A             |
+-------------------------+

```
