## 説明

MASK関数は、機密情報を保護するためにデータをマスクするために使用されます。デフォルトの動作では、大文字を`X`に、小文字を`x`に、数字を`n`に変換します。

## 構文

```sql
MASK(<str>[, <upper>[, <lower>[, <number>]]])
```
## パラメータ

| パラメータ | 説明 |
| ---------- | ----------------------------------------- |
| `<str>` | マスクする文字列。型: VARCHAR |
| `<upper>` | 大文字を置き換える文字、デフォルトは `X`（オプション）。型: VARCHAR |
| `<lower>` | 小文字を置き換える文字、デフォルトは `x`（オプション）。型: VARCHAR |
| `<number>` | 数字を置き換える文字、デフォルトは `n`（オプション）。型: VARCHAR |

## 戻り値

VARCHAR型を返し、文字と数字が置き換えられた文字列。

特殊なケース:
- いずれかのパラメータがNULLの場合、NULLを返す
- 英数字以外の文字は変更されない
- 置換文字パラメータに複数文字が含まれる場合、最初の文字のみが使用される

## 例

1. 基本的な使用法: デフォルトの置換ルール

```sql
SELECT mask('abc123XYZ');
```
```text
+-------------------+
| mask('abc123XYZ') |
+-------------------+
| xxxnnnXXX         |
+-------------------+
```
2. カスタム置換文字

```sql
SELECT mask('abc123XYZ', '*', '#', '$');
```
```text
+----------------------------------+
| mask('abc123XYZ', '*', '#', '$') |
+----------------------------------+
| ###$$$***                        |
+----------------------------------+
```
3. 特殊文字は変更されません

```sql
SELECT mask('Hello-123!');
```
```text
+--------------------+
| mask('Hello-123!') |
+--------------------+
| Xxxxx-nnn!         |
+--------------------+
```
4. NULL値の処理

```sql
SELECT mask(NULL);
```
```text
+------------+
| mask(NULL) |
+------------+
| NULL       |
+------------+
```
5. 数字のみの文字列

```sql
SELECT mask('1234567890');
```
```text
+--------------------+
| mask('1234567890') |
+--------------------+
| nnnnnnnnnn         |
+--------------------+
```
6. 文字のみの文字列

```sql
SELECT mask('AbCdEfGh');
```
```text
+------------------+
| mask('AbCdEfGh') |
+------------------+
| XxXxXxXx         |
+------------------+
```
7. 空文字列の処理

```sql
SELECT mask('');
```
```text
+----------+
| mask('') |
+----------+
|          |
+----------+
```
8. 単一文字置換（複数から最初の文字を取得）

```sql
SELECT mask('Test123', 'ABC', 'xyz', '999');
```
```text
+--------------------------------------+
| mask('Test123', 'ABC', 'xyz', '999') |
+--------------------------------------+
| Xxxx999                              |
+--------------------------------------+
```
9. クレジットカード番号をマスクする

```sql
SELECT mask('1234-5678-9012-3456');
```
```text
+-----------------------------+
| mask('1234-5678-9012-3456') |
+-----------------------------+
| nnnn-nnnn-nnnn-nnnn         |
+-----------------------------+
```
10. メールアドレスをマスクする

```sql
SELECT mask('user@example.com');
```
```text
+--------------------------+
| mask('user@example.com') |
+--------------------------+
| xxxx@xxxxxxx.xxx         |
+--------------------------+
```
```sql
SELECT mask('eeeéèêëìí1234');
```
```text
+-----------------------------+
| mask('eeeéèêëìí1234')       |
+-----------------------------+
| xxxéèêëìínnnn               |
+-----------------------------+
```
