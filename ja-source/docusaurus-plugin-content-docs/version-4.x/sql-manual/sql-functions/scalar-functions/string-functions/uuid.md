## 説明

UUID関数は、ランダムなUniversally Unique Identifier (UUID)を生成します。生成されるUUIDはRFC 4122標準に準拠し、8-4-4-4-12の形式（ハイフンを含む36文字）です。

## 構文

```sql
UUID()
```
## パラメータ

パラメータはありません。

## 戻り値

VARCHAR型を返します。ランダムに生成されたUUID文字列です。

特殊なケース:
- 各呼び出しで異なるUUIDが生成されます
- UUIDフォーマット: xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx
- 生成されるUUIDはグローバルに一意です

## 例

1. 基本的な使用方法: 単一のUUIDを生成

```sql
SELECT UUID();
```
```text
+--------------------------------------+
| UUID()                               |
+--------------------------------------+
| 29077778-fc5e-4603-8368-6b5f8fd55c24 |
+--------------------------------------+
```
2. 複数回の呼び出しにより異なるUUIDが生成される

```sql
SELECT UUID(), UUID();
```
```text
+--------------------------------------+--------------------------------------+
| UUID()                               | UUID()                               |
+--------------------------------------+--------------------------------------+
| a1b2c3d4-e5f6-47a8-b9c0-d1e2f3a4b5c6 | f7e8d9c0-b1a2-4938-8756-c4d3e2f1a0b9 |
+--------------------------------------+--------------------------------------+
```
### キーワード
