## ファンクション

lambda関数が指定されていない場合、配列要素は昇順でソートされます。それ以外の場合は、lambda関数に従ってソートが実行されます。

## Syntax

- `ARRAY_SORT(arr)`
- `ARRAY_SORT(lambda, arr)`

## パラメータ

- `lambda`: ソートルールを定義するために使用される`lambda`式で、戻り値は-1、0、または1である必要があります（それぞれ未満、以下または等しい、および超過を表します）。
- `arr`: `ARRAY<T>`。ここで`T`は数値、boolean、文字列、datetime、IPなどにすることができます。

## Return value

- `lambda`式を含まない場合
  - 入力と同じ型の`ARRAY<T>`を返します。
  - `NULL`要素は返される配列の先頭に配置されます。
- `lambda`式を含む場合、lambda式に従ってソートし、入力と同じ型の`ARRAY<T>`を返します。

## Usage notes

- `lambda`式を含まない場合
  - 入力が`NULL`の場合は`NULL`を返します。入力が空の配列`[]`の場合は空の配列を返します。
  - `ARRAY_SORT`は昇順でソートし、`ARRAY_REVERSE_SORT`は降順でソートします。
- `lambda`式を含む場合
  - `lambda`式の値は-1、0、または1である必要があります。`NULL`値が存在する場合は、`NULL`値を配列の前に配置するか後に配置するかを明示的に記述する必要があります。

## Examples

1. Basic: `NULL`要素は返される配列の先頭に配置されます

```sql
SELECT ARRAY_SORT([2,1,3,null]);
```
```text
+--------------------------+
| ARRAY_SORT([2,1,3,null]) |
+--------------------------+
| [null, 1, 2, 3]          |
+--------------------------+
```
2. 入力が`NULL`の場合は`NULL`を返し、入力が空の配列`[]`の場合は空の配列を返します。

```sql
SELECT ARRAY_SORT(NULL);
```
```text
+------------------+
| ARRAY_SORT(NULL) |
+------------------+
| NULL             |
+------------------+
```
```sql
SELECT ARRAY_SORT([]);
```
```text
+----------------+
| ARRAY_SORT([]) |
+----------------+
| []             |
+----------------+
```
3. `lambda`式を使用する

```sql
SELECT array_sort((x, y) -> IF(x < y, 1, IF(x = y, 0, -1)), [3, 2, 5, 1, 2]);
```
```text
+-----------------------------------------------------------------------+
| array_sort((x, y) -> IF(x < y, 1, IF(x = y, 0, -1)), [3, 2, 5, 1, 2]) |
+-----------------------------------------------------------------------+
| [5, 3, 2, 2, 1]                                                       |
+-----------------------------------------------------------------------+
```
```sql
SELECT array_sort((x, y) -> IF(x < y, 1, IF(x = y, 0, -1)), ['bc', 'ab', 'dc']);
```
```text
+--------------------------------------------------------------------------+
| array_sort((x, y) -> IF(x < y, 1, IF(x = y, 0, -1)), ['bc', 'ab', 'dc']) |
+--------------------------------------------------------------------------+
| ["dc", "bc", "ab"]                                                       |
+--------------------------------------------------------------------------+
```
```sql
SELECT array_sort((x, y) -> CASE WHEN x IS NULL THEN -1
                              WHEN y IS NULL THEN 1
                              WHEN x < y THEN 1
                              WHEN x = y THEN 0
                              ELSE -1 END,
                              [3, 2, null, 5, null, 1, 2]);
```
```text
+------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| array_sort((x, y) -> CASE WHEN x IS NULL THEN -1
                              WHEN y IS NULL THEN 1
                              WHEN x < y THEN 1
                              WHEN x = y THEN 0

                              [3, 2, null, 5, null, 1, 2]) |
+------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| [null, null, 5, 3, 2, 2, 1]                                                                                                                                                                                                                                                                                |
+------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
```
```sql
SELECT array_sort((x, y) -> CASE WHEN x IS NULL THEN 1
                              WHEN y IS NULL THEN -1

                              WHEN x = y THEN 0
```

```sql
SELECT array_sort((x, y) -> IF(length(x) < length(y), -1,
                            IF(length(x) = length(y), 0, 1)),
                            ['a', 'abcd', 'abc']);
```
```text
+--------------------------------------------------------------------------------------------------------------------------------------------------------------------+

+--------------------------------------------------------------------------------------------------------------------------------------------------------------------+
```
```sql
SELECT array_sort((x, y) -> IF(cardinality(x) < cardinality(y), -1,
                            IF(cardinality(x) = cardinality(y), 0, 1)),
                            [[2, 3, 1], [4, 2, 1, 4], [1, 2]]);
```
```text
+-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
```
