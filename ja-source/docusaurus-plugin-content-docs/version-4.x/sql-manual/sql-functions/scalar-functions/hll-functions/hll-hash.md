## 説明

指定された値をHLL（HyperLogLog）型に変換します。この関数は通常、生データからHLLを作成するためのデータ読み込み時に使用されます。

## 構文

```sql
HLL_HASH(<value>)
```
## パラメータ

| Parameter | デスクリプション                                                   |
|-----------|---------------------------------------------------------------|
| `<value>` | HLL型に変換される値。これは文字列、数値、または任意のデータ型を指定できます。 |

## Examples

```sql
SELECT HLL_CARDINALITY(HLL_HASH('abc'));
```
```text
+----------------------------------+
| hll_cardinality(HLL_HASH('abc')) |
+----------------------------------+
|                                1 |
+----------------------------------+
```
