## 概要

複数の文字列のSM3 256ビットを計算

## 構文

```sql
SM3SUM( <str> [ , <str> ... ] )
```
## パラメータ

| parameter | description |
|-----------|-------------|
| `<str>`   | 計算が必要なsm3の値   |

## Return Value

入力された複数の文字列のsm3値を返します

## Examples

```sql
select sm3sum("ab","cd");
```
```text
+------------------------------------------------------------------+
| sm3sum('ab', 'cd')                                               |
+------------------------------------------------------------------+
| 82ec580fe6d36ae4f81cae3c73f4a5b3b5a09c943172dc9053c69fd8e18dca1e |
+------------------------------------------------------------------+
```
