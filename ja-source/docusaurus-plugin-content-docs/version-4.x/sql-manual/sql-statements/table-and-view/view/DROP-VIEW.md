## 説明

現在のデータベースまたは指定されたデータベース内のビューを削除します。

## 構文

```sql
DROP VIEW [ IF EXISTS ] <name>
```
## 必須パラメータ

削除対象のビューの名前。

## オプションパラメータ

**[ IF EXISTS ]**

このパラメータが指定された場合、ビューが存在しない時にエラーは発生せず、削除操作は直接スキップされます。

## アクセス制御要件

このSQLコマンドを実行するユーザーは、最低限以下の権限を持っている必要があります：

| 権限 | オブジェクト | 備考 |
| :-------- | :----- | :---- |
| DROP_PRIV | Table  |       |

## 使用上の注意

削除されたビューは復元できないため、再作成する必要があります。

## 例

```sql
CREATE VIEW vtest AS SELECT 1, 'test';
DROP VIEW IF EXISTS vtest;
```
