## 説明

この文は、プラグインをアンインストールするために使用されます。

## 構文：

```sql
UNINSTALL PLUGIN <plugin_name>;
```
## 必須パラメータ

** 1. `<plugin_name>`**
>  アンインストールするプラグインの名前

## 権限制御

このSQLコマンドを実行するユーザーは、少なくとも以下の権限を持っている必要があります：

| 権限         | オブジェクト   | 備考            |
|:-----------|:-----|:--------------|
| ADMIN_PRIV | クラスタ全体 | クラスタ全体に対する管理者権限が必要 |

## 注意事項

ビルトイン以外のプラグインのみアンインストール可能

## 例

- プラグインをアンインストールする場合：

    ```sql
    UNINSTALL PLUGIN auditdemo;
    ```
