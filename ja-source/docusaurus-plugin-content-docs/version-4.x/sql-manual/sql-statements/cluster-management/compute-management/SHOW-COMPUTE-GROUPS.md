## 説明

コンピュートストレージ分離モードにおいて、現在のユーザーが使用権限を持つコンピュートグループのリストを表示します。

## 構文

```sql
SHOW COMPUTE GROUPS
```
## 戻り値

現在のユーザーが権限を持つコンピュートグループのリストを返します。

- Name - コンピュートグループの名前
- IsCurrent - 現在のユーザーがこのコンピュートグループを使用しているかどうか
- Users - このコンピュートグループをデフォルトのコンピュートグループとして設定しているユーザー名
- BackendNum - このコンピュートグループが持つバックエンドの数

## 例

`compute_cluster`という名前のコンピュートグループの使用を指定します。

```sql
SHOW COMPUTE GROUPS;
```
結果は以下の通りです:

```sql
+-----------------+-----------+-------+------------+
| Name           | IsCurrent  | Users | BackendNum |
+-----------------+-----------+-------+------------+
| compute_cluster | TRUE      |       | 3          |
+-----------------+-----------+-------+------------+
```
## 使用上の注意

現在のユーザーがいずれのcompute groupに対しても権限を持たない場合、`SHOW COMPUTE GROUPS`は空のリストを返します。
