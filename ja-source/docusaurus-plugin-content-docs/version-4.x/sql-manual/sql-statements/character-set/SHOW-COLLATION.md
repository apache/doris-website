## 説明

Dorisでは、SHOW COLLATIONコマンドはデータベースで利用可能な文字セットの照合順序を表示するために使用されます。照合順序とは、データがどのようにソートされ比較されるかを決定する一連のルールです。これらのルールは文字データの格納と取得に影響を与えます。

## 構文

```sql
SHOW COLLATION
```
## 戻り値

| column name | description  |
| -- |--------------|
| Collation | 照合順序名         |
| Charset | 文字セット          |
| Id | 照合順序のID        |
| Default | この文字セットのデフォルト照合順序かどうか |
| Compiled | 照合順序がコンパイルされているかどうか        |
| Sortlen | ソート長         |



## 使用上の注意

Dorisでは、MySQLの照合順序設定コマンドとの互換性はありますが、実際には効果がありません。実行時には、比較ルールとして常にutf8mb4_0900_binが使用されます。

## 例

```sql
SHOW COLLATION;
```
```text
+--------------------+---------+------+---------+----------+---------+
| Collation          | Charset | Id   | Default | Compiled | Sortlen |
+--------------------+---------+------+---------+----------+---------+
| utf8mb4_0900_bin   | utf8mb4 |  309 | Yes     | Yes      |       1 |
| utf8mb3_general_ci | utf8mb3 |   33 | Yes     | Yes      |       1 |
+--------------------+---------+------+---------+----------+---------+
```
