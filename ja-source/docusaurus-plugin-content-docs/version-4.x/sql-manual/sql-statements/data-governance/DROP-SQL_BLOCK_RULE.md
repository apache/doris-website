## デスクリプション

1つ以上のSQLブロッキングルールを削除します。複数のルールは、カンマで区切ることで一度に削除できます。

## Syntax

```sql
DROP SQL_BLOCK_RULE <rule_name>[, ...]
```
## 必須パラメータ

<rule_name>
削除対象のSQLブロックルールの名前。複数のルール名を指定する場合は、カンマで区切って指定できます。

## アクセス制御要件

このSQLコマンドを実行するユーザーは、少なくとも以下の権限を持っている必要があります：
| 権限 | オブジェクト | 備考                |
| :---------------- | :------------- | :---------------------------- |
| ADMIN        | User または Role   | ADMIN権限を持つユーザーまたはロールのみがDROP操作を実行できます。 |


## 例

`test_rule1`と`test_rule2`のブロックルールを削除

```sql
DROP SQL_BLOCK_RULE test_rule1, test_rule2;
```
