### 説明

このステートメントは、データベースid に基づいて対応するデータベース名を検索するために使用されます（管理者のみが使用）

## 構文

```sql
SHOW DATABASE <database_id>
```
## 必須パラメータ

**1. `<database_id>`**
> 対応するデータベースのID番号

## 戻り値

| カラム | 説明 |
|:---------|:-----------|
| DbName | データベース名 |

## 権限制御

このSQLコマンドを実行するユーザーは、少なくとも以下の権限を持つ必要があります：

| 権限 | オブジェクト | 備考 |
|:-----------|:-----|:--------------|
| ADMIN_PRIV | クラスター全体 | クラスター全体の管理者権限が必要 |

## 例

- データベースIDに対応するデータベース名を検索する

    ```sql
    SHOW DATABASE 10396;
    ```
    ```text
    +------------+
    | DbName     |
    +------------+
    | example_db |
    +------------+
    ```
