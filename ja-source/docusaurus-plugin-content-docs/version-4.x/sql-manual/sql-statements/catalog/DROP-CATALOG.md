## デスクリプション

このステートメントは、外部カタログを削除するために使用されます。

## Syntax

```sql
DROP CATALOG [IF EXISTS] <catalog_name>;
```
## 必須パラメータ

**1. `<catalog_name>`**
削除するカタログの名前。

## アクセス制御要件
| 権限 | オブジェクト | 注意事項 |
|:----------|:--------|:--------------------------------------------------------------------|
| DROP_PRIV | カタログ | 対応するカタログに対するDROP_PRIV権限が必要です。 |


## 例

1. カタログhiveを削除

   ```sql
   DROP CATALOG hive;
   ```
