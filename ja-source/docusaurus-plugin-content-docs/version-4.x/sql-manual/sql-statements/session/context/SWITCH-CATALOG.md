## 説明

このステートメントはカタログを切り替えるために使用されます。

## 構文

```sql
SWITCH <catalog_name>
```
## 必須パラメータ

**1. `<catalog_name>`**
> 切り替え先のカタログの名前。

## アクセス制御要件

| 権限  | オブジェクト     | 注記                                  |
|------------|------------|----------------------------------------|
| SELECT_PRIV  | カタログ    | 切り替え先のカタログに対してSELECT_PRIV権限が必要です。 |

## 例

1. カタログ`hive`に切り替える

   ```sql
   SWITCH hive;
   ```
