## はじめに
Quick BIは、企業が印象的な視覚的分析を迅速に構築するのに役立つデータウェアハウスベースのビジネスインテリジェンスツールです。Quick BIは、MySQL、Oracle、SQL サーバー、Apache Dorisなどのデータベース、およびExcel、CSV、JSONなどのファイル形式を含む、さまざまなデータソースをサポートします。テーブル、チャート、マップなどの豊富な視覚化コンポーネントを提供し、ユーザーが簡単なドラッグアンドドロップ操作でデータ可視化を容易に実現できます。

## データ接続とアプリケーション
1. Quick BIにログインし、ワークスペースを作成します。
2. 現在のワークスペース下のData Sourceをクリックします。

   ![create workspace](/images/bi-quickbi-en-1.png)

3. 既に作成されたデータソースでApache Dorisを選択し、対応するApache Doris接続情報を入力します。

   ![Doris information](/images/bi-quickbi-en-2.png)

4. 接続が成功すると、データソースリストに作成したデータソースが表示されます。

   ![data source](/images/bi-quickbi-en-3.png)

5. 作成したデータソースでデータセットを作成します。TPC-Hデータセットを例として使用します。データセットが作成された後、対応するレポートを設定できます。

   ![Doris table](/images/bi-quickbi-en-4.png)
