# Config Dir

FEとBEの設定ファイルディレクトリは`conf/`です。このディレクトリは、デフォルトのfe.conf、be.confやその他のファイルを格納するだけでなく、共通設定ファイル格納ディレクトリとしても使用されます。

ユーザーは一部の設定ファイルをそこに格納でき、システムが自動的にそれらを読み込みます。

## hdfs-site.xmlとhive-site.xml

Dorisの一部の機能では、HDFS上のデータにアクセスしたり、Hive metastoreにアクセスしたりする必要があります。

該当機能のステートメントで様々なHDFS/Hiveパラメータを手動で入力することができます。

しかし、これらのパラメータは非常に多く、すべてを手動で入力するのは非常に面倒です。

そのため、ユーザーはHDFSやHiveの設定ファイルhdfs-site.xml/hive-site.xmlを直接`conf/`ディレクトリに配置できます。Dorisは自動的にこれらの設定ファイルを読み込みます。

ユーザーがコマンドで入力する設定は、設定ファイル内の設定項目を上書きします。

このように、ユーザーは少量の設定を入力するだけでHDFS/Hiveへのアクセスを完了できます。
