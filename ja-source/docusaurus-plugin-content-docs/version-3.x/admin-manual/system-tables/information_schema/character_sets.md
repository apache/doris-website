## 概要

サポートされている文字セットを表示します。このテーブルはMySQLの動作との互換性のためにのみ使用され、Dorisで実際に使用される文字セットを真に反映するものではありません。

## データベース

`information_schema`

## テーブル情報

| カラム名             | 型           | 説明                                                         |
| -------------------- | ------------ | ------------------------------------------------------------ |
| CHARACTER_SET_NAME   | varchar(512) | 文字セットの名前                                             |
| DEFAULT_COLLATE_NAME | varchar(64)  | デフォルトの照合順序名                                       |
| DESCRIPTION          | varchar(64)  | 文字セットの詳細説明                                         |
| MAXLEN               | bigint       | 文字セット内の単一文字が占有する最大バイト数                 |
