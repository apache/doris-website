# Logout Action


## リクエスト

```
POST /rest/v1/logout
```
## 説明

Logout Actionは現在のログインからログアウトするために使用されます。

## パスパラメータ

なし

## クエリパラメータ

なし

## リクエストボディ

なし

{
	"msg": "OK",
	"code": 0
}
