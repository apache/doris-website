## デスクリプション

`job_name`で識別されるデータベース内で実行中の常駐データ同期ジョブを一時停止します。停止されたジョブは、ユーザーによって再開されるまで、最新の消費位置を保持しながらデータの同期を停止します。

## Syntax

```sql
PAUSE SYNC JOB [<db>.]<job_name>
```
## 必須パラメータ

**1. `<job_name>`**

> 一時停止する同期ジョブの名前を指定します。

## オプションパラメータ
**1. `<db>`**
> `[<db>.]`プレフィックスを使用してデータベースが指定された場合、そのデータベース内でジョブが検索されます。指定されない場合は、現在のデータベースが使用されます。

## アクセス制御要件

任意のユーザーまたはロールがこの操作を実行できます。

## 例

1. `job_name`という名前のデータ同期ジョブを一時停止します。

   ```sql
   PAUSE SYNC JOB `job_name`;
   ```
