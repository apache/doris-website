## 説明

このステートメントは、現在作成されているwarehouseを表示するために使用されます

## 構文

```sql
SHOW REPOSITORIES;
```
## 戻り値

| Field           | デスクリプション                              |
|-----------------|------------------------------------------|
| **RepoId**      | リポジトリの一意識別子（ID） |
| **RepoName**    | リポジトリの名前               |
| **CreateTime**  | リポジトリの作成時刻      |
| **IsReadOnly**  | リポジトリが読み取り専用かどうか。`false`は読み取り専用ではない、`true`は読み取り専用を意味する |
| **Location**    | リポジトリでデータをバックアップするために使用されるルートディレクトリ |
| **Broker**      | -                                        |
| **タイプ**        | リポジトリタイプ、現在S3とHDFSをサポート |
| **ErrMsg**      | リポジトリのエラーメッセージ。通常、エラーが発生しない場合は`NULL` |


## 例

作成されたリポジトリを表示：

```sql
SHOW REPOSITORIES;
```
```text
+--------+--------------+---------------------+------------+----------+--------+------+--------+
| RepoId | RepoName     | CreateTime          | IsReadOnly | Location | Broker | タイプ | ErrMsg |
+--------+--------------+---------------------+------------+----------+--------+------+--------+
| 43411  | example_repo | 2025-01-17 18:50:47 | false      | s3://rep1  | -      | S3   | NULL   |
+--------+--------------+---------------------+------------+----------+--------+------+--------+
```
