## デスクリプション

この文は作成されたリポジトリを削除するために使用されます。

## Syntax

```sql
DROP REPOSITORY <repo_name>;
```
## 必要なパラメータ
**<repo_name>**
> リポジトリの一意な名前。

## アクセス制御要件

| 権限               | オブジェクト                         | 備考                                               |
|:-------------------|:-----------------------------|:----------------------------------------------------|
| ADMIN_PRIV         | クラスタ全体の管理権限 | rootまたはスーパーユーザーのみがリポジトリを作成できます  |


## 使用上の注意
- リポジトリを削除してもDoris内のマッピングが削除されるのみで、実際のリポジトリデータは削除されません。削除後、同じLOCATIONを指定することでリポジトリを再度マップできます。

## 例

example_repoという名前のリポジトリを削除する場合：

```sql
DROP REPOSITORY `example_repo`;
```
