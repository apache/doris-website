## 説明

このステートメントはデータベースを削除するために使用されます。

## 構文

```sql
DROP DATABASE [IF EXISTS] <db_name> [FORCE];
```
## 必須パラメータ

** 1. `<db_name>`**
>  データベース名

## オプションパラメータ

** 1. `FORCE`**
>  ごみ箱に移動せずに強制削除

## 権限制御

このSQLコマンドを実行するユーザーは、最低限以下の権限を持つ必要があります：

| 権限         | オブジェクト    | 備考             |
|:-----------|:------|:---------------|
| DROP_PRIV | 対応するデータベース | 対応するデータベースに対する削除権限が必要です |


## 注意事項

DROP DATABASE FORCEを実行した場合、システムはデータベース内に未完了のトランザクションがあるかどうかを確認しません。データベースは直接削除され、復元できません。この操作は一般的に推奨されません。

## 例

- データベースdb_testの削除

    ```sql
    DROP DATABASE db_test;
    ```
