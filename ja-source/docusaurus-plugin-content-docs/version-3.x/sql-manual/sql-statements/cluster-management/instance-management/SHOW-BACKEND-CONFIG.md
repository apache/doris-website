## デスクリプション

BACKEND（すなわちBE）の設定項目とその現在の値を表示します。

## Syntax

```sql
[ ADMIN ] SHOW BACKEND CONFIG [ LIKE <config_key_pattern> ] [ FROM <backend_id> ]
```
## オプションパラメータ

**<config_key_pattern>**

> BE設定項目にマッチするワイルドカードパターンを提供します。マッチングルールはLIKE式と同じです。記述ルールについては、「マッチング式」セクションを参照してください。

**<backend_id>**

> BEのIDです。指定されたIDを持つBEの設定を表示するために使用されます。BEのIDはSHOW BACKENDSコマンドで取得できます。詳細については、「SHOW BACKENDS」コマンドを参照してください。

## 戻り値

- BackendId: BEのID
- Host: BEのホストアドレス
- Key: 設定項目の名前
- Value: 設定項目の対応する値
- タイプ: 設定値の型

## アクセス制御要件

このSQLコマンドを実行するユーザーは、少なくともADMIN_PRIV権限を持つ必要があります。

## 例

### すべての設定項目を照会

```sql
SHOW BACKEND CONFIG
```
結果:

```sql
+-----------+--------------+--------------------------------+-------+--------------------------+-----------+
| BackendId | Host         | Key                            | Value | タイプ                     | IsMutable |
+-----------+--------------+--------------------------------+-------+--------------------------+-----------+
| 12793     | 172.16.123.1 | LZ4_HC_compression_level       | 9     | int64_t                  | true      |
| 12793     | 172.16.123.1 | agent_task_trace_threshold_sec | 2     | int32_t                  | true      |
...
| 12794     | 172.16.123.2 | zone_map_row_num_threshold     | 20    | int32_t                  | true      |
+-----------+--------------+--------------------------------+-------+--------------------------+-----------+
```
### 指定されたBE IDのクエリ設定項目

```sql
SHOW BACKEND CONFIG FROM 12793
```
結果:

```sql
+-----------+--------------+--------------------------------+-------+--------------------------+-----------+
| BackendId | Host         | Key                            | Value | タイプ                     | IsMutable |
+-----------+--------------+--------------------------------+-------+--------------------------+-----------+
| 12793     | 172.16.123.1 | LZ4_HC_compression_level       | 9     | int64_t                  | true      |
| 12793     | 172.16.123.1 | agent_task_trace_threshold_sec | 2     | int32_t                  | true      |
...
| 12793     | 172.16.123.1 | zone_map_row_num_threshold     | 20    | int32_t                  | true      |
+-----------+--------------+--------------------------------+-------+--------------------------+-----------+
```
### 指定されたパターンに一致する設定項目のクエリ

```sql
SHOW BACKEND CONFIG LIKE '%compression_level%'
```
結果:

```sql
+-----------+--------------+--------------------------+-------+---------+-----------+
| BackendId | Host         | Key                      | Value | タイプ    | IsMutable |
+-----------+--------------+--------------------------+-------+---------+-----------+
| 12793     | 172.16.123.1 | LZ4_HC_compression_level | 9     | int64_t | true      |
| 12794     | 172.16.123.2 | LZ4_HC_compression_level | 9     | int64_t | true      |
+-----------+--------------+--------------------------+-------+---------+-----------+
```
