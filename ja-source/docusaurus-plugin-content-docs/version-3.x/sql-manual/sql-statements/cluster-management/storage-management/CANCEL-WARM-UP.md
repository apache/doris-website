## デスクリプション

Dorisで指定されたウォームアップジョブを終了するために使用します。

## Syntax

```sql
CANCEL WARM UP JOB WHERE id = <id>;
```
## 必須パラメータ

> 終了したいウォームアップジョブのidです。これは`SHOW WARM UP JOB`コマンドで照会できます。

## 権限制御

このSQLコマンドを実行するユーザーは、少なくともADMIN_PRIV権限を持っている必要があります。

## 例

`SHOW WARM UP JOB`を通じて、システムで現在実行中のウォームアップジョブを照会します：

```sql
SHOW WARM UP JOB
```
その結果は以下の通りです：

```C++
+----------------+---------------+---------+-------+-------------------------+-------------+----------+------------+--------+
| JobId          | ClusterName   | Status  | タイプ  | CreateTime              | FinishBatch | AllBatch | FinishTime | ErrMsg |
+----------------+---------------+---------+-------+-------------------------+-------------+----------+------------+--------+
| 90290165739458 | CloudCluster1 | RUNNING | TABLE | 2024-11-11 11:11:42.700 | 1           | 3        | NULL       |        |
+----------------+---------------+---------+-------+-------------------------+-------------+----------+------------+--------+
```
その後、`CANCEL WARM UP`を通じて終了します：

```C++
CANCEL WARM UP WHERE id = 90290165739458;
```
以下の内容が返される場合、指定されたidに対応するwarm-upジョブが存在しないことを示しています：

```C++
ERROR 1105 (HY000): errCode = 2, detailMessage = job id: 110 does not exist.
```
正常に戻った後、`SHOW WARM UP JOB`を再度使用して、ジョブステータスがRUNNINGからCANCELLEDに変更されたことを確認できます：

```C++
+----------------+---------------+-----------+-------+-------------------------+-------------+----------+-------------------------+-------------+
| JobId          | ClusterName   | Status    | タイプ  | CreateTime              | FinishBatch | AllBatch | FinishTime              | ErrMsg      |
+----------------+---------------+-----------+-------+-------------------------+-------------+----------+-------------------------+-------------+
| 90290165739458 | CloudCluster1 | CANCELLED | TABLE | 2024-11-11 11:11:42.700 | 1           | 3        | 2024-11-11 11:11:43.700 | user cancel |
+----------------+---------------+-----------+-------+-------------------------+-------------+----------+-------------------------+-------------+
```
