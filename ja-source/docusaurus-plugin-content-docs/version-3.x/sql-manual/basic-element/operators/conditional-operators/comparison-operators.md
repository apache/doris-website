## 説明

比較条件は、ある式と別の式を比較します。比較の結果は、TRUE、FALSE、またはUNKNOWNになります。

## 演算子の紹介

| 演算子 | 機能 | 例 |
| ------------------- | ----------------------------------------------------------- | ------------------- |
| `=` | 等価比較。比較のどちらかの側がUNKNOWNの場合、結果はUNKNOWNになります。 | `SELECT 1 = 1` |
| `<=>` | NULL安全等価比較。等価比較とは異なり、NULL安全等価はNULLを比較可能な値として扱います。両側がNULLの場合、TRUEを返します。片側のみがNULLの場合、FALSEを返します。この演算子はUNKNOWNを返すことはありません。 | `SELECT NULL <=> NULL` |
| `!=` `<>` | 不等価比較 | `SELECT 1 != 1` |
| `<` `>` | より大きいとより小さい比較 | `SELECT 1 > 1` |
| `<=` `>=` | 以上と以下の比較 | `SELECT 1 >= 1` |
| `<x> BETWEEN <y> AND <z>` | `<x> >= <y> and <x> <= <z>`と等価。`<y>`以上かつ`<z>`以下 | `SELECT 1 BETWEEN 0 AND 2` |
