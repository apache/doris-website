## 説明

情報はSHA2を使用してダイジェスト化されます。

## 構文

```sql
SHA2( <str>, <digest_length>)
```
## パラメータ

| parameter         | description                |
|-------------------|----------------------------|
| `<str>`           | 暗号化される内容                     |
| `<digest_length>` | サマリー長、224、256、384、512をサポート |


## Return Value
入力文字列のsha2値を返します

## Examples

```sql
select sha2('abc', 224), sha2('abc', 384), sha2(NULL, 512);
```
