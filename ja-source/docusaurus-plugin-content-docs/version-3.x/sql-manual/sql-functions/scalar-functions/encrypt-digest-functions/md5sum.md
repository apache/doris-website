## description

文字列のMD5 128ビットチェックサムを計算します

## Syntax

```sql
MD5SUM( <str> [ , <str> ... ] )
```
## パラメータ

|  parameter | description |
|---------| -- |
| `<str>` | 計算されるMD5値 |

## Return Value

複数の文字列のMD5値を返します

## Examples

```sql
select md5("abcd"),md5sum("ab","cd");
```
```text
+----------------------------------+----------------------------------+
| md5('abcd')                      | md5sum('ab', 'cd')               |
+----------------------------------+----------------------------------+
| e2fc714c4727ee9395f324cd2e7f331f | e2fc714c4727ee9395f324cd2e7f331f |
+----------------------------------+----------------------------------+
```
