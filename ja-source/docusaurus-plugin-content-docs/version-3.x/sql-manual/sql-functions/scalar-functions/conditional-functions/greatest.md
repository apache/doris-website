## 説明

複数の式を比較し、その中で最大の値を返します。引数のいずれかが`NULL`の場合、関数は`NULL`を返します。

## 構文

```sql
GREATEST(<expr> [, ...])
```
## パラメータ

| Parameter   | デスクリプション |
|------------|-------------|
| `<expr>`  | 比較する式のリスト。`TINYINT`、`SMALLINT`、`INT`、`BIGINT`、`LARGEINT`、`FLOAT`、`DOUBLE`、`STRING`、`DATETIME`、および`DECIMAL`タイプをサポートします。 |

## Return Value

- 与えられた式の中で最大の値を返します。
- いずれかの引数が`NULL`の場合、`NULL`を返します。

## Examples

```sql
SELECT GREATEST(-1, 0, 5, 8);
```
```text
+-----------------------+
| GREATEST(-1, 0, 5, 8) |
+-----------------------+
|                     8 |
+-----------------------+
```
```sql
SELECT GREATEST(-1, 0, 5, NULL);
```
```text
+--------------------------+
| GREATEST(-1, 0, 5, NULL) |
+--------------------------+
| NULL                     |
+--------------------------+
```
```sql
SELECT GREATEST(6.3, 4.29, 7.6876);
```
```text
+-----------------------------+
| GREATEST(6.3, 4.29, 7.6876) |
+-----------------------------+
|                      7.6876 |
+-----------------------------+
```
```sql
SELECT GREATEST('2022-02-26 20:02:11', '2020-01-23 20:02:11', '2020-06-22 20:02:11');
```
```text
+-------------------------------------------------------------------------------+
| GREATEST('2022-02-26 20:02:11', '2020-01-23 20:02:11', '2020-06-22 20:02:11') |
+-------------------------------------------------------------------------------+
| 2022-02-26 20:02:11                                                           |
+-------------------------------------------------------------------------------+
```
