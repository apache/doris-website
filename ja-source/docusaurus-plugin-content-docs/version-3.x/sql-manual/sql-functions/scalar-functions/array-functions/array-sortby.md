## デスクリプション

最初にkey列を昇順に並べ替え、次にこの順序でソートされたsrc列の対応する列を結果として返します。
入力配列srcがNULLの場合、NULLを返します。
入力配列keyがNULLの場合、srcが返される順序は変更されません。
入力配列keyの要素にNULLが含まれている場合、出力されるソート済み配列ではNULLが最初に配置されます。

## Syntax

```sql
ARRAY_SORTBY(<src>, <key>)
ARRAY_SORTBY(<lambda>, <arr> [, ...])
```
## パラメータ

| Parameter | デスクリプション | 
| --- | --- |
| `<lambda>` | 入力パラメータが指定された配列の列数と一致する必要があるラムダ式。この式は有効なスカラー関数を実行できますが、集計関数はサポートしていません。 |
| `<arr>` | ARRAY配列 |

## Return Value

ソートされたARRAY型の結果を返します。

## Example

```sql
select array_sortby(['a','b','c'],[3,2,1]);
```
```text
+----------------------------------------------------+
| array_sortby(ARRAY('a', 'b', 'c'), ARRAY(3, 2, 1)) |
+----------------------------------------------------+
| ['c', 'b', 'a']                                    |
+----------------------------------------------------+
```
```sql
select array_sortby([1,2,3,4,5],[10,5,1,20,80]);
```
```text
+-------------------------------------------------------------+
| array_sortby(ARRAY(1, 2, 3, 4, 5), ARRAY(10, 5, 1, 20, 80)) |
+-------------------------------------------------------------+
| [3, 2, 1, 4, 5]                                             |
+-------------------------------------------------------------+
```
