## 説明

左の配列と右の配列が共通の要素を含むかどうかを判定します

## 構文

```sql
ARRAYS_OVERLAP(<left>, <right>)
```
## パラメータ

| Parameter | デスクリプション |
|--|--|
| `<left>` | 判定対象の配列 |
| `<right>` | 判定対象の配列 |

## Return Value

判定結果を返します：1：left配列とright配列に共通の要素がある；0：left配列とright配列に共通の要素がない；NULL：left配列またはright配列がNULL、またはleft配列とright配列のいずれかの要素がNULL

## Example

```sql
SELECT ARRAYS_OVERLAP(['a', 'b', 'c'], [1, 2, 'b']);
```
```text
+--------------------------------------------------+
| arrays_overlap(['a', 'b', 'c'], ['1', '2', 'b']) |
+--------------------------------------------------+
|                                                1 |
+--------------------------------------------------+
```
