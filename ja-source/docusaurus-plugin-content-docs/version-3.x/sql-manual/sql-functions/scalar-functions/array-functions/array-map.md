## description

lambda式を入力パラメータとして使用し、他の入力ARRAY パラメータの内部データに対応する式を計算します。
lambda式に入力されるパラメータの数は1つ以上で、入力配列列数と一致している必要があります。
lambdaではスカラー関数を実行でき、集約関数はサポートされていません。

## Syntax

```sql
ARRAY_MAP(lambda, <arr> [ , <arr> ... ] )
```
## パラメータ

| パラメータ | 説明 |
| --- |---|
| `lambda` | 1つ以上の入力パラメータを持つラムダ式で、入力配列の数と一致する必要があります。ラムダは有効なスカラー関数を実行できますが、集計関数はサポートしていません。 |
| `<arr>` | ARRAY配列 |

## 戻り値

ラムダ式を通じて処理されたARRAY。
## example

```sql
CREATE TABLE array_test2 (
                             id INT,
                             c_array1 ARRAY<INT>,
                             c_array2 ARRAY<INT>
)
    duplicate key (id)
distributed by hash(id) buckets 1
properties(
    'replication_num' = '1'
    );
INSERT INTO array_test2 (id, c_array1, c_array2) VALUES
                                                     (1, [1, 2, 3, 4, 5], [10, 20, -40, 80, -100]),
                                                     (2, [6, 7, 8], [10, 12, 13]),
                                                     (3, [1], [-100]),
                                                     (4, NULL, NULL);
select *, array_map(x->x,[1,2,3]) from array_test2 order by id;
```
