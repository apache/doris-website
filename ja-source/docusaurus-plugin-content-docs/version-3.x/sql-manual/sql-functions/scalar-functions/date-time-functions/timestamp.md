## 説明

TIMESTAMP関数には2つの用途があります：

1. datetime文字列をDATETIME型に変換する
2. 2つの引数を組み合わせてDATETIME型にする

## 構文

```sql
TIMESTAMP(string)
TIMESTAMP(date, time)
```
## パラメータ

| Parameter | デスクリプション                                      |
|-----------|--------------------------------------------------|
| `string`      | 日時文字列 |
| `date`     | 日付値。DATE型または適切にフォーマットされた日付文字列を指定可能 |
| `time`     | 時刻値。TIME型または適切にフォーマットされた時刻文字列を指定可能 |

## Return Value

DATETIME型の値を返します。

## Example

```sql
-- Convert a string to DATETIME
SELECT TIMESTAMP('2019-01-01 12:00:00');
```
```text
+------------------------------------+
| timestamp('2019-01-01 12:00:00')   |
+------------------------------------+
| 2019-01-01 12:00:00                |
+------------------------------------+
```
```sql
-- Combine date and time into DATETIME
SELECT TIMESTAMP('2019-01-01', '12:00:00');
```
```text
+----------------------------------------+
| timestamp('2019-01-01', '12:00:00')    |
+----------------------------------------+
| 2019-01-01 12:00:00                    |
+----------------------------------------+
```
