## デスクリプション
指定された日付または時刻の値から指定された週数を減算します（つまり、weeks * 7日を減算します）。

## Syntax

```sql
WEEKS_SUB(<date_value>, <week_period>)
```
## 必須パラメータ
| Parameter       | デスクリプション                                                                                         |
|-----------------|-----------------------------------------------------------------------------------------------------|
| `date_value`    | `DATE`または`DATETIME`型の入力値。                                                              |
| `week_period`   | 整数。減算する週数を表す（正の値で減少、負の値で増加）。 |


## 例

1. datetime `2020-02-02 02:02:02`から1週間を減算する

    ```sql
    select weeks_sub("2020-02-02 02:02:02", 1);
    ```
    ```text
    +-------------------------------------+
    | weeks_sub('2020-02-02 02:02:02', 1) |
    +-------------------------------------+
    | 2020-01-26 02:02:02                 |
    +-------------------------------------+
    ```
2. 日付`2020-02-02`から1週間を引く

    ```sql
    select weeks_sub("2020-02-02", 1);
    ```
    ```text
    +--------------------------------------------+
    | weeks_sub(cast('2020-02-02' as DATEV2), 1) |
    +--------------------------------------------+
    | 2020-01-26                                 |
    +--------------------------------------------+
    ```
