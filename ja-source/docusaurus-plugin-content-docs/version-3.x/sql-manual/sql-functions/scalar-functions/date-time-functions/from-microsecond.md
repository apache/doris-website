## 説明

この関数は、Unix タイムスタンプ（マイクロ秒単位）を`DATETIME`値に変換します。


## 構文

```sql
FROM_MICROSECOND(<unix_timestamp>)
```
## パラメータ

| Parameter          | デスクリプション                                                                                                 |
|--------------------|-------------------------------------------------------------------------------------------------------------|
| `<unix_timestamp>` | 必須。1970-01-01 00:00:00 UTCからの経過マイクロ秒数を表すUnixタイムスタンプ。 |

## Return Value

- 指定されたUnixタイムスタンプに対応する日付と時刻を表すDATETIME値を返します。
- `<unix_timestamp>`がNULLの場合、この関数はNULLを返します。
- `<unix_timestamp>`が有効な範囲外の場合、この関数はエラーを返します。

## Example

```sql
SELECT FROM_MICROSECOND(1700000000000000);
```
```text
+------------------------------------+
| from_microsecond(1700000000000000) |
+------------------------------------+
| 2023-11-15 06:13:20                |
+------------------------------------+
```
