# TO_ISO8601

## デスクリプション

datetime値をISO8601形式の文字列に変換します。

## Syntax

```sql
TO_ISO8601(<dt>)
```
## パラメータ

| Parameter | デスクリプション                                      |
|-----------|--------------------------------------------------|
| `<dt>`      | 入力される日時の値。DATETIME型またはDATE型を指定可能 |

## Return Value

ISO8601形式の日時を表すVARCHAR型の値を返します。

## Example

```sql
SELECT TO_ISO8601('2020-01-01 12:30:45');
```
```text
+-------------------------------------+
| to_iso8601('2020-01-01 12:30:45.0') |
+-------------------------------------+
| 2020-01-01T12:30:45                 |
+-------------------------------------+
```
