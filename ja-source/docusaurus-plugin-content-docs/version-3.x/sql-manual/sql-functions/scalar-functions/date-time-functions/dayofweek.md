## 説明

日付の曜日インデックスを返します。日曜日が1、月曜日が2、土曜日が7となります。

## 構文

```sql
DAYOFWEEK(<dt>)
```
## パラメータ

| Parameter | デスクリプション |
| -- | -- |
| `<dt>` | 計算される日付式 |

## Return Value

日付の曜日インデックスを返します。

## Examples

```sql
select dayofweek('2019-06-25');
```
```text
+----------------------------------+
| dayofweek('2019-06-25 00:00:00') |
+----------------------------------+
|                                3 |
+----------------------------------+
```text
```sql
select dayofweek(cast(20190625 as date)); 
```
```text
+-----------------------------------+
| dayofweek(CAST(20190625 AS DATE)) |
+-----------------------------------+
|                                 3 |
+-----------------------------------+
```
