## 説明

入力されたdatetime値から指定した年数を減算した結果となる新しいdatetime値を返します。

## 構文

```sql
YEARS_SUB(<date>, <years>)
```
## パラメータ

| Parameter | デスクリプション                                      |
|-----------|--------------------------------------------------|
| `<date>`      | 入力される日時の値で、DATETIME型またはDATE型を指定できます |
| `<years>`     | 減算する年数で、INT型です         |

## Return Value

入力された`<date>`と同じ型（DATETIMEまたはDATE）の値を返します。これは、入力された日時から指定された年数を減算した後の時間値を表します。

## Example

```sql
SELECT YEARS_SUB('2020-02-02 02:02:02', 1);
```
```text
+-------------------------------------+
| years_sub('2020-02-02 02:02:02', 1) |
+-------------------------------------+
| 2019-02-02 02:02:02                 |
+-------------------------------------+
```
