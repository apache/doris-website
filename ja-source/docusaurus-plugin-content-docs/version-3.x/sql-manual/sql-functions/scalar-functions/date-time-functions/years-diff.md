## 説明

2つのdatetime値間の年数の差を計算します。

## 構文

```sql
YEARS_DIFF(<enddate>, <startdate>)
```
## パラメータ

| Parameter | デスクリプション                                      |
|-----------|--------------------------------------------------|
| `<enddate>`      | 終了日。DATETIME型またはDATE型を指定可能 |
| `<startdate>`     | 開始日。DATETIME型またはDATE型を指定可能 |

## Return Value

INT型の値を返します。この値は2つの日付間の年数を表します。

## Example

```sql
SELECT YEARS_DIFF('2020-12-25', '2019-10-25');
```
```text
+----------------------------------------------------------+
| years_diff('2020-12-25 00:00:00', '2019-10-25 00:00:00') |
+----------------------------------------------------------+
|                                                        1 |
+----------------------------------------------------------+
```
