## 説明
この関数は、指定された日付またはdatetime値に対して、指定された四半期数を加算または減算し、結果の日付を返します。

## 構文

```sql
QUARTERS_ADD(<date/datetime>, <quarters>)
```
## パラメータ

| Parameter         | デスクリプション                                                                                                           |
|-------------------|-----------------------------------------------------------------------------------------------------------------------|
| `<date/datetime>` | 入力となる日付またはdatetime値。DATEまたはDATETIME型をサポートします。                                                    |
| `<quarters>`      | 加算または減算する四半期の数。正の整数は四半期を加算し、負の整数は四半期を減算します。 |

## Return Value
- 入力された日付型と一致する日付値を返します。
- <date/datetime>がNULLの場合、関数はNULLを返します。
- <date/datetime>が無効な日付（例：0000-00-00）の場合、関数はNULLを返します。

### Example

```sql
 select quarters_add("2020-01-31 02:02:02", 1);
```
```text
+---------------------------------------------------------------+
| quarters_add(cast('2020-01-31 02:02:02' as DATETIMEV2(0)), 1) |
+---------------------------------------------------------------+
| 2020-04-30 02:02:02                                           |
+---------------------------------------------------------------+
```
