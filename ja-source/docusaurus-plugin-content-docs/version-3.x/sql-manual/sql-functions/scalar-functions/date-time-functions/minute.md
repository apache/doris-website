## 説明

datetime値から分の部分を抽出します。返される値は0から59の範囲です。

## 構文

```sql
MINUTE(<datetime>)
```
## パラメータ

| Parameter | デスクリプション                                      |
|-----------|--------------------------------------------------|
| `<datetime>`  | 入力する日時値。DATE、DATETIME、DATETIMEV2、またはTIME型を指定できます |

## Return Value

分の値を表すINT型を返します。範囲は0-59です。

## Example

```sql
SELECT MINUTE('2018-12-31 23:59:59');
```
```text
+------------------------------------------------------+
| minute(cast('2018-12-31 23:59:59' as DATETIMEV2(0))) |
+------------------------------------------------------+
|                                                   59 |
+------------------------------------------------------+
```
**注意:**
- 入力パラメータは様々な時間関連の型を使用できます。
- 戻り値は常に0から59までの整数です。
- 入力パラメータがNULLの場合、この関数はNULLを返します。
