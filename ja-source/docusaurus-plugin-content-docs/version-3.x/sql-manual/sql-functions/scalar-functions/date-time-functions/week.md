## week
### 説明
#### 構文

`INT WEEK(DATE date[, INT mode])`

dateの週番号を返します。mode引数の値はデフォルトで0です。
次の表はmode引数の動作を説明しています。

|Mode |週の最初の日      |範囲   |週1は最初の週…                |
|:----|:-----------------|:------|:-----------------------------|
|0    |Sunday            |0-53   |この年でSundayがある週        |
|1    |Monday            |0-53   |この年で4日以上ある週         |
|2    |Sunday            |1-53   |この年でSundayがある週        |
|3    |Monday            |1-53   |この年で4日以上ある週         |
|4    |Sunday            |0-53   |この年で4日以上ある週         |
|5    |Monday            |0-53   |この年でMondayがある週        |
|6    |Sunday            |1-53   |この年で4日以上ある週         |
|7    |Monday            |1-53   |この年でMondayがある週        |

パラメータはDateまたはDatetime型です

### 例

```
mysql> select week('2020-1-1');
+------------------+
| week('2020-1-1') |
+------------------+
|                0 |
+------------------+
```
```
mysql> select week('2020-7-1',1);
+---------------------+
| week('2020-7-1', 1) |
+---------------------+
|                  27 |
+---------------------+
```
