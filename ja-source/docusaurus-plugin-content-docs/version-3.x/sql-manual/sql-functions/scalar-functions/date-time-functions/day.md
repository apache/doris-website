## 説明

日付から日の情報を取得します。戻り値の範囲は1から31です。

## エイリアス

- dayofmonth

## 構文

```sql
DAY(<dt>)
```
## パラメータ

| Parameter | デスクリプション |
| -- | -- |
| <`dt`> | 有効な日付式 |

## Return Value

指定された日付から日の情報を返します。

## Examples

```sql
select day('1987-01-31');
```
```text
+----------------------------+
| day('1987-01-31 00:00:00') |
+----------------------------+
|                         31 |
+----------------------------+
```
