## 説明

提供されたテキストのURL encodingをUTF-8 encodingを使用して完了します。通常、URLの一部として渡されるパラメータ情報をencodeするために使用されます。

## 構文

```sql
URL_ENCODE(  <str>  ) 
```
## 必須パラメータ

## 必須パラメータ
| パラメータ | デスクリプション |
|------|------|
| `<str>` | エンコードされる文字列 |

## 戻り値

UTF-8エンコーディングにより提供されたテキストのURLエンコーディングが完了します

## 例

```sql
select  URL_ENCODE('Doris Q&A');
```
```sql
+-------------------------+
| url_encode('Doris Q&A') |
+-------------------------+
| Doris+Q%26A             |
+-------------------------+

```
