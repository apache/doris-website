## 説明

RTRIM関数は、文字列の右側（末尾）から連続するスペースまたは指定された文字を削除するために使用されます。

## 構文

```sql
RTRIM( <str> [, <trim_chars> ] )
```
## パラメータ

| Parameter      | デスクリプション                                                                                                                                                                                                                                                                      |
|----------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `<str>`        | トリムが必要な文字列。                                                                                                                                                                                                                                                             |
| `<trim_chars>` | オプションパラメータ。このパラメータが指定された場合、RTRIM関数は`<str>`の右側に現れる`<trim_chars>`内のすべての文字を削除します。このパラメータが指定されない場合、RTRIM関数は`<str>`の右側のスペース文字のみを削除します。 |

## Return Value

末尾のスペースまたは`<trim_chars>`が削除された文字列を返します。特殊ケース：

- いずれかのParameterがNULLの場合、NULLが返されます。


## Examples

```sql
SELECT rtrim('ab d   ') str;
```
```text
+------+
| str  |
+------+
| ab d |
+------+
```
```sql
SELECT rtrim('ababccaab','ab') str;
```
```text
+---------+
| str     |
+---------+
| ababcca |
+---------+
```
