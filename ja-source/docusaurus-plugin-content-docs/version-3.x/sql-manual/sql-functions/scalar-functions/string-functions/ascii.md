## 説明

文字列の最初の文字のASCIIコードを返します

## 構文

```sql
ASCII ( <str> )
```
## パラメータ

| Parameter | デスクリプション |
|-----------|-------------------------|
| `<str>`   | 最初の文字のASCIIコードを計算する必要がある文字列 |

## 戻り値

パラメータ <str> の最初の文字のASCIIコード

## 例

```sql
SELECT ASCII('1'),ASCII('234')
```
```text
+------------+--------------+
| ascii('1') | ascii('234') |
+------------+--------------+
|         49 |           50 |
+------------+--------------+
```
