## デスクリプション

指定された数のスペースで構成される文字列を生成します。

## Syntax

```sql
SPACE ( <len> )
```
## パラメータ

| パラメータ | 説明                    |
|-----------|------------------------|
| `<len>`   | 生成するスペースの数 |

## 戻り値

指定された数のスペースで構成される文字列を返します。特殊なケース：

- いずれかのパラメータがNULLの場合、NULLが返されます。
- `<len>`が0未満の場合、空文字列が返されます。

## 例

```sql
SELECT space(10);
```
```text
+------------+
| space(10)  |
+------------+
|            |
+------------+
```
```sql
SELECT space(null);
```
```text
+-------------+
| space(NULL) |
+-------------+
| NULL        |
+-------------+
```
