## デスクリプション

指定されたインデックスの文字列を返します。特別なケース：

- 指定されたインデックスに文字列が存在しない場合、NULLが返されます。

## Syntax

```sql
ELT ( <pos> , <str> [ , <str> ] )
```
## パラメータ

| Parameter | デスクリプション |
|-----------|------------|
| `<pos>`   | 指定されたインデックス値 |
| `<str>`   | インデックス対象の文字列 |

## Return value

Parameter `<str>` インデックス対象の文字列。特殊なケース：

- 指定されたインデックスに文字列が存在しない場合、NULLが返されます。

## Example

```sql
SELECT ELT(1, 'aaa', 'bbb'),ELT(2, 'aaa', 'bbb'), ELT(0, 'aaa', 'bbb'),ELT(2, 'aaa', 'bbb')
```
```text
+----------------------+----------------------+----------------------+----------------------+
| elt(1, 'aaa', 'bbb') | elt(2, 'aaa', 'bbb') | elt(0, 'aaa', 'bbb') | elt(2, 'aaa', 'bbb') |
+----------------------+----------------------+----------------------+----------------------+
| aaa                  | bbb                  | NULL                 | bbb                  |
+----------------------+----------------------+----------------------+----------------------+
```
