## デスクリプション

パラメータ内のすべての文字列を小文字に変換します。

## Alias

- LOWER

## Syntax

```sql
LCASE ( <str> )
LOWER ( <str> )
```
## パラメータ

| Parameter | デスクリプション |
|-----------|--------------|
| `<str>`   | 小文字に変換される文字列 |

## Return Value

Parameter `<str>` 小文字に変換された文字列

## Example

```sql
SELECT LCASE("AbC123"),LOWER("AbC123")
```
```text
+-----------------+-----------------+
| lower('AbC123') | lower('AbC123') |
+-----------------+-----------------+
| abc123          | abc123          |
+-----------------+-----------------+
```
```sql
SELECT LOWER("ҚAZAҚ123"),LCASE("ҒАРЫШ");
```
```text
+---------------------+---------------------+
| LOWER("ҚAZAҚ123")   | LCASE("ҒАРЫШ")      |
+---------------------+---------------------+
| қazaқ123            | ғарыш               |
+---------------------+---------------------+
```
