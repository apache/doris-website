## デスクリプション

TO_BASE64関数は、入力文字列をBase64エンコード形式に変換するために使用されます。Base64エンコーディングは、任意のバイナリデータを64文字で構成される文字列に変換できます。

## Syntax

```sql
TO_BASE64(<str>)
```
## パラメータ
| Parameter | デスクリプション                                    |
| --------- | ---------------------------------------------- |
| `<str>` | Base64エンコードする文字列。型: VARCHAR |

## Return Value

VARCHAR型を返し、Base64エンコードされた文字列を表します。

特殊なケース:
- 入力がNULLの場合、NULLを返します
- 入力が空文字列の場合、空文字列を返します

## Examples

1. 単一文字のエンコード

```sql
SELECT to_base64('1');
```
```text
+----------------+
| to_base64('1') |
+----------------+
| MQ==           |
+----------------+
```
2. 複数文字エンコーディング

```sql
SELECT to_base64('234');
```
```text
+------------------+
| to_base64('234') |
+------------------+
| MjM0             |
+------------------+
```
