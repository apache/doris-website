## 説明

文字列内のバイト数を返します。

## 構文

```sql
LENGTH ( <str> )
```
## パラメータ

| Parameter | デスクリプション |
|-----------| --------------- |
| `<str>`   | バイト数を計算する必要がある文字列 |

## Return Value

文字列 `<str>` のバイト数。

## Example

```sql
SELECT LENGTH("abc"),length("中国")
```
```text
+---------------+------------------+
| length('abc') | length('中国')   |
+---------------+------------------+
|             3 |                6 |
+---------------+------------------+
```
