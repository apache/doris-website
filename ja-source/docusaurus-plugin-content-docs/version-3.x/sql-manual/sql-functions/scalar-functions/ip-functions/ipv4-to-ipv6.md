## 説明
ipv4形式のアドレスをipv6形式のアドレスに変換します。

## 構文

```sql
IPV4_TO_IPV6(<ipv4>)
```
## パラメータ
| Parameter | デスクリプション                                      |
|-----------|--------------------------------------------------|
| `<ipv4>`      | IPv4形式のアドレス  |

## Return Value
変換されたIPv6形式のアドレスを返します。

## Example

```sql
select ipv6_num_to_string(ipv4_to_ipv6(to_ipv4('192.168.0.1')));
```
```text
+----------------------------------------------------------+
| ipv6_num_to_string(ipv4_to_ipv6(to_ipv4('192.168.0.1'))) |
+----------------------------------------------------------+
| ::ffff:192.168.0.1                                       |
+----------------------------------------------------------+
```
