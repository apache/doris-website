## 説明
この関数は、IPv4アドレスの文字列形式を受け取り、ipv4_string_to_numが返す値とバイナリ的に等しい値であるIPv4型の値を返すipv4_string_to_numのような関数です。

## 構文

```sql
TO_IPV4_OR_DEFAULT(<ipv4_str>)
```
## パラメータ
| Parameter | デスクリプション                                      |
|-----------|--------------------------------------------------|
| `<ipv4_str>`      | String型のIPv4アドレス |


## Return Value
IPv4型の値を返します。
- IPv4アドレスの形式が無効な場合、0.0.0.0（IPv4として0）を返します。


## Example

```sql
SELECT to_ipv4_or_default('255.255.255.255'), to_ipv4_or_default('.'), to_ipv4_or_default(NULL);
```
```text
+---------------------------------------+-------------------------+--------------------------+
| to_ipv4_or_default('255.255.255.255') | to_ipv4_or_default('.') | to_ipv4_or_default(NULL) |
+---------------------------------------+-------------------------+--------------------------+
| 255.255.255.255                       | 0.0.0.0                 | 0.0.0.0                  |
+---------------------------------------+-------------------------+--------------------------+
```
