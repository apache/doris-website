## 説明
IPv6アドレスの文字列形式をIPv6型に変換します。これはipv6_string_to_numによって返される値とバイナリ等価です。

## 構文

```sql
TO_IPV6(<ipv6_str>)
```
## パラメータ
| Parameter | デスクリプション                                      |
|-----------|--------------------------------------------------|
| `<ipv6_str>`      | String型のIPv6アドレス |


## Return Value
IPv6型の値を返します。
- IPv6アドレスの形式が無効な場合は、例外をスローします


## Example

```sql
SELECT to_ipv6('::'),to_ipv6('2001:1b70:a1:610::b102:2');
```
```text
+---------------+-------------------------------------+
| to_ipv6('::') | to_ipv6('2001:1b70:a1:610::b102:2') |
+---------------+-------------------------------------+
| ::            | 2001:1b70:a1:610::b102:2            |
+---------------+-------------------------------------+
```
