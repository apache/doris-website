## 説明

Dorisが接続している現在のユーザー名とIPを取得します。

## 構文

```sql
USER()
```
## 戻り値

Dorisが接続している現在のユーザー名とIPを返します。
形式:`<user_name>@<ip>`

## 例

```sql
select user();
```
```text
+---------------------+
| user()              |
+---------------------+
| 'root'@'10.244.2.5' |
+---------------------+
```
