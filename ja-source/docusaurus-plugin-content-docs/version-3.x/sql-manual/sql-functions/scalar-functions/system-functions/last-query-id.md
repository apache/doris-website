## 説明

現在のユーザーの最後のクエリのクエリIDを取得します。

## 構文

```sql
LAST_QUERY_ID()
```
## 戻り値

現在のユーザーの最後のクエリのクエリIDを返します。

## 例

```sql
select last_query_id();
```
```text
+-----------------------------------+
| last_query_id()                   |
+-----------------------------------+
| 128f913c3ec84a1e-b834bd0262cc090a |
+-----------------------------------+
```
