## デスクリプション

`x` の逆正接を返します。ここで `x` はラジアン単位です。

## Syntax

```sql
ATAN(<x>)
```
## パラメータ

| Parameter | デスクリプション |
| -- | -- |
| `<x>` | atan値を計算する対象の値 |

## 戻り値

パラメータ`x`のatan値。

## 例

```sql
select atan(0);
```
```text
+-----------+
| atan(0.0) |
+-----------+
|         0 |
+-----------+
```
```sql
select atan(2);
```
```text
+--------------------+
| atan(2.0)          |
+--------------------+
| 1.1071487177940904 |
+--------------------+
```
