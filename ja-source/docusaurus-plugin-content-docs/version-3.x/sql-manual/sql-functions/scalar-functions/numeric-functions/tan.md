## 説明

xの正接を返します。ここでxはラジアン単位の値です。

## 構文

```sql
TAN(<x>)
```
## パラメータ

| Parameter | デスクリプション |
| -- | -- |
| `<x>` | 正接を計算する値 |

## Return Value

xの正接を返します。

## Example

```sql
select tan(0),tan(1),tan(-1);
```
```text
+------------------------+------------------------+-------------------------+
| tan(cast(0 as DOUBLE)) | tan(cast(1 as DOUBLE)) | tan(cast(-1 as DOUBLE)) |
+------------------------+------------------------+-------------------------+
|                      0 |     1.5574077246549023 |     -1.5574077246549023 |
+------------------------+------------------------+-------------------------+
```
