## 説明

`x`の双曲線余弦を返します。

## 構文

`COSH(<x>)`

## パラメータ

| Parameter | デスクリプション |
| -- | -- |
| `<x>` | 双曲線余弦を計算する値 |

## 戻り値

パラメータ`x`の双曲線余弦

## 例

```sql
select cosh(0);
```
```text
+-------------------------+
| cosh(cast(0 as DOUBLE)) |
+-------------------------+
|                     1.0 |
+-------------------------+
```
```sql
select cosh(1);
```
```text
+-------------------------+
| cosh(cast(1 as DOUBLE)) |
+-------------------------+
|       1.543080634815244 |
+-------------------------+
```
```sql
select cosh(-1);
```
```text
+--------------------------+
| cosh(cast(-1 as DOUBLE)) |
+--------------------------+
|        1.543080634815244 |
+--------------------------+
```
