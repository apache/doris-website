## デスクリプション

整数型における a を b で割った余りを求めます。浮動小数点型の場合は、fmod関数を使用してください。

## Syntax

```sql
MOD(<col_a> , <col_b>)
```
## パラメータ

| Parameter | デスクリプション |
|-----------|------------|
| `<col_a>`   | 被除数 |
| `<col_b>`   | 除数は0であってはいけません |

## Return value

整数型を返します。特殊なケース：

col_a IS NULL または col_b IS NULL の場合、NULLを返します。

## Example

```sql
select mod(10, 3);
```
```text
+----------+
| (10 % 3) |
+----------+
|        1 |
+----------+
```
```sql
select mod(10, 0);
```
```text
+----------+
| (10 % 0) |
+----------+
|     NULL |
+----------+
```
