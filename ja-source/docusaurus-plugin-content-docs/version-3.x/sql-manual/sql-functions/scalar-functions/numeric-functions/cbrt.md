## 説明

パラメータの立方根を計算します

## 構文

```sql
CBRT(<a>)
```
## パラメータ

| Parameter | デスクリプション |
| -- | -- |
| `<a>` | 浮動小数点パラメータ |

## Return Value

パラメータ `<a>` の立方根、浮動小数点数。

## Examples

```sql
select cbrt(0);
```
```text
+-------------------------+
| cbrt(cast(0 as DOUBLE)) |
+-------------------------+
|                     0.0 |
+-------------------------+
```
```sql
select cbrt(-111);
```
```text
+----------------------------+
| cbrt(cast(-111 as DOUBLE)) |
+----------------------------+
|         -4.805895533705333 |
+----------------------------+
```
```sql
select cbrt(1234);
```
```text
+----------------------------+
| cbrt(cast(1234 as DOUBLE)) |
+----------------------------+
|         10.726014668827325 |
+----------------------------+
```
