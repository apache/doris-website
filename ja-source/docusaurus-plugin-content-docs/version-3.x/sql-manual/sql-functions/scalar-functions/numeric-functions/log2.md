## 説明

`x` の底が `2` の自然対数を返します。

## 構文

```sql
LOG2(<x>)
```
## パラメータ

| Parameter | デスクリプション |
|-----------|------------|
| `<x>`   | 真数は0より大きい値である必要があります |

## Return value

浮動小数点数を返します。

- xがNULLの場合: `NULL`を返します

## Example

```sql
select log2(1);
```
```text
+-------------------------+
| log2(cast(1 as DOUBLE)) |
+-------------------------+
|                     0.0 |
+-------------------------+
```
```sql
select log2(2);
```
```text
+-------------------------+
| log2(cast(2 as DOUBLE)) |
+-------------------------+
|                     1.0 |
+-------------------------+
```
```sql
select log2(10);
```
```text
+--------------------------+
| log2(cast(10 as DOUBLE)) |
+--------------------------+
|       3.3219280948873626 |
+--------------------------+
```
