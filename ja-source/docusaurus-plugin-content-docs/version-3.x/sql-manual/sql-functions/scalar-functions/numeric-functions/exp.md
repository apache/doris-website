## 説明

`x`を底`e`で累乗した値を返します。

## 別名

- DEXP

## 構文

```sql
EXP(<x>)
```
## パラメータ

| Parameter | デスクリプション |
| -- | -- |
| `<x>` | 独立変数 |

## Return Value

double型の値を返す
- パラメータがnull値の場合、NULLを返す

## Example

```sql
select exp(2);
```
```text
+------------------+
| exp(2.0)         |
+------------------+
| 7.38905609893065 |
+------------------+
```
```
select exp(3.4);
```
```text
+--------------------+
| exp(3.4)           |
+--------------------+
| 29.964100047397011 |
+--------------------+
```
