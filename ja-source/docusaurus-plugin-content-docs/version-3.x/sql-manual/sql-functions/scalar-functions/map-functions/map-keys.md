## 説明

指定された`map`のキーを対応する型の`ARRAY`に抽出します

## 構文

```sql
MAP_KEYS(<map>)
```
## パラメータ

| Parameter | デスクリプション |
| -- | -- |
| `<map>` | 入力するマップの内容 |

## Return Value

指定された`map`のキーを抽出し、対応する型の`ARRAY`として返す

## Example

```sql
select map_keys(map()),map_keys(map(1, "100", 0.1, 2));
```
```text
+-----------------+-------------------------------------------------------------------------------------------------+
| map_keys(map()) | map_keys(map(cast(1 as DECIMALV3(2, 1)), '100', cast(0.1 as DECIMALV3(2, 1)), cast(2 as TEXT))) |
+-----------------+-------------------------------------------------------------------------------------------------+
| []              | [1.0, 0.1]                                                                                      |
+-----------------+-------------------------------------------------------------------------------------------------+
```
