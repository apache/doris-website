## 説明

LAG()は、自己結合を実行せずに前の行からデータにアクセスするウィンドウ関数です。パーティション内で現在の行からN行前の行から値を取得します。

## 構文

```sql
LAG ( <expr>, <offset>, <default> )
```
## パラメータ
| Parameter           | デスクリプション                                                                             |
| ------------------- |-----------------------------------------------------------------------------------------|
| expr                | 値を取得する式                                           |
| offset              | 先読みする行数|
| default             | offsetがウィンドウの境界を超えた場合に返される値    |

## Return Value

入力式と同じデータ型を返します。

## Examples

各営業担当者の現在の売上金額と前日の売上金額の差を計算する：

```sql
select stock_symbol, closing_date, closing_price,    
lag(closing_price,1, 0) over (partition by stock_symbol order by closing_date) as "yesterday closing"   
from stock_ticker   
order by closing_date;
```
