DMLプランチューニングでは、まずパフォーマンスボトルネックがインポートプロセスによるものか、クエリセクションによるものかを特定する必要があります。クエリセクションにおけるパフォーマンスボトルネックのトラブルシューティングとチューニングについては、[Plan Tuning](optimizing-table-schema.md)の他のサブセクションを詳細について参照してください。

Dorisは複数のデータソースからのデータインポートをサポートしています。Dorisが提供する様々なインポート機能を柔軟に活用することで、様々なソースからのデータを効率的にDorisにインポートして分析することができます。ベストプラクティスの詳細については、[Import 概要](../../../data-operate/import/load-manual.md)を参照してください。
