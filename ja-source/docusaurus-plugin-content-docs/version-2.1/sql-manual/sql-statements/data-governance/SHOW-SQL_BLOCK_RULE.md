## 説明
設定されたSQL阻止ルールを表示します。ルール名が指定されていない場合は、すべてのルールが表示されます。

## 構文

```sql
SHOW SQL_BLOCK_RULE [FOR <rule_name>];
```
## オプションパラメータ

`<rule_name>`

表示するSQLブロッキングルールの名前。省略した場合、すべてのルールが表示されます。|

## アクセス制御要件

このコマンドを実行するユーザーは以下の権限を持っている必要があります：

| 権限 | オブジェクト | 備考 |
|-----------|--------------|--------|
| `ADMIN`   | User または Role | この操作を実行するために必要。|

## 例

1. すべてのSQLブロッキングルールを表示する

```sql
SHOW SQL_BLOCK_RULE;
```
```text
+------------+----------------------------+---------+-------------+------------+-------------+--------+--------+
| Name       | Sql                        | SqlHash | PartitionNum | TabletNum  | Cardinality | Global | Enable |
+------------+----------------------------+---------+-------------+------------+-------------+--------+--------+
| test_rule  | select * from order_analysis | NULL    | 0           | 0          | 0           | true   | true   |
| test_rule2 | NULL                        | NULL    | 30          | 0          | 10000000000 | false  | true   |
+------------+----------------------------+---------+-------------+------------+-------------+--------+--------+
```
2. 特定のSQLブロックルールを表示する

```sql
SHOW SQL_BLOCK_RULE FOR test_rule2;
```
```text
+------------+------+---------+-------------+------------+-------------+--------+--------+
| Name       | Sql  | SqlHash | PartitionNum | TabletNum  | Cardinality | Global | Enable |
+------------+------+---------+-------------+------------+-------------+--------+--------+
| test_rule2 | NULL | NULL    | 30          | 0          | 10000000000 | false  | true   |
+------------+------+---------+-------------+------------+-------------+--------+--------+
```
