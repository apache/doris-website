## 説明

`SHOW DATA SKEW`文は、テーブルまたはパーティションのデータスキューを表示するために使用されます。この文には以下の機能があります：

- テーブル全体のデータ分散を表示できます。
- 指定されたパーティションのデータ分散を表示できます。
- 各バケットの行数、データ量、および割合を表示します。
- パーティション化されたテーブルとパーティション化されていないテーブルの両方をサポートします。

## 構文

```sql
SHOW DATA SKEW FROM [<db_name>.]<table_name> [ PARTITION (<partition_name> [, ...]) ];
```
## 必須パラメータ

**1. `FROM [<db_name>.]<table_name>`**

> 表示するテーブルの名前を指定します。データベース名を含めることができます。
>
> テーブル名は、そのデータベース内で一意である必要があります。

## オプションパラメータ

**1. `PARTITION (<partition_name> [, ...])`**

> 表示するパーティション名のリストを指定します。
>
> このパラメータが指定されていない場合、テーブル内のすべてのパーティションのデータ分散が表示されます。
>
> 非パーティションテーブルの場合、パーティション名はテーブル名と同じです。

## 戻り値

| カラム名         | 説明                                |
|------------------|-------------------------------------|
| PartitionName    | パーティション名                    |
| BucketIdx        | バケットインデックス番号            |
| AvgRowCount      | 平均行数                            |
| AvgDataSize      | 平均データサイズ（バイト単位）      |
| Graph            | データ分散の可視化チャート          |
| Percent          | 総データ量に対するこのバケットのデータ量の割合 |

## アクセス制御要件

このSQLコマンドを実行するユーザーは、少なくとも以下の権限を持つ必要があります：

| 権限            | オブジェクト | 備考                                         |
| :-------------- | :---------- | :-------------------------------------------- |
| SELECT          | Table       | テーブルを表示するにはSELECT権限が必要です。 |

## 使用上の注意

- データ分散は、パーティションとバケットの2つの次元で表示されます。
- Graphカラムは文字`>`を使用してデータ分散比を視覚的に表現します。
- パーセンテージは小数点以下2桁まで正確です。
- 非パーティションテーブルの場合、クエリ結果のパーティション名はテーブル名と同じです。

## 例

- パーティションテーブルを作成し、そのデータ分散を表示します：

    ```sql
    CREATE TABLE test_show_data_skew
    (
        id int,
        name string,
        pdate date
    )
    PARTITION BY RANGE(pdate)
    (
        FROM ("2023-04-16") TO ("2023-04-20") INTERVAL 1 DAY
    )
    DISTRIBUTED BY HASH(id) BUCKETS 5
    PROPERTIES (
        "replication_num" = "1"
    );
    ```
