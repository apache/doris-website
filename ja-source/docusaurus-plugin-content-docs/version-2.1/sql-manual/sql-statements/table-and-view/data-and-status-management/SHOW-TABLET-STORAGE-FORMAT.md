## 説明

このステートメントは、Backend上のストレージフォーマット情報を表示するために使用されます。

## 構文

```sql
SHOW TABLET STORAGE FORMAT [VERBOSE]
```
## オプションパラメータ

** 1. `VERBOSE` **

  より詳細な情報を表示します。

## 戻り値

| Column        | DataType | Note                                                                 |
|---------------|----------|----------------------------------------------------------------------|
| BackendId     | Int      | タブレットレプリカが配置されているBE（Backend）ノードのIDです。 |
| V1Count       | Int      | V1バージョンのタブレット数です。                                        |
| V2Count       | Int      | V2バージョンのタブレット数です。                                        |
| TabletId      | Int      | タブレットの一意識別子です。                                 |
| StorageFormat | String   | タブレットのバージョンで、V1またはV2です。                          |

## アクセス制御要件

このSQLコマンドを実行するユーザーは、少なくとも以下の権限を持つ必要があります：

| Privilege  | Object   | Notes                                                                                                                            |
|:-----------|:---------|:---------------------------------------------------------------------------------------------------------------------------------|
| Admin_priv | Database | テーブル、パーティション、システムレベルのコマンドの管理を含む、データベースの管理操作を実行するために必要です。 |

## 例

- verboseパラメータなしでステートメントを実行する

  ```sql
  show tablet storage format;
  ```
  ```text
  +-----------+---------+---------+
  | BackendId | V1Count | V2Count |
  +-----------+---------+---------+
  | 10002     | 0       | 2867    |
  +-----------+---------+---------+
  ```
- verbose パラメータを使用してステートメントを実行する

  ```sql
  show tablet storage format verbose;
  ```
  ```text
  +-----------+----------+---------------+
  | BackendId | TabletId | StorageFormat |
  +-----------+----------+---------------+
  | 10002     | 39227    | V2            |
  | 10002     | 39221    | V2            |
  | 10002     | 39215    | V2            |
  | 10002     | 39199    | V2            |
  +-----------+----------+---------------+
  ```
