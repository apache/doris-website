## 説明

SHOW PARTITIONは、指定されたパーティションに関する詳細情報を表示するために使用されます。これには、関連付けられたデータベースの名前とID、関連付けられたテーブルの名前とID、およびパーティション名が含まれます。

## 構文

```sql
SHOW PARTITION <partition_id>
```
## 必須パラメータ

**<partition_id>**

> パーティションのID。パーティションIDはSHOW PARTITIONSなどの方法で取得できます。詳細については、「SHOW PARTITIONS」セクションを参照してください。

## アクセス制御要件

このSQLコマンドを実行するユーザーは、少なくとも`ADMIN_PRIV`権限を持っている必要があります。

## 例

パーティションID 13004のパーティション情報を照会：

```sql
SHOW PARTITION 13004;
```
結果:

```sql
+--------+-----------+---------------+-------+---------+
| DbName | TableName | PartitionName | DbId  | TableId |
+--------+-----------+---------------+-------+---------+
| ods    | sales     | sales         | 13003 | 13005   |
+--------+-----------+---------------+-------+---------+
```
