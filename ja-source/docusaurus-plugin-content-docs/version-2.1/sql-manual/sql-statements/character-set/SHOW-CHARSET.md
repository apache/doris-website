## 説明

「SHOW CHARSET」コマンドは、現在のデータベース管理システムで利用可能な文字セットと、各文字セットに関連するいくつかのプロパティを表示するために使用されます。

これらのプロパティには、文字セットの名前、デフォルト照合順序、最大バイト長などが含まれる場合があります。システムでサポートされている文字セットのリストとその詳細は、「SHOW CHARSET」コマンドを実行することで確認できます。

## 構文

```sql
SHOW CHARSET
```
## 戻り値
| column name | description |
| -- |-------------|
| Charset | 文字セット         |
| Description | 説明          |
| Default Collation | デフォルトの照合順序名      |
| Maxlen | 最大バイト長      |

## 例

```sql
SHOW CHARSET;
```
```text
+---------+---------------+-------------------+--------+
| Charset | Description   | Default collation | Maxlen |
+---------+---------------+-------------------+--------+
| utf8mb4 | UTF-8 Unicode | utf8mb4_0900_bin  | 4      |
+---------+---------------+-------------------+--------+
```
