## 説明

このステートメントは、実行中のRESTOREタスクをキャンセルするために使用されます。

## 構文

```sql
CANCEL RESTORE FROM <db_name>;
```
## パラメータ

**1.`<db_name>`**

復旧タスクが属するデータベースの名前。

## 使用上の注意

- キャンセルが復旧のCOMMITまたはそれ以降の段階で行われた場合、復旧中のテーブルにアクセスできなくなる可能性があります。この場合、復旧ジョブを再実行することによってのみデータ復旧を実行できます。

## 例

1. example_db配下のRESTOREタスクをキャンセルする。

```sql
CANCEL RESTORE FROM example_db;
```
