## 説明

2つの数値変数間の標本共分散を計算します。

## 構文

```sql
COVAR_SAMP(<expr1>, <expr2>)
```
## パラメータ

| パラメータ | 説明 |
| -- | -- |
| `<expr1>` | 数値式または列 |
| `<expr2>` | 数値式または列 |

## 戻り値

expr1とexpr2の標本共分散を返します。特殊なケース：

- expr1またはexpr2の列がNULLの場合、その行のデータは最終結果にカウントされません。

## 例

```
select covar_samp(x,y) from baseall;
```
```text
+---------------------+
| covar_samp(x, y)    |
+---------------------+
| 0.89442719099991586 |
+---------------------+
```
