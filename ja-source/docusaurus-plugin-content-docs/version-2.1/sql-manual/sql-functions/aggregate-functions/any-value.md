## 説明

グループ内の式または列から任意の値を返します。NULL以外の値がある場合は、任意のNULL以外の値を返し、そうでなければNULLを返します。

## エイリアス

- ANY

## 構文

```sql
ANY_VALUE(<expr>)
```
## パラメータ

| パラメータ | 説明 |
| -- | -- |
| `<expr>` | 集計対象の列または式。 |

## 戻り値

NULL以外の値が存在する場合はいずれかのNULL以外の値を返し、そうでなければNULLを返します。

## 例

```sql
select id, any_value(name) from cost2 group by id;
```
```text
+------+-------------------+
| id   | any_value(`name`) |
+------+-------------------+
|    3 | jack              |
|    2 | jack              |
+------+-------------------+
```
