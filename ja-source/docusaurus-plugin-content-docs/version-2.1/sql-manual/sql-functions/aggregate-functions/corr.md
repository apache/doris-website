## 説明

2つの確率変数のPearson係数を計算します。

## 構文

```sql
CORR(<expr1>, <expr2>)
```
## パラメータ

| パラメータ | 説明 |
| -- | -- |
| `<expr1>` | 数値式（列） |
| `<expr2>` | 数値式（列） |

## 戻り値

戻り値はDOUBLE型で、expr1とexpr2の標準偏差の積を除いたexpr1とexpr2の共分散です。特別なケース：

- expr1またはexpr2の標準偏差が0の場合、0が返されます。
- expr1またはexpr2の列がNULLの場合、その行のデータは最終結果にカウントされません。

## 例

```sql
select * from test_corr;
```
```text
+------+------+------+
| id   | k1   | k2   |
+------+------+------+
|    1 |   20 |   22 |
|    1 |   10 |   20 |
|    2 |   36 |   21 |
|    2 |   30 |   22 |
|    2 |   25 |   20 |
|    3 |   25 | NULL |
|    4 |   25 |   21 |
|    4 |   25 |   22 |
|    4 |   25 |   20 |
+------+------+------+
```
```sql
select id,corr(k1,k2) from test_corr group by id;
```
```text
+------+--------------------+
| id   | corr(k1, k2)       |
+------+--------------------+
|    4 |                  0 |
|    1 |                  1 |
|    3 |               NULL |
|    2 | 0.4539206495016019 |
+------+--------------------+
```
