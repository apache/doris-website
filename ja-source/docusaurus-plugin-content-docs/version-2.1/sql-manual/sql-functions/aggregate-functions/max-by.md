## 説明

MAX_BY関数は、指定された列の最大値に基づいて、対応する関連値を返すために使用されます。

## 構文

```sql
MAX_BY(<expr1>, <expr2>)
```
## パラメータ

| パラメータ | 説明 |
| -- | -- |
| `<expr1>` | 対応する関連付けを指定するために使用される式。 |
| `<expr2>` | 統計の最大値を指定するために使用される式。 |

## 戻り値

入力式 <expr1> と同じデータ型を返します。

## 例

```sql
select * from tbl;
```
```text
+------+------+------+------+
| k1   | k2   | k3   | k4   |
+------+------+------+------+
|    0 | 3    | 2    |  100 |
|    1 | 2    | 3    |    4 |
|    4 | 3    | 2    |    1 |
|    3 | 4    | 2    |    1 |
+------+------+------+------+
```
```sql
select max_by(k1, k4) from tbl;
```
```text
+--------------------+
| max_by(`k1`, `k4`) |
+--------------------+
|                  0 |
+--------------------+ 
```
