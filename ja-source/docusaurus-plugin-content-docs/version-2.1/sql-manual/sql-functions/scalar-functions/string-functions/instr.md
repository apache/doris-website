## 説明

str内でsubstrが最初に出現する位置を返します（カウントは1から始まります）。特殊なケース：

- substrがstr内に出現しない場合、0を返します。

## 構文

```sql
INSTR ( <str> , <substr> )
```
## パラメータ

| パラメータ | 説明 |
|--------|-----------|
| `<str>` | 検索対象の文字列 |
| `<substr>` | 検索する文字列 |

## 戻り値

パラメータ `<str>`内での`<substr>`の最初の出現位置（カウントは1から開始）。特殊なケース：

- `<substr>`が`<str>`内に存在しない場合、0を返します。

## 例

```sql
SELECT INSTR("abc", "b"),INSTR("abc", "d")
```
```text
+-------------------+-------------------+
| instr('abc', 'b') | instr('abc', 'd') |
+-------------------+-------------------+
|                 2 |                 0 |
+-------------------+-------------------+
```
