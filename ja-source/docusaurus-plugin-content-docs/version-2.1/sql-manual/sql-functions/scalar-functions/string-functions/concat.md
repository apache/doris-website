## 説明

複数の文字列を連結します。特殊なケース：

- パラメータ値のいずれかがNULLの場合、返される結果はNULLです

## 構文

```sql
CONCAT ( <expr> [ , <expr> ... ] )
```
## パラメータ

| パラメータ | 説明 |
|-----------|--------------|
| `<expr>`  | 連結される文字列 |

## 戻り値

パラメータリスト `<expr>` 連結される文字列。特殊なケース:

- パラメータ値のいずれかがNULLの場合、返される結果はNULLになります

## 例

```sql
SELECT  CONCAT("a", "b"),CONCAT("a", "b", "c"),CONCAT("a", null, "c")
```
```text
+------------------+-----------------------+------------------------+
| concat('a', 'b') | concat('a', 'b', 'c') | concat('a', NULL, 'c') |
+------------------+-----------------------+------------------------+
| ab               | abc                   | NULL                   |
+------------------+-----------------------+------------------------+
```
