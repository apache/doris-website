## 説明

入力文字列をBase64デコードした結果を返します。特殊なケース：

- 入力文字列が不正な場合（Base64エンコード後では不可能な文字列が現れる場合）、NULLが返されます

## 構文

```sql
FROM_BASE64 ( <str> )
```
## パラメータ

| パラメータ | 説明 |
|------------|-----------------|
| `<str>`    | Base64デコードする文字列 |

## 戻り値

パラメータ <str> Base64デコードの結果。特殊なケース：

- 入力文字列が正しくない場合（Base64エンコード後にあり得ない文字列が現れる場合）、NULLが返されます。

## 例

```sql
SELECT FROM_BASE64('MQ=='),FROM_BASE64('MjM0'),FROM_BASE64(NULL)
```
```text
+---------------------+---------------------+-------------------+
| from_base64('MQ==') | from_base64('MjM0') | from_base64(NULL) |
+---------------------+---------------------+-------------------+
| 1                   | 234                 | NULL              |
+---------------------+---------------------+-------------------+
```
