## 説明

ランダムに生成されたUUID文字列を返すために使用されます。

## 構文

```sql
UUID()
```
## 戻り値
ランダムに生成されたUUID文字列を返します。

## 例

```sql
select UUID();
```
```sql
+--------------------------------------+
| UUID()                               |
+--------------------------------------+
| 29077778-fc5e-4603-8368-6b5f8fd55c24 |
+--------------------------------------+

```
