## 説明

`digital_masking`関数は数値をマスクするために使用されます。指定されたマスクルールに基づいて、数値内の特定の文字が*に置き換えられます。この関数は元の関数`concat(left(id, 3), '****', right(id, 4))`のエイリアスです。

## 構文

```sql
DIGITAL_MASKING( <digital_number> )
```
## パラメータ

| パラメータ | 説明 |
| -- | -- |
| `<digital_number>` | マスクが必要なデジタル文字列 |

## 戻り値

マスクされたデジタル文字列を返します。

## 例

```sql
select digital_masking(13812345678);
```
```
+------------------------------+
| digital_masking(13812345678) |
+------------------------------+
| 138****5678                  |
+------------------------------+
```
