## 説明

入力パラメータが数値の場合、16進数値の文字列表現を返します。

入力パラメータが文字列の場合、各文字を2つの16進文字に変換し、変換されたすべての文字を文字列に連結して出力します。

## 構文

```sql
HEX ( <str> )
```
## パラメータ

| パラメータ | 説明 |
|-------|--------------|
| `<str>` | 入力パラメータは数値または文字列 |

## 戻り値

パラメータ `<str>` の16進数の結果。

## 例

入力パラメータは数値

```sql
SELECT HEX(12),HEX(-1)
```
```text
+---------+------------------+
| hex(12) | hex(-1)          |
+---------+------------------+
| C       | FFFFFFFFFFFFFFFF |
+---------+------------------+
```
入力パラメータは文字列です

```sql
SELECT HEX('1'),HEX('@'),HEX('12')
```
```text
+----------+----------+-----------+
| hex('1') | hex('@') | hex('12') |
+----------+----------+-----------+
| 31       | 40       | 3132      |
+----------+----------+-----------+
```
