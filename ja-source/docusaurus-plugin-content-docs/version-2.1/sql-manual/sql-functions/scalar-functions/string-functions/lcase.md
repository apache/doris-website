## 説明

パラメータ内のすべての文字列を小文字に変換します。

## エイリアス

- LOWER

## 構文

```sql
LCASE ( <str> )
LOWER ( <str> )
```
## パラメータ

| パラメータ | 説明 |
|-----------|--------------|
| `<str>`   | 小文字に変換される文字列 |

## 戻り値

パラメータ `<str>` 小文字に変換された文字列

## 例

```sql
SELECT LCASE("AbC123"),LOWER("AbC123")
```
```text
+-----------------+-----------------+
| lower('AbC123') | lower('AbC123') |
+-----------------+-----------------+
| abc123          | abc123          |
+-----------------+-----------------+
```
```sql
SELECT LOWER("ҚAZAҚ123"),LCASE("ҒАРЫШ");
```
```text
+---------------------+---------------------+
| LOWER("ҚAZAҚ123")   | LCASE("ҒАРЫШ")      |
+---------------------+---------------------+
| қazaқ123            | ғарыш               |
+---------------------+---------------------+
```
