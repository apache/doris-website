## 説明

文字列の長さを計算します。マルチバイト文字の場合、文字数を返します。

現在は`utf8`エンコーディングのみをサポートしています

## エイリアス

- CHARACTER_LENGTH

## 構文

```sql
CHAR_LENGTH ( <str> )
```
## パラメータ

| パラメータ | 説明 |
|-----------|------------|
| `<str>`   | 長さを計算する文字列 |

## 戻り値

文字列 <str> の長さ。

## 例

```sql
select CHAR_LENGTH("abc"),CHAR_LENGTH("中国")
```
```text
+-------------------------+----------------------------+
| character_length('abc') | character_length('中国')   |
+-------------------------+----------------------------+
|                       3 |                          2 |
+-------------------------+----------------------------+
```
