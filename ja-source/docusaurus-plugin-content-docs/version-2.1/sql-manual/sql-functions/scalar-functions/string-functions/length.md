## 説明

文字列のバイト数を返します。

## 構文

```sql
LENGTH ( <str> )
```
## パラメータ

| パラメータ | 説明 |
|-----------| --------------- |
| `<str>`   | バイト数を計算する必要がある文字列 |

## 戻り値

文字列 `<str>` のバイト数。

## 例

```sql
SELECT LENGTH("abc"),length("中国")
```
```text
+---------------+------------------+
| length('abc') | length('中国')   |
+---------------+------------------+
|             3 |                6 |
+---------------+------------------+
```
