## 説明

TO_BASE64関数は、入力文字列をBase64エンコード形式に変換するために使用されます。Base64エンコーディングは、任意のバイナリデータを64文字で構成される文字列に変換することができます。

## 構文

```sql
TO_BASE64(<str>)
```
## パラメータ
| パラメータ | 説明                                    |
| --------- | ---------------------------------------------- |
| `<str>` | Base64エンコードする文字列。型: VARCHAR |

## 戻り値

VARCHAR型を返し、Base64エンコードされた文字列を表します。

特殊なケース:
- 入力がNULLの場合、NULLを返す
- 入力が空文字列の場合、空文字列を返す

## 例

1. 単一文字のエンコード

```sql
SELECT to_base64('1');
```
```text
+----------------+
| to_base64('1') |
+----------------+
| MQ==           |
+----------------+
```
2. 複数文字エンコーディング

```sql
SELECT to_base64('234');
```
```text
+------------------+
| to_base64('234') |
+------------------+
| MjM0             |
+------------------+
```
