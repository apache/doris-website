## 説明
この関数は、指定された日付の四半期（1から4）を返します。各四半期は3か月を含みます：
- Q1: 1月から3月
- Q2: 4月から6月
- Q3: 7月から9月
- Q4: 10月から12月

## 構文

```sql
QUARTER(<datetime>)
```
## パラメータ

| Parameter    | Description                                              |
|--------------|----------------------------------------------------------|
| `<datetime>` | 四半期を決定するための有効なDATEまたはDATETIME値。 |

## 戻り値
- 入力日付の四半期を表す1から4の範囲の整数を返します。
- 入力がNULLの場合、この関数はNULLを返します。
- 入力が無効な日付（例：0000-00-00）の場合、この関数はNULLを返します。

## 例

```sql
SELECT QUARTER('2025-01-16'),QUARTER('2025-01-16 01:11:10');
```
```text
+-----------------------------------------+--------------------------------------------------+
| quarter(cast('2025-01-16' as DATETIME)) | quarter(cast('2025-01-16 01:11:10' as DATETIME)) |
+-----------------------------------------+--------------------------------------------------+
|                                       1 |                                                1 |
+-----------------------------------------+--------------------------------------------------+
```
