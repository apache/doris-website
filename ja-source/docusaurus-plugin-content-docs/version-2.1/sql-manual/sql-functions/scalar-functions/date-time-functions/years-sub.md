## 説明

入力されたdatetimeから指定された年数を減算した結果である新しいdatetime値を返します。

## 構文

```sql
YEARS_SUB(<date>, <years>)
```
## パラメータ

| Parameter | Description                                      |
|-----------|--------------------------------------------------|
| `<date>`      | 入力される日時値。DATETIME型またはDATE型を指定可能 |
| `<years>`     | 減算する年数。INT型 |

## 戻り値

入力された`<date>`と同じ型（DATETIME型またはDATE型）の値を返します。入力された日時から指定された年数を減算した時間値を表します。

## 例

```sql
SELECT YEARS_SUB('2020-02-02 02:02:02', 1);
```
```text
+-------------------------------------+
| years_sub('2020-02-02 02:02:02', 1) |
+-------------------------------------+
| 2019-02-02 02:02:02                 |
+-------------------------------------+
```
