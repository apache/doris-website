## 説明

入力されたdatetimeに指定された年数を加算した結果である、新しいdatetime値を返します。

## 構文

```sql
YEARS_ADD(<date>, <years>)
```
## パラメータ

| パラメータ | 説明                                      |
|-----------|--------------------------------------------------|
| `<date>`      | 入力となる日時の値で、DATETIME型またはDATE型を指定可能 |
| `<years>`     | 追加する年数、INT型         |

## 戻り値

入力の`<date>`と同じ型（DATETIMEまたはDATE）の値を返し、入力された日時に指定された年数を加算した後の時間値を表します。

## 例

```sql
SELECT YEARS_ADD('2020-01-31 02:02:02', 1);
```
```text
+-------------------------------------+
| years_add('2020-01-31 02:02:02', 1) |
+-------------------------------------+
| 2021-01-31 02:02:02                 |
+-------------------------------------+
```
