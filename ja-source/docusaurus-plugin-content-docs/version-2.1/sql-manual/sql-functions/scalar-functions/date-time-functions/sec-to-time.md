## 説明
`SEC_TO_TIME`関数は秒単位の値を`TIME`型に変換し、`HH:MM:SS`形式で結果を返します。  
入力される秒数は、1日の開始時刻（`00:00:00`）からの経過時間を表します。


## 構文

```sql
SEC_TO_TIME(<seconds>)
```
## パラメータ

| パラメータ | 説明 |
|---------------|-------------------------------------------------------------------------------------------------------------------------------------------------------|
| `<seconds>` | 必須。入力する秒数。一日の開始時刻（00:00:00）からの経過時間を表します。正の整数または負の整数をサポートします。 |

## 戻り値
- 一日の開始時刻（00:00:00）から計算された時刻を表す、`HH:MM:SS`形式のTIME値を返します。
- `<seconds>`がNULLの場合、この関数はNULLを返します。

## 例

```sql
SELECT SEC_TO_TIME(59738);
```
```text
+--------------------+
| sec_to_time(59738) |
+--------------------+
| 16:35:38           |
+--------------------+
```
