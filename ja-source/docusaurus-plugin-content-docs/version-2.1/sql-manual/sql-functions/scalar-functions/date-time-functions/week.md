## week
### 説明
#### 構文

`INT WEEK(DATE date[, INT mode])`

日付の週番号を返します。mode引数の値はデフォルトで0です。
以下の表はmode引数がどのように動作するかを説明します。

|Mode |週の最初の日 |範囲  |週1は最初の週…    |
|:----|:-----------------|:------|:-----------------------------|
|0    |Sunday            |0-53   |この年のSundayがある週    |
|1    |Monday            |0-53   |この年の4日以上がある週 |
|2    |Sunday            |1-53   |この年のSundayがある週    |
|3    |Monday            |1-53   |この年の4日以上がある週 |
|4    |Sunday            |0-53   |この年の4日以上がある週 |
|5    |Monday            |0-53   |この年のMondayがある週    |
|6    |Sunday            |1-53   |この年の4日以上がある週 |
|7    |Monday            |1-53   |この年のMondayがある週    |

パラメータはDateまたはDatetime型です

### example

```
mysql> select week('2020-1-1');
+------------------+
| week('2020-1-1') |
+------------------+
|                0 |
+------------------+
```
```
mysql> select week('2020-7-1',1);
+---------------------+
| week('2020-7-1', 1) |
+---------------------+
|                  27 |
+---------------------+
```
