## 説明

datetime値に指定した分数を加算し、新しいdatetime値を返します。

## 構文

```sql
MINUTES_ADD(<datetime>, <minutes>)
```
## パラメータ

| Parameter | Description                                      |
|-----------|--------------------------------------------------|
| `<datetime>`  | 入力される日時値。DATE、DATETIME、またはDATETIMEV2型を使用可能 |
| `<minutes>`   | 加算する分数。INT型で正の値または負の値を指定可能 |

## 戻り値

指定された分数を加算した後の日時値を表すDATETIME型の値を返します。

## 例

```sql
SELECT MINUTES_ADD("2020-02-02", 1);
```
```text
+-----------------------------------------------------+
| minutes_add(cast('2020-02-02' as DATETIMEV2(0)), 1) |
+-----------------------------------------------------+
| 2020-02-02 00:01:00                                 |
+-----------------------------------------------------+
```
**注意:**
- 追加する分数が負の値の場合、対応する分数が効果的に減算されます。
- この関数は時間や日をまたぐケースを自動的に処理します。
- 入力パラメータがNULLの場合、関数はNULLを返します。
