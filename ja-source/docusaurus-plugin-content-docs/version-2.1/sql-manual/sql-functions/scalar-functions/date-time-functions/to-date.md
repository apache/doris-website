## 説明
日付変換関数。日時（DATETIME）を日付型（DATE）に変換するために使用されます。つまり、時刻部分を削除し、日付（YYYY-MM-DD）のみを保持します。

## 構文

```sql
TO_DATE(<datetime_value>)
```
## 必須パラメータ
| Parameter        | Description               |
|-----------------|--------------------------|
| `datetime_value` | DATETIME型の日時 |


## 例

`2020-02-02 00:00:00`を`2020-02-02`に変換

```sql
select to_date("2020-02-02 00:00:00");
```
```text
+--------------------------------+
| to_date('2020-02-02 00:00:00') |
+--------------------------------+
| 2020-02-02                     |
+--------------------------------+
```
