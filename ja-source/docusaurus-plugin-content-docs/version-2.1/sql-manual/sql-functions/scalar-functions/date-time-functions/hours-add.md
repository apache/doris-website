## 説明

入力されたdatetimeに指定された時間数を加算した結果となる新しいdatetime値を返します。

## 構文

```sql
HOURS_ADD(<date>, <hours>)
```
## パラメータ

| Parameter | Description                                      |
|-----------|--------------------------------------------------|
| `<date>`      | 入力される日時値。DATETIME型またはDATE型を指定可能 |
| `<hours>`     | 追加する時間数。INT型 |

## 戻り値

入力された日時に指定した時間数を追加した後の時刻値を表すDATETIME型の値を返します。

## 例

```sql
SELECT HOURS_ADD('2020-02-02 02:02:02', 1);
```
```text
+------------------------------------------------------------+
| hours_add(cast('2020-02-02 02:02:02' as DATETIMEV2(0)), 1) |
+------------------------------------------------------------+
| 2020-02-02 03:02:02                                        |
+------------------------------------------------------------+
```
