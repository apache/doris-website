## 説明
この関数は、指定された日時値の秒の部分を返します。秒の範囲は0から59です。

## 構文

```sql
SECOND(<datetime>)
```
## パラメータ

| パラメータ    | 説明                                                        |
|--------------|--------------------------------------------------------------------|
| `<datetime>` | 入力される日付またはdatetime値。DATEまたはDATETIME型をサポートします。 |

## 戻り値
- 入力されたdatetime値の秒部分を表す整数を返します。範囲は0から59です。
- 入力がNULLの場合、関数はNULLを返します。
- 入力が無効な日付の場合（例：0000-00-00 00:00:00）、関数はNULLを返します。

## 例

```sql
select second('2018-12-31 23:59:59');
```
```text
+---------------------------------------------+
| second(cast('2018-12-30' as DATETIMEV2(0))) |
+---------------------------------------------+
|                                           0 |
+---------------------------------------------+
```
