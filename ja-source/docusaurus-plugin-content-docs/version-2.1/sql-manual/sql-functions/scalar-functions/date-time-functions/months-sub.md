## 説明
MONTHS_SUB関数は、指定された日付に指定された月数を加算または減算し、結果の日付を返すために使用されます。

## 構文

```sql
MONTHS_SUB(<datetime/date>,  <nums>)
```
## パラメータ

| パラメータ        | 説明                                                |
|-------------------|------------------------------------------------------------|
| `<datetime/date>` | 月を加算または減算する日付値 |
| `<nums>`          | 加算または減算する月数                    |

## 戻り値
戻り値は入力の<datetime/date>と同じ型です。
特殊なケース:
- <datetime/date>入力が0000-00-00または0000-00-00 00:00:00の場合、関数はNULLを返します。
- <datetime/date>入力がNULLの場合、関数はNULLを返します。
- 入力がMONTHS_SUB("9999-12-31", -1)の場合、関数はNULLを返します。

## 例

``` sql
select months_sub("2020-01-31 02:02:02", 1),months_sub("2020-01-31", 1),months_sub("2020-01-31", -1);
```
```text
+-------------------------------------------------------------+---------------------------------------------+----------------------------------------------+
| months_sub(cast('2020-01-31 02:02:02' as DATETIMEV2(0)), 1) | months_sub(cast('2020-01-31' as DATEV2), 1) | months_sub(cast('2020-01-31' as DATEV2), -1) |
+-------------------------------------------------------------+---------------------------------------------+----------------------------------------------+
| 2019-12-31 02:02:02                                         | 2019-12-31                                  | 2020-02-29                                   |
+-------------------------------------------------------------+---------------------------------------------+----------------------------------------------+
```
