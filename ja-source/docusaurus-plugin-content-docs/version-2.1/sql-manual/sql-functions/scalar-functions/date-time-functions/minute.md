## 説明

datetime値から分の部分を抽出します。返される値の範囲は0から59です。

## 構文

```sql
MINUTE(<datetime>)
```
## パラメータ

| パラメータ | 説明                                      |
|-----------|--------------------------------------------------|
| `<datetime>`  | 入力される日時値。DATE、DATETIME、DATETIMEV2、またはTIME型を指定可能 |

## 戻り値

0-59の範囲で分の値を表すINT型を返します。

## 例

```sql
SELECT MINUTE('2018-12-31 23:59:59');
```
```text
+------------------------------------------------------+
| minute(cast('2018-12-31 23:59:59' as DATETIMEV2(0))) |
+------------------------------------------------------+
|                                                   59 |
+------------------------------------------------------+
```
**注意:**
- 入力パラメータは様々な時間関連の型を使用できます。
- 戻り値は常に0から59の間の整数です。
- 入力パラメータがNULLの場合、関数はNULLを返します。
