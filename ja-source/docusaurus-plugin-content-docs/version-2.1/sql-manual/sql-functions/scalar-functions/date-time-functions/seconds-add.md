## 説明

この関数は、指定されたdatetime値に対して指定された秒数を加算または減算し、結果のdatetimeを返します。

## 構文

```sql
SECONDS_ADD(<datetime>, <seconds>)
```
## パラメータ

| パラメータ    | 説明                                                                                                                                         |
|--------------|-----------------------------------------------------------------------------------------------------------------------------------------------------|
| `<datetime>` | 必須。入力する日時値。DATETIME型とDATE型をサポートします。                                                                            |
| `<seconds>`  | 必須。加算または減算する秒数。整数（INT）をサポートします。正の数は秒を加算し、負の数は秒を減算します。 |

## 戻り値
- 入力した<datetime>と同じ型の日時値を返します。
- `<datetime>`がNULLの場合、関数はNULLを返します。
- `<datetime>`が無効な日付（例：0000-00-00T00:00:00）の場合、関数はNULLを返します。

## 例

```
SELECT SECONDS_ADD('2025-01-23 12:34:56', 30),SECONDS_ADD('2025-01-23 12:34:56', -30);
```
```text
+---------------------------------------------------------------+----------------------------------------------------------------+
| seconds_add(cast('2025-01-23 12:34:56' as DATETIMEV2(0)), 30) | seconds_add(cast('2025-01-23 12:34:56' as DATETIMEV2(0)), -30) |
+---------------------------------------------------------------+----------------------------------------------------------------+
| 2025-01-23 12:35:26                                           | 2025-01-23 12:34:26                                            |
+---------------------------------------------------------------+----------------------------------------------------------------+
```
