## 説明

この関数は、Unixタイムスタンプ（ミリ秒単位）を`DATETIME`値に変換します。

## 構文

```sql
FROM_MILLISECOND(<millisecond>)
```
## パラメータ

| パラメータ | 説明 |
|-----------------|-------------------------------------------------------------------------------------------------------------|
| `<millisecond>` | 必須。1970-01-01 00:00:00 UTCからの経過ミリ秒数を表すUnixタイムスタンプ。 |

## 戻り値

- 指定されたUnixタイムスタンプに対応する日付と時刻を表すDATETIME値を返します。
- <millisecond>がNULLの場合、関数はNULLを返します。
- <millisecond>が有効な範囲外の場合、関数はエラーを返します。

## 例

```sql
SELECT FROM_MILLISECOND(1700000000000);
```
```text
+---------------------------------+
| from_millisecond(1700000000000) |
+---------------------------------+
| 2023-11-15 06:13:20             |
+---------------------------------+
```
