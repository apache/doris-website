## 説明

指定された日付に対応する年間通算日を取得します。

## 構文

```sql
DAYOFYEAR(<dt>)
```
## パラメータ

| パラメータ | 説明 |
| -- | -- |
| `<dt>` | 計算対象となる日付式 |

## 戻り値

指定された日付に対応する年内の日数を返します。

## 例

```sql
select dayofyear('2007-02-03 00:00:00');
```
```text
+----------------------------------+
| dayofyear('2007-02-03 00:00:00') |
+----------------------------------+
|                               34 |
+----------------------------------+
```
