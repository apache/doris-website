## 説明
この関数は、指定された日付または時刻の値から特定の週数を加算（または減算）するために使用されます。

## 構文

```sql
WEEKS_ADD(<datetime_or_date_value>, <weeks_value>)
```
## 必須パラメータ
| パラメータ                  | 説明                                                                                                                              |
|----------------------------|------------------------------------------------------------------------------------------------------------------------------------------|
| `<datetime_or_date_value>` | `DATETIME`または`DATE`型の日付入力値                                                                                               |
| `<weeks_value>`            | 整数、増減する週数を示す（正の数は増加、負の数は減少を示す） |


## 例


1. 時刻`2020-02-02 02:02:02`に1週間を追加する

    ```sql
    select weeks_add("2020-02-02 02:02:02", 1);
    ```
    ```text
      +-------------------------------------+
      | weeks_add('2020-02-02 02:02:02', 1) |
      +-------------------------------------+
      | 2020-02-09 02:02:02                 |
      +-------------------------------------+
    ```
2. 時刻 `2020-02-02 02:02:02` から1週間を減算する

    ```sql
    select weeks_add("2020-02-02 02:02:02", -1);
    ```
    ```text
    +-------------------------------------------------------------+
    | weeks_add(cast('2020-02-02 02:02:02' as DATETIMEV2(0)), -1) |
    +-------------------------------------------------------------+
    | 2020-01-26 02:02:02                                         |
    +-------------------------------------------------------------+
    ```
3. 日付 `2020-02-02` に1週間を追加する

    ```sql
    select weeks_add("2020-02-02", 1);
    ```
    ```text
    +--------------------------------------------+
    | weeks_add(cast('2020-02-02' as DATEV2), 1) |
    +--------------------------------------------+
    | 2020-02-09                                 |
    +--------------------------------------------+
    ```
