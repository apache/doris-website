## year
### 説明
#### 構文

`INT YEAR(DATETIME date)`

1000から9999の範囲で、date型の年の部分を返します

パラメータはDateまたはDatetime型です

### 例

```
mysql> select year('1987-01-01');
+-----------------------------+
| year('1987-01-01 00:00:00') |
+-----------------------------+
|                        1987 |
+-----------------------------+
```
### キーワード
    YEAR
