# TO_ISO8601

## 説明

datetime値をISO8601形式の文字列に変換します。

## 構文

```sql
TO_ISO8601(<dt>)
```
## パラメータ

| パラメータ | 説明                                      |
|-----------|--------------------------------------------------|
| `<dt>`      | 入力されるdatetime値で、DATETIME型またはDATE型を指定可能 |

## 戻り値

ISO8601形式のdatetimeを表すVARCHAR型の値を返します。

## 例

```sql
SELECT TO_ISO8601('2020-01-01 12:30:45');
```
```text
+-------------------------------------+
| to_iso8601('2020-01-01 12:30:45.0') |
+-------------------------------------+
| 2020-01-01T12:30:45                 |
+-------------------------------------+
```
