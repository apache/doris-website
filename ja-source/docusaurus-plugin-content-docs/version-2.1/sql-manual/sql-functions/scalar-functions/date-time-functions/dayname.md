## 説明

与えられた日付式に対応する曜日名を計算します。

## 構文

```sql
DAYNAME(<dt>)
```
## パラメータ

| パラメータ | 説明 |
| -- | -- |
| `<dt>` | 計算対象となる日付式 |

## 戻り値

指定された日付式に対応する曜日名を返します。

## 例

```sql
select dayname('2007-02-03 00:00:00');
```
```text
+--------------------------------+
| dayname('2007-02-03 00:00:00') |
+--------------------------------+
| Saturday                       |
+--------------------------------+
```
