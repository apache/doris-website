## 説明

パラメータリスト内で左から右へ最初のnull以外の式を返します。すべての引数がNULLの場合、NULLを返します。

## 構文

```sql
COALESCE(<expr> [, ...])
```
## パラメータ

| パラメータ      | 説明                                                                   |
| -------------- | ----------------------------------------------------------------------------- |
| `<expr>` | 評価する式のシーケンス。すべての式は互換性のあるデータ型である必要があります。 |

## 戻り値

パラメータリスト内の最初の非null式。すべての引数がNULLの場合はNULLを返します。

## 例

```sql
SELECT COALESCE(NULL, '1111', '0000');
```
```text
+--------------------------------+
| coalesce(NULL, '1111', '0000') |
+--------------------------------+
| 1111                           |
+--------------------------------+
```
