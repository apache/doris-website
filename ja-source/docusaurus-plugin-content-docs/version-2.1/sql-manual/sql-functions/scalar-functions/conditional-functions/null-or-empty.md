## 説明

`null_or_empty`関数は、指定された値がNULLまたは空文字列かどうかをチェックします。入力値がNULLまたは空文字列の場合はtrueを返し、そうでなければfalseを返します。

## 構文

```sql
NULL_OR_EMPTY (<str>)
```
## パラメータ

| パラメータ | 説明                   |
| ---------- | ---------------------- |
| `<str>`    | チェックする文字列。   |

## 戻り値

文字列が空文字列またはNULLの場合はtrueを返し、そうでなければfalseを返します。

## 例

```sql
select null_or_empty(null);
```
```text
+---------------------+
| null_or_empty(NULL) |
+---------------------+
|                   1 |
+---------------------+
```
```sql
select null_or_empty("");
```
```text
+-------------------+
| null_or_empty('') |
+-------------------+
|                 1 |
+-------------------+
```
```sql
select null_or_empty("a");
```
```text
+--------------------+
| null_or_empty('a') |
+--------------------+
|                  0 |
+--------------------+
```
