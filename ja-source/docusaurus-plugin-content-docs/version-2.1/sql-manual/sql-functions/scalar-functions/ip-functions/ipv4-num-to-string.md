## 説明
Int16、Int32、Int64の数値を受け取ります。それをビッグエンディアンのIPv4アドレスとして解釈します。A.B.C.D形式（10進数をドット区切りした形式）の対応するIPv4アドレスを含む文字列を返します。

## エイリアス
- INET_NTOA

## 構文

```sql
IPV4_NUM_TO_STRING(<ipv4_num>)
```
## パラメータ
| パラメータ | 説明                                      |
|-----------|--------------------------------------------------|
| `<ipv4_num>`      | ipv4から変換されたInt型  |


## 戻り値
A.B.C.D形式（10進数をドット区切りした数値）の対応するIPv4アドレスを含む文字列を返します。特殊ケース：
- 入力パラメータが負の値または`4294967295`（`'255.255.255.255'`の数値）より大きい場合は`NULL`を返します

## 例

```sql
select ipv4_num_to_string(3232235521);
```
```text
+--------------------------------+
| ipv4_num_to_string(3232235521) |
+--------------------------------+
| 192.168.0.1                    |
+--------------------------------+
```
```sql
select num,ipv4_num_to_string(num) from ipv4_bi;
```
```text
+------------+---------------------------+
| num        | ipv4_num_to_string(`num`) |
+------------+---------------------------+
|         -1 | NULL                      |
|          0 | 0.0.0.0                   |
| 2130706433 | 127.0.0.1                 |
| 4294967295 | 255.255.255.255           |
| 4294967296 | NULL                      |
+------------+---------------------------+
```
