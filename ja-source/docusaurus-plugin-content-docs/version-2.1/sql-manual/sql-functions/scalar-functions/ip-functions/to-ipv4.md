## 説明
この関数は、IPv4アドレスの文字列形式を受け取り、IPv4型の値を返すipv4_string_to_numのような関数で、ipv4_string_to_numが返す値とバイナリ的に等しい値を返します。

## 構文

```sql
TO_IPV4(<ipv4_str>)
```
## パラメータ
| Parameter | Description                                      |
|-----------|--------------------------------------------------|
| `<ipv4_str>`      | String型のIPv4アドレス |


## 戻り値
IPv4型の値を返します。これはipv4_string_to_numが返す値とバイナリ的に等価です。
- IPv4アドレスが無効な形式の場合、例外をスローします


## 例

```sql
SELECT to_ipv4('255.255.255.255');
```
```text
+----------------------------+
| to_ipv4('255.255.255.255') |
+----------------------------+
| 255.255.255.255            |
+----------------------------+
```
