## 説明
他の入力ARRAY パラメータの内部データに対して対応する式計算を実行するために、オプションのlambda式を入力パラメータとして使用します。計算が0以外の値を返す場合は1を返し、それ以外の場合は0を返します。
lambda式には1つ以上のパラメータが入力され、後の入力配列列数と一致する必要があります。lambdaでは正当なスカラー関数を実行できますが、集約関数などはサポートされていません。

lambda式がパラメータとして使用されない場合、array1が計算結果として使用されます。


## 構文

```sql
ARRAY_EXISTS([ <lambda>, ] <arr1> [, <arr2> , ...] )
```
## Parameters
| Parameter | Description |  
|---|---|
| `<lambda>` | 有効なスカラー関数を実行できるラムダ式ですが、集約関数はサポートしていません。 |  
| `<arr1>` | 計算が必要な配列arr1 |  
| `<arr2>` | 計算が必要な配列arr2 |  

## Return Value
式を使用して計算された配列を返します。特殊なケース:
- 配列に`NULL`値が含まれているか、配列自体が`NULL`の場合、結果は`NULL`になります。

## Example

```sql
CREATE TABLE array_test2 (
    id INT,
    c_array1 ARRAY<INT>,
    c_array2 ARRAY<INT>
)
duplicate key (id)
distributed by hash(id) buckets 1
properties(
  'replication_num' = '1'
);
INSERT INTO array_test2 VALUES
(1, [1, 2, 3, 4, 5], [10, 20, -40, 80, -100]),
(2, [6, 7, 8], [10, 12, 13]),
(3, [1], [-100]),
(4, NULL, NULL);
select *, array_exists(x->x>1,[1,2,3]) from array_test2 order by id;
```
```text
+------+-----------------+-------------------------+-----------------------------------------------+
| id   | c_array1        | c_array2                | array_exists([x] -> x(0) > 1, ARRAY(1, 2, 3)) |
+------+-----------------+-------------------------+-----------------------------------------------+
|    1 | [1, 2, 3, 4, 5] | [10, 20, -40, 80, -100] | [0, 1, 1]                                     |
|    2 | [6, 7, 8]       | [10, 12, 13]            | [0, 1, 1]                                     |
|    3 | [1]             | [-100]                  | [0, 1, 1]                                     |
|    4 | NULL            | NULL                    | [0, 1, 1]                                     |
+------+-----------------+-------------------------+-----------------------------------------------+
```
```sql
select c_array1, c_array2, array_exists(x->x%2=0,[1,2,3]) from array_test2 order by id;
```

```text
| [1]             | [-100]                  | [0, 1, 0]                                         |
| NULL            | NULL                    | [0, 1, 0]                                         |
+-----------------+-------------------------+---------------------------------------------------+
```
