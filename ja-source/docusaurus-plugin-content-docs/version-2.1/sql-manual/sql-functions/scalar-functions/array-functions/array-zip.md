## 説明

すべての配列を単一の配列にマージします。結果の配列には、引数リストの順序でグループ化された、ソース配列の対応する要素が含まれます。

## 構文

```sql
ARRAY_ZIP(<array>[, <array> ])
```
## パラメータ

| Parameter | Description |
|--|--|
| `<array>` | 入力配列 |

## 戻り値

ソース配列の要素を構造体にグループ化した配列を返します。構造体内のデータ型は入力配列と同じであり、配列が渡された順序になります。

## 例

```sql
SELECT ARRAY_ZIP(['a', 'b', 'c'], [1, 2, 3]);
```
```text
+--------------------------------------------------------+
| array_zip(['a', 'b', 'c'], [1, 2, 3])                  |
+--------------------------------------------------------+
| [{"1":"a", "2":1}, {"1":"b", "2":2}, {"1":"c", "2":3}] |
+--------------------------------------------------------+
```
