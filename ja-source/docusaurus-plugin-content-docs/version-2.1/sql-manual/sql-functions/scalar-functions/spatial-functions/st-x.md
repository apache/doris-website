## 説明

pointが有効なPOINT型の場合、対応するx座標値を返します

## 構文

```sql
ST_X( <point>)
```
## パラメータ

| パラメータ | 説明 |
|------|----------|
| `<point>` | 二次元の点の幾何座標 |

## 戻り値

幾何座標のX値

## 例

```sql
SELECT ST_X(ST_Point(24.7, 56.7));
```
```text
+----------------------------+
| st_x(st_point(24.7, 56.7)) |
+----------------------------+
|                       24.7 |
+----------------------------+
```
