## 説明

WKT（Well Known Text）を対応する多角形メモリ形式に変換する

## エイリアス

- ST_POLYGONFROMTEXT
- ST_POLYFROMTEXT

## 構文

```sql
ST_POLYGON( <wkt>)
```
## パラメータ

| パラメータ | 説明 |
|------|--------------------|
| `<wkt>` | polygon関数によって生成されたPOLYGON |

## 戻り値

ポリゴンのメモリ形式

## 例

```sql
SELECT ST_AsText(ST_Polygon("POLYGON ((0 0, 10 0, 10 10, 0 10, 0 0))"));
```
```text
+------------------------------------------------------------------+
| st_astext(st_polygon('POLYGON ((0 0, 10 0, 10 10, 0 10, 0 0))')) |
+------------------------------------------------------------------+
| POLYGON ((0 0, 10 0, 10 10, 0 10, 0 0))                          |
+------------------------------------------------------------------+
```
