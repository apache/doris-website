## 説明

地球上の2点間の球面距離をメートル単位で計算します。渡されるパラメータは、点Xの経度、点Xの緯度、点Yの経度、点Yの緯度です。

## 構文

```sql
ST_DISTANCE_SPHERE( <x_lng>, <x_lat>, <y_lng>, <y_lat>)
```
## パラメータ

| パラメータ | 説明 |
| -- | -- |
| `<x_lng>` | 経度データ、妥当な値の範囲は [-180, 180] |
| `<y_lng>` | 経度データ、妥当な値の範囲は [-180, 180] |
| `<x_lat>` | 緯度データ、妥当な値の範囲は [-90, 90] |
| `<y_lat>` | 緯度データ、妥当な値の範囲は [-90, 90] |

## 戻り値

2点間の球面距離

## 例

```sql
select st_distance_sphere(116.35620117, 39.939093, 116.4274406433, 39.9020987219);
```
