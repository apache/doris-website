## 説明

`x`の逆正接を返します。ここで`x`はラジアン単位です。

## 構文

```sql
ATAN(<x>)
```
## パラメータ

| パラメータ | 説明 |
| -- | -- |
| `<x>` | atan値を計算する値 |

## 戻り値

パラメータ`x`のatan値。

## 例

```sql
select atan(0);
```
```text
+-----------+
| atan(0.0) |
+-----------+
|         0 |
+-----------+
```
```sql
select atan(2);
```
```text
+--------------------+
| atan(2.0)          |
+--------------------+
| 1.1071487177940904 |
+--------------------+
```
