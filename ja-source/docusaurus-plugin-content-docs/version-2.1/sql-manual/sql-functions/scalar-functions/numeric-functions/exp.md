## 説明

`x`をベース`e`で累乗した値を返します。

## エイリアス

- DEXP

## 構文

```sql
EXP(<x>)
```
## パラメータ

| パラメータ | 説明 |
| -- | -- |
| `<x>` | 独立変数 |

## 戻り値

double型の値を返す
- パラメータがnull値の場合、NULLを返す

## 例

```sql
select exp(2);
```
```text
+------------------+
| exp(2.0)         |
+------------------+
| 7.38905609893065 |
+------------------+
```
```
select exp(3.4);
```
```text
+--------------------+
| exp(3.4)           |
+--------------------+
| 29.964100047397011 |
+--------------------+
```
