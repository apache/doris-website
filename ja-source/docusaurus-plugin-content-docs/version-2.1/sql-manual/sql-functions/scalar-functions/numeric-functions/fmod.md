## 説明

浮動小数点型のa / bの余りを求めます。整数型の場合は、mod関数を使用してください。

## 構文

```sql
MOD(<col_a> , <col_b>)
```
## パラメータ

| パラメータ | 説明 |
|-----------|------------|
| `<col_a>`   | 被除数 |
| `<col_b>`   | 除数は0であってはならない |

## 戻り値

浮動小数点型を返します。特殊なケース：

col_a が NULL または col_b が NULL の場合、NULL を返します。

## 例

```sql
select fmod(10.1, 3.2);
```
```text
+-----------------+
| fmod(10.1, 3.2) |
+-----------------+
|      0.50000024 |
+-----------------+
```
```sql
select fmod(10.1, 0);
```
```text
+---------------+
| fmod(10.1, 0) |
+---------------+
|          NULL |
+---------------+
```
