## 説明

パラメータのコサインを計算します

## 構文

```sql
COS(<a>)
```
## パラメータ

| パラメータ | 説明 |
| -- | -- |
| `<a>` | 浮動小数点数、計算するパラメータのラジアン値 |

## 戻り値

パラメータ `<a>` のコサイン値（ラジアンで表現）。

## 例

```sql
select cos(1);
```
```text
+---------------------+
| cos(1.0)            |
+---------------------+
| 0.54030230586813977 |
+---------------------+
```
```sql
select cos(0);
```
```text
+------------------------+
| cos(cast(0 as DOUBLE)) |
+------------------------+
|                    1.0 |
+------------------------+
```
```sql
select cos(Pi());
```
```text
+-----------+
| cos(pi()) |
+-----------+
|        -1 |
+-----------+
```
