## 説明

整数型において、aをbで割った余りを求めます。浮動小数点型の場合は、fmod関数を使用してください。

## 構文

```sql
MOD(<col_a> , <col_b>)
```
## パラメータ

| Parameter | Description |
|-----------|------------|
| `<col_a>`   | 被除数 |
| `<col_b>`   | 除数は0であってはならない |

## 戻り値

整数型を返します。特殊なケース：

col_a が NULL または col_b が NULL の場合、NULL を返します。

## 例

```sql
select mod(10, 3);
```
```text
+----------+
| (10 % 3) |
+----------+
|        1 |
+----------+
```
```sql
select mod(10, 0);
```
```text
+----------+
| (10 % 0) |
+----------+
|     NULL |
+----------+
```
