## 説明

`x`の常用対数（底が`10`の対数）を返します。

## エイリアス

- DLOG10

## 構文

```sql
LOG10(<x>)
```
## パラメータ

| パラメータ | 説明 |
|-----------|------------|
| `<x>`   | 真数は0より大きい値である必要があります |

## 戻り値

浮動小数点数を返します。

- xがNULLの場合: `NULL`を返す

## 例

```sql
select log10(1);
```
```text
+--------------------------+
| log10(cast(1 as DOUBLE)) |
+--------------------------+
|                      0.0 |
+--------------------------+
```
```sql
select log10(10);
```
```text
+---------------------------+
| log10(cast(10 as DOUBLE)) |
+---------------------------+
|                       1.0 |
+---------------------------+
```
```sql
select log10(16);
```
```text
+---------------------------+
| log10(cast(16 as DOUBLE)) |
+---------------------------+
|        1.2041199826559248 |
+---------------------------+
```
