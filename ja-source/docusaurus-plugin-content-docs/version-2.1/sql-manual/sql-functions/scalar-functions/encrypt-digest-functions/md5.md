## 説明

文字列に対してMD5 128ビットチェックサムを計算します

## 構文

```sql
MD5( <str> )
```
## パラメータ

| parameter | description |
| -- | -- |
| `<str>` | 計算されるMD5値 |

## 戻り値

文字列のMD5値を返します。

## 例

```sql
select md5("abc");
```
```text
+----------------------------------+
| md5('abc')                       |
+----------------------------------+
| 900150983cd24fb0d6963f7d28e17f72 |
+----------------------------------+
```
