## 説明

指定された値をHLL（HyperLogLog）型に変換します。この関数は通常、データ読み込み時に生データからHLLを作成するために使用されます。

## 構文

```sql
HLL_HASH(<value>)
```
## パラメータ

| パラメータ | 説明                                                   |
|-----------|---------------------------------------------------------------|
| `<value>` | HLL型に変換される値。文字列、数値、または任意のデータ型を指定できます。 |

## 例

```sql
SELECT HLL_CARDINALITY(HLL_HASH('abc'));
```
```text
+----------------------------------+
| hll_cardinality(HLL_HASH('abc')) |
+----------------------------------+
|                                1 |
+----------------------------------+
```
