## 説明

table関数はフロントエンドの一時テーブルを生成し、現在のDorisクラスター内のFEノード情報を表示することができます。

## 構文

```sql
FRONTENDS()
```
## アクセス制御要件

| 権限  | オブジェクト | 備考 |
| :--------- |:-------|:------|
| ADMIN_PRIV | global |       |

## 戻り値
| フィールド                    | 説明                                                                                             |
|--------------------------|---------------------------------------------------------------------------------------------------------|
| **Name**                 | フロントエンドノードの一意名。                                                                   |
| **Host**                 | フロントエンドノードのIPアドレスまたはホスト名。                                                        |
| **EditLogPort**          | 編集ログ通信に使用されるポート。                                                               |
| **HttpPort**             | フロントエンドノードのHTTPポート。                                                                    |
| **QueryPort**            | フロントエンドノードでのクエリ実行に使用されるポート。                                                 |
| **RpcPort**              | RPC通信に使用されるポート。                                                                    |
| **ArrowFlightSqlPort**   | Arrow Flight SQLポート（高性能データ転送のためのApache Arrowとの統合に使用）。 |
| **Role**                 | フロントエンドノードの役割（例：`FOLLOWER`）。                                                       |
| **IsMaster**             | ノードがマスターノードかどうかを示す（true/false）。                                               |
| **ClusterId**            | フロントエンドノードが属するクラスターの識別子。                                      |
| **Join**                 | フロントエンドノードがクラスターの一部かどうかを示す（true/false）。                                     |
| **Alive**                | フロントエンドノードが生存しているかどうかを示す（true/false）。                                               |
| **ReplayedJournalId**    | フロントエンドノードによって再生された最後のジャーナルID。                                             |
| **LastStartTime**        | フロントエンドノードの最後の開始時刻のタイムスタンプ。                              |
| **LastHeartbeat**        | フロントエンドノードから受信した最後のハートビートのタイムスタンプ。                                   |
| **IsHelper**             | フロントエンドノードがヘルパーノードかどうかを示す（true/false）。                                      |
| **ErrMsg**               | フロントエンドノードによって報告されたエラーメッセージ。                                                       |
| **Version**              | フロントエンドノードのバージョン。                                                                       |
