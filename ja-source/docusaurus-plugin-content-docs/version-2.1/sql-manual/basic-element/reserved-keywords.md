## 説明

予約キーワードは、SQLで特別な意味を持つ単語です。予約キーワードはキーワードの特別なカテゴリです。これらをオブジェクト識別子として使用したい場合は、バッククォート（`）を使用する必要があります。例えば、```FROM```のようにします。オブジェクト識別子の詳細については、[Object Identifiers](../basic-element/object-identifiers)セクションを参照してください。
