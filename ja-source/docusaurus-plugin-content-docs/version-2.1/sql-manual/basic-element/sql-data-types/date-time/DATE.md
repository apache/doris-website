# DATE

## DATE

DATE

### 説明
#### 構文
date
日付型、現在の値の範囲は['0000-01-01','9999-12-31']で、デフォルトの出力形式は'yyyy-MM-dd'です。

### 例

```sql
SELECT DATE('2003-12-31 01:02:03');
+-----------------------------+
| DATE('2003-12-31 01:02:03') |
+-----------------------------+
| 2003-12-31                  |
+-----------------------------+
```
### キーワード
DATE
