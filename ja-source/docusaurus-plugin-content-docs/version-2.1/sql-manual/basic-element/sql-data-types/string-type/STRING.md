## STRING
### 説明
STRING (M)
可変長文字列。デフォルトサポートは1048576バイト（1M）で、最大2147483643バイト（2G）まで調整可能です。Stringタイプの長さはbeのstring_type_length_soft_limit_bytes（文字列タイプ長のソフト制限）設定によっても制限されます。Stringタイプは値列でのみ使用でき、キー列およびパーティションとバケット列では使用できません。

注意：可変長文字列はUTF-8エンコーディングで保存されるため、通常英字は1バイト、漢字は3バイトを占有します。
